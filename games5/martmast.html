[ 故事背景 ]¶

[ 故事背景 ]
────────────────────────────────────
    清末年间，满清积弱不振，遭受列强满箩筐的欺凌，全国民怨四起，这时人心
从不信任清廷转而全依赖宗教，而白莲教就趁此良机崛起，打着白莲下凡，万民翻
身，扶清灭洋，天下太平的口号，深受民众信赖，因此凡是与西洋有关的人、事、
物，一律都加以毁坏，而与他们为敌的，就被冠上妖孽的罪名处以极刑，形意门掌
门龙拳与宝芝林黄飞鸿，看不惯白莲教众到处杀害无辜，因而多次阻挠，于是双方
结下间隙，一天，白莲教主下书约见形意门掌门及黄飞鸿，望能解除双方误会，虽
有人劝说白莲教此举乃是陷阱，而在黄飞鸿外出采药未归前不可前去，但龙拳不愿
放弃此能开导白莲教主的机会，而不思有诈单独赴约，结果中计被擒，双方门下为
此发生数次严重冲突，种下日后纷乱的源头。

    现今广州暗潮汹涌，危机四伏，到底有何能人可排除阴谋的危机，力挽动荡的
狂澜呢？就让我们拭目以待吧！

[ 基本操作 ]
────────────────────────────────────
以1P角色为准(即游戏角色面向右)

①操纵杆
  ↖↑↗  后跳/垂直跳跃/前跳
  ←Ｎ→  后退&站立防御/回中/前进
  ↙↓↘  下蹲&下蹲防御/下蹲/下蹲

  →→：快速前进(第二下→不放就会持续快跑前进)
  ←←：快速后退
  角色瞬间向后移动，是贴在地面滑行向后的移动技巧。

  残影跳：
  瞬间输入↙or↓or↘或快速前进中，再输入↖or↑or↗
  比一般跳跃跳得更高的技巧，在使出过程中角色会出现残影。

②按钮
  Ａ：轻拳
  Ｂ：轻脚
  Ｃ：重拳
  Ｄ：重脚
  开始键：开始键

③摔技      近敌时Ａ＋ＤorＢ＋Ｄ
  接近对手使用，让对手无法防御并将对手摔出的技巧。
  
  基本摔挣脱
  对手的投技成立瞬间按下Ａ＋Ｄ or Ｂ＋Ｄ，就可以解除摔技动作。
 
④奇袭       ↓＋ＡＢ
  玩家角色瞬间小跳跃后使出中段攻击，可破坏对手
  蹲下防御的技巧，兼具偷袭与闪避下盘攻击的效果。
 
⑤后闪奇袭   ←＋ＡＢ
  兼具闪避与发动攻击的技巧。先后闪再快速向前跳跃，
  跳跃途中可进行攻击。
 
⑥倒地回避   倒地前瞬间←＋ＡＢ or →＋ＡＢ
  玩家角色倒地前瞬间回避，用来躲避对手倒地追击的技巧，
  还可制造攻击的机会。
 
⑦起身回避   起身前瞬间←or→ / 倒地中按住←or→不放
  角色从倒地状态起身前瞬间，向后方或前方翻滚移动的
  闪避技巧，目的是避免受到对手在我方起身时的立即攻击。 

⑧倒地追击   对手倒地中↓or↑＋ＡorＢorＣorＤ
  对手倒地时进行的追加攻击，使用时机为对手倒地之后
  到爬起之前。
  如果对手顺势使出倒地回避，则有可能让此招落空。
 
⑨挑衅       Ｃ＋Ｄ
  让角色作出挑衅动作的技巧，除了让对手心情浮动之外，
  主要的目的是累积气量表。
  部分角色的挑衅具有些许攻击效果，须在贴近对手的距离使出才有效果。

[ 特殊操作方法说明 ]¶

[ 特殊操作方法说明 ]
────────────────────────────────────
  以1P角色为准(角色面向右)必须消耗气量表才能使出的技巧

①分身攻击  消耗１～2条气量表
 击飞型分身：↓↘→＋ＢＣ
 援护型分身：↓↙←＋ＢＣ

 击飞型分身起招时，本体和分身都有相当长的无敌时间，发生的速度很快，
 通常用在连段中或起身反击时。在地面击中对手时最后一段会将对手击飞，
 击飞后还有四次空中连击的机会(能量槽上方会显示 COUNTER = 4，随击中
 次数减少)。
 追击的规则是同一招式只能使用2次。如击飞·空中Ｃ·空中Ｃ·空中Ｃ的话，
 第三个空中Ｃ就会落空，但可以换接其它招式，如空中Ｄ。
 若击飞分身被对手防住会马上就消失。
 发动此技巧会消耗1条气量表。

 援护型分身起招时，本体和分身都有相当长的无敌时间，发生的速度慢，
 原则上该分身是不能当成招式的连接来使用的，不过援护型分身的好处是不论
 是否击中对手，分身都能持续一段时间的攻击，使对手难以行动，此时本体可
 上前进行上下二择或绕背攻击，是相当好用的攻击辅助系统。
 发动此技巧会消耗1条气量表。
 
②背闪 消耗１条气量表 
  近距离←↙↓↘→＋ＡＢ
  接近对手后破坏其防御，瞬间移到对手后方从背后进行攻击，造成更大的伤害。
  在背闪后展开的攻击其破坏力是一般攻击的1.35倍，威力十分可观。
 
③觉醒 消耗１条气量表 
  防御中Ｃ＋Ｄ
  在防御状态下，用来震开对手并取得主攻权。
  被震开的对手会倒地或暂时僵硬，玩家可以使用倒地追击或发动攻击取得先机。
 
④超必杀技 消耗1条气量表 指令依角色不同
  依角色不同的超级必杀技，使出时画面会出现闪光而且角色会出现残影。
  威力与架势都相当威猛的攻击招式，
  每招超必杀技都具有许多连招数，
  运用得宜顺利命中对手就可以令连招数(HITS)大量增加

[ 各种量表 ]¶

[ 各种量表 ]
────────────────────────────────────
各种量表以棒状图表示的游戏量表。

①体力量表
  表示角色体力的量表，当玩家受到攻击或防御攻击时都会减少。
  当此量表减为0时，玩家就输一回合。
  在对战模式中，当玩家打倒对手进入下一回合时，此量表会少量恢复。
 
②气量表
  位于画面下方的量表，当攻击命中对手或遭到攻击、
  防御时都会累积气量表，累积数最高为9。
  使用分身攻击、背闪、觉醒及超必杀技时将会消耗1个
  气量表另外，使用挑拨指令可以大量累积气量表。
 
③防御崩溃量表
  位于体力量表下方的红色线状量表，当玩家防御住对手
  攻击时就会逐渐累积(此时依然会少量失血)，
  当累积到满时就会防御崩溃。
  只要玩家操作角色解除防御动作，此量表就会逐渐减少

○挑战者出现条件：    
在单打模式中的前4场共8个回合战斗中，玩家全部回合都没有输过(中途也不可以
有其它玩家插入对战)，且玩家以超必杀技及分身攻击K.O对手4次以上(含4次)，
在第五场战斗开始前就会出现挑战者向玩家挑战。

角色对应之挑战者：
  黄 飞 鸿 → 白 少 廷 ； 醉 不 归 → 虎    克
  白 少 廷 → 鬼 脚 七 ； 虎    克 → 纪 云 袖
  罗    汉 → 灵    儿 ； 鬼 脚 七 → 曲 嫣 红
  孙 铁 儿 → 鬼 脚 七 ； 曲 嫣 红 → 十 三 姨
  纪 云 袖 → 曲 嫣 红 ； 灵    儿 → 醉 不 归
  十 三 姨 → 曲 嫣 红 ； 白莲教主 → 纪 云 袖

○直接挑战宿命敌人
方法一：指定角色破关一次后，在角色选择画面中将游标移至该角色，
        按住开始键不放再按Ａ选择
方法二：在标题画面中依序按下Ａ、Ｂ、Ｃ、Ｄ、→、→，
        如果成功则会听到特殊的效果音。
        之后在角色选择画面中将游标移至该角色，按住开始键不放再按Ａ选择

        角色对应的之后敌人如下：
         黄 飞 鸿 → 白莲教主
         醉 不 归 → 白莲教主
         白 少 廷 → 黄 飞 鸿
         虎    克 → 白莲教主
         罗    汉 → 孙 铁 儿
         鬼 脚 七 → 白 少 廷
         孙 铁 儿 → 罗    汉
         曲 嫣 红 → 纪 云 袖
         纪 云 袖 → 白莲教主
         灵    儿 → 白莲教主
         十 三 姨 → 黄 飞 鸿
         白莲教主 → 黄 飞 鸿

○专家模式(EXPERT)
模式解锁：
方法一：所有的角色(包括十三姨)都破关之后或者。
方法二：机台运行一个月之后，该模式自动开放。
方法三：在“PRESS 1P OR 2P START BUTTON”标题画面
中依序按下Ａ、Ｂ、Ｃ、Ｄ、↑、↑
成立指令：模式解锁后，“PRESS 1P OR 2P START BUTTON”标题画面中
          ，按住"ＡＢＣＤ"键不放，再按开始键开始游戏

○满气模式(CRAZY)
模式解锁：
方法一：游戏运行"三个月"后，此模式将自动开放。
方法二：在标题画面中依序按下Ａ、Ｂ、Ｃ、Ｄ、↓、↓
成立指令：模式解锁后，角色选择完成瞬间，
          1P及2P摇杆同时按住↑＋ＡＢＣＤ＋开始键
本秘技仅适用于双打。

○十三姨选择方法
方法一：游戏运行20天后，角色选择界面出现99枚投币数倒数选框，
        投币99枚后出现十三姨选项。
方法二：在角色选择画面中将游标移至任一角色，逆时钟移动游标绕选框一圈
        回到该角色，再移至问号选框按下ＡorＢorＣorＤ中任一键即可。
