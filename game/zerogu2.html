zerogu2 零式枪手2 (C) Psikyo 2001¶

零式枪手2 (C) Psikyo 2001

枫枫的游戏机 粉丝专属街机游戏秘籍 街机游戏名称代码：zerogu2

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────────
  ↖↑↗
  ←Ｎ→
  ↙↓↘

  Ａ：射击(可按住连射)
  Ｂ：锁定
  Ｃ：特殊攻击

击毁敌机会爆出Ｅ道具，吃这些道具可增加特殊槽。
特殊槽每满一次，可增加一次特殊攻击。
如果你不开火，Ｅ道具会被自机自动吸取。
DC版的特殊槽增加速度比街机版快许多。

[ DC版隐藏要素 ]¶

[ DC版隐藏要素 ]
───────────────────────
·Expert模式
默认或更高难度下通游戏。

·Replay模式
通游戏。

·增加币数
玩几次后就会增加币数。玩的次数越多，币数越多。
最终会是无限币。

[ Rank系统 ]¶

[ Rank系统 ]
───────────────────────
和彩京其它射击游戏相同，该游戏的Rank系统也是：
火力最大时、玩家生存时间越久，Rank难度会逐渐
提升，敌人弹速和射击频率就会加快。
但有一点和其它彩京射击游戏的Rank不同：
撞击敌机和火力等级下降不会降低Rank难度。
