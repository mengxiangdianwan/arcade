[ 勋章系统 ]¶

[ 勋章系统 ]
───────────────────────
    得到那种貌似盾牌的徽章时，便出现一个Combo
时间槽，在这段时间内得一定分数就会得到勋章。
时间槽结束时，会根据得到的勋章数，显示英文。
最终打满时显示METALISH。
    勋章共4级：蓝色、绿色、红色、银色。每级4个
，最终银色打满便勋章满级。中途如果死命，所得的
勋章便全部消失。到了关底根据你打满的那一级勋章
来加分：蓝色加1万分，绿色加5万分，红色加10万分
，银色加50万分。

攻击方式得分表：
近身匕首：500
武器/手雷：100
近身匕首击杀敌人是打满勋章的最有效手段，蹲下匕
首攻击的速度更快。

部分关卡勋章满级要点：
第1关：关卡开头的卡车上的战俘可得到Combo徽章。
杀死敌兵后，卡车右边会不断补兵，所以在杀死第一
波敌人后，迅速在卡车上最右边蹲下堵着敌人对右边
匕首攻击可迅速提升勋章等级。
上路，摩托车行驶场景段。第6辆货车内有Combo徽章
。等货车行驶到靠左边一些再击破，不然徽章不容易
得到。得到徽章后，会出现直升机群，先击毁红色的
，15万分得手。

第2关：关卡开头的木箱里有Combo徽章。站在画面最
左边向右射击货车里出现的敌兵，很容易打满勋章。

第3关：走下路才有Combo徽章。得到第一个Combo勋
章后，原地向上射击可打出隐藏的3万分加分道具，
勋章满级。

第4关：
僵尸路线——打出隐藏COMBO徽章的场景，迅速击杀
橙衣防化兵后，蹲在下落敌兵的两条绳索之间的地点
向左射击，可迅速满级。
木乃伊路线——吃到COMBO徽章后，蹲在画面最左端
向右射击，可轻松获得银色勋章。

第5关：关卡开头第一个COMBO徽章的场景，得到徽章
后再迅速击毁坦克和后方的两个集装箱就能打满勋章。

第6关：这关利用COMBO徽章获得银色成就勋章要求比
较苛刻，只有唯一的1个。
01红数字场景破坏一辆坦克后的[L]枪不要浪费掉。
下一个场景吃到COMBO徽章后马上将左侧的地雷车干
掉，然后迅速将右边的坦克引出来干掉。这样就能成
功得到银色勋章。

[ 状态说明 ]¶

[ 状态说明 ]
───────────────────────
[胖子]
食物道具吃多后的状态。
移动速度和射击速率下降、火力的攻击范围变大。
“减肥药”道具可解除胖子状态。

[木乃伊]
被木乃伊攻击后的变身形态。
武器被限制为手枪。不能坐乘物。
移动速度大幅降低、手榴弹投掷速度也变慢。
一定概率下，扔手榴弹时会把头扔出去。
跳跃能力提高。
得到强化武器都会变成1000分。
“圣水”道具可解除该状态。

[僵尸]
被僵尸攻击后变成的形态。
武器被限制为手枪。不能坐乘物。
移动速度比木乃伊状态更慢。
不能下蹲，跳跃力几乎丧失。
但这个形态下角色不怕普通敌人的攻击
(丧尸、雪人除外)，
而副武器也会变成威力巨大的横扫角色
面前90度范围的喷血攻击。
“急救箱”道具可解除该状态。

[猩猩]
被第1关和最后一关的白衣博士攻击后变身状态。
移动速度下降、连射速率提升。
不能使用其它武器。不能坐乘物和转摇把。
跳跃力极强，可以抓住墙上的管道快速移动，
此时比地面移动速度快。
“药剂”道具解除该状态。

[ 特殊设定 ]¶

[ 特殊设定 ]
───────────────────────
蹲着射击会提升射击速度。

攻击取消：蹲下和站起能取消射击动作。这个设定可
用于节省[H]枪和[L]枪弹药。

重合快刀：和敌人重合在一起时，蹲下左右快速摇方
向并不断攻击。

蹲跳快刀：等敌人贴近时，蹲下砍一刀后立刻同时按
下攻击和跳，就可以在原本砍出一刀的时间内出两刀。

跳跃刀：敌人近身后，射击和跳一起按（或先按射击
、后按跳）就是跳跃刀。该设定的好处是可以在攻击
敌人的同时躲避攻击。

手雷“熏”：就是利用手雷爆炸后的烟雾攻击敌人，
用这种方法可以多赚分。

从二代开始，乘物过关加分条件改为必须使用乘物击
毁BOSS，而在一代中只要击毁BOSS时场景中有乘物即
可。

从二代开始，用刀杀死敌人500分，过关后每名战俘
奖励有10000分，特殊人物（相川玛莉娜，龟波老头
-一文字百太郎）有30000加分，乘物有100000分，
打掉有捆绑战俘的绳索和打隐藏战俘点都也有加分
，所以能用刀砍尽量用刀，减少重型武器的使用是
拿高分的方法。此外打部分隐藏点有额外加成分，
方法是只能用手枪来打，会得到剩余隐藏东西成倍
的加分。比如隐藏点有3名战俘，用手枪打一枪可以
得300分，还剩2名战俘时打一枪有200分，以此类推。

可射击打破相川玛莉娜的背包，会掉出三个加分道具
。注意若长时间不去撞玛莉娜，她会跑掉。若打破背
包，放出三个道具一秒后也会跑掉，所以必须在打破
背包放出第三个加分道具后立刻撞她。

鸡生蛋贪分：第一关，不要把标有HOG的货车两边的
轿车破坏。击毁货车有3只鸡。由于两边小车挡住去
路，鸡便会一直延续鸡生蛋、蛋生鸡的规律，注意时
间就行。

猩猩博士：脸对脸贴近白衣博士，他会用注射器扎你
。若躲开，则会扎到自己变为猩猩。猩猩是2000分加
分道具。白衣博士只在第一关下路分支以及最后一关
才出现。

[ 武器简介 ]¶

[ 武器简介 ]
───────────────────────
[-]手枪(弹药无限)：
基本武器。
速度：5/5
威力：1/5
射程：5/5

[H](Heavy Machine Gun)：
机枪。可斜方向射击，火力威力为手枪两倍。
弹药量：200
补给量：150

[Ｈ](BIG Heavy Machine Gun)：
弹药量：200
补给量：150
连射速度提升、子弹体积变大。
射击死角变小。威力同[H]。

[2H](Dual Machine Gun)：
双机枪。全游戏射击速率最快的武器，
威力也大，但子弹消耗非常大。
弹药量：200
补给量：150

[S](Shotgun)：
散弹枪。射程为1/3画面，但是全武器中威力最大的
，可贯穿墙壁。
弹药量：30
补给量：10

[Ｓ](BIG Shotgun)：
弹药量：15
补给量：10
射程距离有半屏。威力同[S]。

[R](Rocket Launcher)：
火箭发射器。有跟踪性能和爆炸范围杀伤判定，
跟踪范围限制在射击方向45度内。威力比手榴弹稍小。
弹药量：30
补给量：20

[Ｒ](BIG Rocket Launcher)：
弹药量：30
补给量：20
子弹体积变大，判定范围也变的更大。威力同[R]。

[F]Flame Shot：
火焰发射器。射程2/3画面。火焰消失前可造成持续
伤害。
弹药量：30
补给量：15

[Ｆ](BIG Flame Shot)：
弹药量：30
补给量：15
火力范围变得很大，火焰消失前可造成持续伤害。
威力同[F]。

[L]Laser Rifle：
镭射枪。威力比机枪大，不能斜向射击，
对普通敌兵有贯穿攻击效果。
弹药量：200
补给量：150

[Ｌ](BIG Laser Rifle)：
弹药量：200
补给量：150
火力直径明显变大，对普通敌兵有贯穿攻击效果。
威力同[L]。

[C](Enemy Chaser)：
跟踪弹。跟踪性能好，发射速率比火箭快，
但威力比火箭小，爆炸火焰可贯通墙壁。
弹药量：40
补给量：15

[G](Super Grenade)：
榴弹枪。攻击方式类似火箭发射器，但威力更大，
威力同手榴弹，爆炸火焰可贯通墙壁。
弹药量：20
补给量：5

[D](Drop Shot)：
跳跳弹。碰到墙壁地面才会慢慢弹跳前进，
爆炸火焰可贯通墙壁。
弹药量：30
补给量：10

[I](Iron Lizard)：
铁蜥蜴。近似于地面导弹，爆炸火焰可贯通墙壁。
弹药量：30
补给量：10

[CLOUD](Thunder Cloud)：
雷暴云。玩家操纵的角色上方将会有一小片乌云跟随
杀敌，并随着主角手枪的攻击发出闪电攻击敌人，遭
受攻击的敌人将会有硬直状态，初始弹药30发。

  燃烧瓶威力小于以上两者，但范围更大
手榴弹：
图示为炸弹箱。
补给量：10

燃烧瓶：
将手榴弹变为燃烧瓶。
补给量：10

石头：
