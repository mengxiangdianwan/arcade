neobombe Neo炸弹人 (C) Hudson Soft 1997¶

Neo炸弹人 (C) Hudson Soft 1997

枫枫的游戏机 粉丝专属街机游戏秘籍 街机游戏名称代码：neobombe

[ 基本操作 ]¶

[ 基本操作 ]
────────────────────────
  Ａ：放炸弹
      踢炸弹：吃到红色皮靴后，站在炸弹旁←→
      举炸弹：吃到蓝色手套后，站在炸弹旁按住Ａ；
              松开Ａ是扔炸弹。

  Ｂ：必杀技
      对战模式中可发动指令必杀技

  Ｃ：引爆遥控炸弹／中止被踢出的炸弹的移动

[ 对战模式的必杀技 ]¶

[ 对战模式的必杀技 ]
────────────────────────
角色特殊技仅限对战模式。有乘物时无法使用必杀技。

○炸弹人 (白／黑／红／蓝)
  无必杀技

○RUBBER：
变身      ↓＋Ｂ
变身史莱姆形态。
史莱姆形态可穿越箱子，但不能吃道具和放炸弹。

○FAKE：
伪装      Ｂ按住不放时↓，再次↓是解除伪装。
外观变得和场地内的箱子完全一样。

○CAT：
暴走      Ｂ按住不放时按住↓蓄力
松开Ｂor↓后会根据蓄力时间暴走。
暴走状态下角色四处乱跑，无敌状态且不可控制。
若碰到其它角色会把他获得的道具打出来。

○HONEY：
爱心      放置炸弹后↑＋Ｂ
炸弹会变为爱心炸弹，四处乱窜直到时间耗尽而爆炸。

○HAYATE：
假炸弹    Ａ＋Ｂ
外观和真炸弹完全一致，但不会爆炸不能堵人。

○KOTETU：
瞬移      ↑＋Ｂ
随机移动到场地另一位置。

○GOLD：
滑行      任意方向＋Ｂ
高速单向滑行一段距离，中途调整方向则中断。

○ATOMIC：
原子护盾  ↓＋Ｂ
对炸弹免疫但不可移动、１５秒冷却时间。

[ 关于道具 ]¶

[ 关于道具 ]
────────────────────────
黑色炸弹：炸弹放置数+１
红色炸弹：炸弹火焰有贯通箱子效果
遥控炸弹：Ｃ可引爆放置的炸弹
红色火焰：火力距离增加１格
金色火焰：火力距离最大
红色皮靴：可踢走炸弹
蓝色手套：可举起并投出炸弹
蓝色溜冰鞋：提高移动速度
闪烁的方块：可穿越箱子
骷髅头：随机状态(方向错乱、自动放炸弹、
                 炸弹距离最小／大、速度最快／慢)
防火服：短时对炸弹火焰无敌
钟表：时间追加３０秒
心脏：死亡无效化１回

[ 关于乘物 ]¶

[ 关于乘物 ]
────────────────────────
骑坐骑时无法使用必杀技，但乘物能帮玩家承受一次
炸弹攻击，且绝大部分有自己的特殊技。

乘物分生物系、机械系两大系列。
生物蛋是生物系乘物、机械扭蛋是机械系乘物。
同系列的乘物被吃到时，会成为备用乘物挂在后方，
最多挂两个。

[ 生物系乘物 ]

○鸵鸟
  超高速移动

○蓝色大嘴鸭  
  Ｂ可把面前的炸弹挑飞至随机位置

○猪猪
  速度极慢，无视蓝色溜冰鞋的加速效果

○毁灭蛋
  Ｂ使场上所有箱子被消掉
  乘物使用特殊技后立即死亡

○蝌蚪
  碰到的炸弹会被伪装成箱子(幻觉)
  被伪装的炸弹的爆炸时间会重置
  使用该乘物时不能使用红色皮靴的踢炸弹技能


[ 机械系乘物 ]

○黑色导弹
  Ｂ将乘物直线飞出撞击目标爆炸
  爆炸威力等于２级威力的炸弹。
  使用特殊技时，玩家跳下乘物时有短时无敌

○绿色齿轮
  任意方向＋Ｂ可快速移动。
  中途调整方向则中断。 

○蓝色水母
  被动效果：无视箱子阻碍

○章鱼
  Ｂ按住不放蓄力，松开Ｂ后会根据蓄力时间暴走。
  暴走状态下角色四处乱跑，无敌状态且不可控制。
  若碰到其它角色会把他获得的道具打出来。

○白骷髅头
  被动效果：骷髅头道具免疫
            给予碰触的敌人随机状态
            方向错乱、自动放炸弹、
            炸弹距离最小／大、速度最快／慢
