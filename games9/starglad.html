  ＡＢＤ

·等离子连携技
  按照一定顺序与节奏输入特定按键即可发动组合技。
  节奏控制在击中敌人的攻击不中断，太快和太慢都会使连携技中断。
  若输入正确的按键组合，会在体力槽下方的PLASMA框中显示。
  连携技输入中输入Ｃ可强制中止连携技。

  实际上，部分等离子连携技并不是严格意义上的组合技，其后续
  攻击可以被防御。部分攻击需要站立防御，部分攻击需要下蹲防
  御，部分攻击则是站立和下蹲防御均可。

·等离子终结技
  等离子连携技输入至特定点(☆)时，按键颜色会紫红色，立刻输入
  终结技指令，即可发动等离子终结技。
  发动终结技后，PLASMA框会显示FINAL。

  比如Hayato的等离子终结技指令为↑＋Ａ，则可以依次输入
  ＡＡＡＤＡorＡＡＢＡＡ，最后输入↑＋Ａ发动终结技。
  每名角色的组合技详见出招表图。

·出界(RINGOUT)
  每个战斗场景都有边界，超越边界会摔下去，掉下去的角色直接
  输掉当前回合。

[ 隐藏要素 ]
──────────────────────────────
①使用2P角色
  选择角色时，按住↑即会出现当前角色的2P形态。

②对决超级比尔斯坦SUPER BILSTEIN
  依机台设置中的单场战斗最高回合数(MAX-ROUND[CPU])的设定的
  不同，规定的时间内击败BILSTEIN，即会对决SUPER BILSTEIN

  TEST MENU - CONFIGURATION - GAME - MAX-ROUND[CPU]
  MAX-ROUND[CPU]数 １……3分钟以内
  MAX-ROUND[CPU]数 ３……6分钟以内
  MAX-ROUND[CPU]数 ５……9分钟以内
  MAX-ROUND[CPU]数 ７……12分钟以内

  击败SUPER BILSTEIN即可看到真结局；败给SUPER BILSTEIN是
  坏结局；如果没有达成SUPER BILSTEIN出现条件，则是另外一
  种坏结局。

  CPU第3战决胜局击败对手时，画面右下角的总用时的最后２位数
  刚好是“００”的话，即会在第4战对阵河童Kappah。

  直至出现WINNER字样时，再松开。
  此时摇杆可控制镜头方向，按对应键可控制视角远近。
