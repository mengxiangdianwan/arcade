1943 1943 (C) Capcom 1987¶

1943 (C) Capcom 1987

枫枫的游戏机 粉丝专属街机游戏秘籍 街机游戏名称代码：1943

[ 基本操作 ]¶

[ 基本操作 ]
─────────────────
  Ａ    ：射击
  Ｂ    ：MegaCrash
          消耗8格能量
  Ａ＋Ｂ：空翻
          每关2次

[ 物品简介 ]¶

[ 物品简介 ]
─────────────────
[ 竹笋 ]
通过特定的云层上面就会出现。
2000分。

[ 荷兰乳牛 ]
射击特定的位置就会出现。
在天空、海面都会出现。
20000分。

[ 桶 ]
射击特定的海面就会出现。
10000分。

[ 碎片 ]
在空中场景中射击特定的舰船就会
出现。
10000分。

[ 蜻蜓 ]
在特定位置使用空翻就会出现？
10000分。

[ 招财猫 ]
在亚也虎关卡中，通过特定的云层就会
出现。
出现条件是随机的？
可得到镭射装备。有时间限制。

[ 佐吉 ]
射击画面上下成一直线飞行的喷气式战
斗机就会出现。
5000分。
PowerShot 的时间增加到64秒。

[ 宇航员 ]
舰船 BOSS 被破坏时出现。
出现概率是1/16。
100000分。

[ POW ]
燃料补充道具。
攻击它，会变成武器道具。

[ 隐藏要素 ]¶

[ 隐藏要素 ]
─────────────────
·弹药全满
  击败关底BOSS后，立刻按住Ａ，直到下一关起飞后再松开。
  玩家将获得弹药全满效果65秒。每3关可使用该方法一次，
  会和武器自选产生冲突。

·武器自选

入下面包含的指令，就可得到以下所列
