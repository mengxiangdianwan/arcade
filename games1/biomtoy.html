当跳到怪物身上或气球上就会变成踩踏，之后会向上腾空

方或高空拿道具。

[ 全BOSS攻略心得 ]¶

[ 全BOSS攻略心得 ]
─────────────────────────
The WizArd Queen
BOSS战前场景，攻击那个大脸，吐出的气球会飞到BOSS场
景，全是补血道具。
第一关的BOSS，难度不大，先开枪射她，如果她放电星就
蹲下继续射，放火就等火到的时候跳起，血量减到一定程
度她就会变三次身，不过都是变成小兵，注意躲避就可以
了。

The Flying King
BOSS战前场景，攻击那个大脸，吐出的前两个气球会飞到
BOSS场景，全是补血道具。	
第二关的BOSS，难度也不大，放飞弹就跳开避过，如果他
往右边放飞弹就利用弹跳台跳到他身后打他，接着他会由
左至右投炸弹，很容易就能避开。

The Virus
第三关的BOSS，小心他放出的病毒就可以了，
如果他迫到你没路可退就利用弹跳台跳到他身后。

The Alien
第四关的BOSS，最好先将头顶那吐火网那个先打掉，
不然就会弄得你很头痛，每打他一段时间就会瞬移，
走开到另外一个地方打就可以了，难度不大。

The Green DrAgon
第五关的BOSS，注意避开他喷出的火焰和火球就可以了。

SCruBBy
最终BOSS，非常烦人，经常跳来跳去，而且跳的时候和落
地时有一段时间是无敌的，小心被他压到，要注意他放出
来的电球会放电柱，还有就是从天上掉下来的石头，抓紧
机会打他，耐心一点就可过关。

[ 道具攻略 ]¶

[ 道具攻略 ]
─────────────────────────
这个游戏一个很特别的地方是走过的场景还可以走回去，
所以除非有地形限制，有遗漏的道具还可以回去拾取。
奖命分数：200W、500W。

关卡1：
1、10W分道具：关卡一开始的画面，对着左边的雕像头射
击。
2、10W分道具：关卡开始不久，对着大型仙人掌中部下面
射击。
3、10W分道具：继续走，上到写着“SALOON”的屋顶后，
向左走，击毁刺球。
吊刺球的架子顶部有一个打火机(2500分)，需要用弹跳器
跳到架子上才能吃到。
4、10W分道具：同上述场景，刺球架左边是一个烟囱，站
在烟囱上向下攻击烟囱。
5、10W分道具：烟囱左边，利用紫色气球连续向左跳至最
高处。除了10W分道具，还有其它加分道具。

关卡2：
1、10W分道具：开始不久那个扑克牌敞篷，向着帐篷口下
面射击。
2、补血道具：利用第一段下坡处的几个紫色气球向下连
续跳到镰刀架上。
3、N多加分道具：升降台处，利用右边第一个蜂巢飞出的
马蜂(共5只)向上跳跃。必须一到升降台上，立刻向右边
的蜂巢跳，才能刚好利用马蜂连续上跳。
本游戏最多加分道具最多气球囤积的地方……
4、补血和加分道具：攻击大脸让它吐出气球，然后利用
气球向右连跳到大飞镖上，全是补血道具。
到顶后，再向左连跳是加分道具。
5、10W分道具：第2个扑克牌帐篷场景后，先不要破坏那
两个摞起来的箱子，站在箱子跳起来向上打紫色气球。
6、加分道具x4：上述场景之后的5个波浪升降地板处
，当地板升到最高处时，向上攻击会打到紫色气球。
右数，第2-第5个升降地板正上方各有一个。
7、10W分道具：继续走，利用怪物跳上镰刀台，在镰刀台
上，先跳向左边那个紫色气球，再迅速向右连续跳跃。
除了10W分道具，还有其它加分道具。
8、2500分道具x5：风力上升处、先向下移动会见到5
个打火机。
9、补血道具：BOSS战前场景，攻击那个大脸，前两个气
球会飞到BOSS场景。BOSS战处利用弹跳器可攻击到右边的
气球。

关卡3：
1、10W分道具：第一节车厢后，利用弹跳器向左跳，再利
用气球连续向左跳。除了10W分道具，还有其它加分道具
2、10W分道具：第三节车厢后，利用弹跳器向左跳，再利
用气球连续向左跳。除了10W分道具，还有其它加分道具
3、10W分道具：火车头车厢里。
4、补血道具：BOSS战处，利用弹跳器攻击上方的气球。

关卡4：
1-UP：第3个岩浆池前，立刻向右跳，按住↗，刚好利用
岩浆池上方洞里飞出的敌人跳到树上踩死蝙蝠怪。
注意——用枪是打不出来的，且只有一次机会，因为洞
里的怪只飞出一批。

关卡5：
10W分道具：在一个平台左边边缘有向右指的白色女孩雕
塑的场景，打烂右边刺球。

关卡6：
1、10W分道具：第2处会陷落的地面后(出现帽子的场景)
，利用弹跳器先向左连续跳，再向右连续跳到最高处。
2、10W分道具：击毁木墩场景尽头陆地上的刺球。

[ 帽子的秘密 ]¶

[ 帽子的秘密 ]
─────────────────────────
游戏中某些地方会出现有一双翅膀的长着眼睛的黑色礼帽
。射击帽子，会掉下宝物或补血道具，如果将帽子中所有
道具全打出，可得额外20W加分。

[ 气球的作用 ]¶

[ 气球的作用 ]
─────────────────────────
分紫色和红色两种。击破紫色气球可得道具，也可利用气
球弹跳。红色气球只用来弹跳。部分道具必须利用气球弹
跳才能吃到。有些地方的气球很隐蔽(比如在地区的最上
端，要用二段跳才能看到)。攻击大脸会吐出气球。

[ 红旗的作用 ]¶

[ 红旗的作用 ]
─────────────────────────
路过插在地上的红旗，会使旗子立起来，这就意味着若死
在该旗子至下一个旗子之间的地段，会在该旗子处继续游
戏。

[ 第一关BOSS的贪分方法 ]¶

[ 第一关BOSS的贪分方法 ]
─────────────────────────
打死第1关BOSS后，她会变成气球，此时对着气球射击可
得不少分数，每枪5000分，直到气球打爆为止。

[ 好结局条件 ]¶

[ 好结局条件 ]
─────────────────────────
将游戏中被困的六个同伴救出，破关是HAPPY END。
