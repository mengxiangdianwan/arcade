spikeout 锐击 (C) Sega 1998¶

锐击 (C) Sega 1998

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：spikeofe,spikeout

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────
Ａ：移动锁定(没敌人时可作出水平移动，有敌人时会自动锁定最近的敌人)
Ｂ：拳(linda的是腿)  
Ｃ：腿(linda的是拳)/蓄力  
Ｄ：跳(共分2段，轻按是小跳，长按是大跳)
开始键：大小地图切换

[ 通用技 ]¶

[ 通用技 ]
───────────────────
蓄力技说明：共分4级。Ｃ1是普通腿(linda的是拳)；
Ｃ2级是升龙技；Ｃ3级是眩晕技；Ｃ4级是吹飞技

○捡取        ：↓＋Ｂ＋Ｃ (ＢorＣ是使用武器攻击)
○投掷道具    ：有武器时Ｂ＋Ｃ
○扫腿        ：Ｂ＋Ｃ(收招前再多按一次Ｂ可以使出追加攻击)
○后扫腿      ：按住Ｄ，背对的方向键＋Ｃ
○小跳浮空攻击：Ｃ＋Ｄ
○小跳下攻击  ：Ｂ＋Ｄ

○冲刺    ：同一方向键连按2次
○冲锋攻击：冲刺中，Ｂ＋Ｄ
○飞踢攻击：冲刺中，Ｃ＋Ｄ

☆无敌技    ：Ｂ＋Ｃ＋Ｄ 
☆合作无敌技：玩家互相抓住后Ｂ＋Ｃ＋Ｄ
☆挑衅攻击  ：顺时针一圈＋Ｂ＋Ｃ＋Ｄ 
(攻击范围窄，无敌时间长，中途可移动)

★Thank you动作：顺时针一圈＋Ｂ (特定部位有攻击判定)
★Sorry动作    ：逆时针一圈＋Ｂ (特定部位有攻击判定)
★Help动作     ：红血状态下顺时针一圈＋Ｂ＋Ｃ (特定部位有攻击判定)

[ 基本连技 ]¶

[ 基本连技 ]
───────────────────
○ＢＢＢＢＢＢ
○ＢＢＢＣＣＣ
○ＢＢＢ>Ｃ2
○ＢＢＢ>Ｃ3
○ＢＢＢ>Ｃ4
(伪连技，即有机会被敌人反击或截击，
由于常战中常用到及ＢＢＢ后可取消蓄力技，所以列入)
○ＢＢＢ>角色背对方向＋Ｂ
(被前后夹击时用的招式，但不稳定，
很多时候扫腿收招的一刻都被之前ＢＢＢ打中的敌人反击，
若后方没有敌人，前方敌人是复数，则会直接扫前面的敌人。)

[ 抓投技 ]¶

[ 抓投技 ]
───────────────────
○抓打击：正面抓着时Ｂ (可以输入2次，第3次就是倒地攻击)
○绕背  ：正面抓着时Ｃ (从角色正面绕至后面抓着敌人)
○普通投：正面抓着时方向键＋Ｂ
○侧投  ：侧面抓着时方向键＋Ｂ
○背投  ：背面抓着时方向键＋Ｂ
○空投  ：背面抓着时Ｄ，方向键＋ＢorＣ
☆旋风投：正面抓住敌人时Ｂ＋Ｃ＋Ｄ
