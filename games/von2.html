von2 电脑战机2 (C) Sega 1998¶

电脑战机2 (C) Sega 1998

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：von2

[ 基本操作 ]¶

[ 基本操作 ]
─────────────────────────
准星右边的数字代表敌人和我方的距离
该游戏是双操纵杆设计：

◎摇杆
·摇杆１
  ↖↑↗
  ←Ｎ→
  ↙↓↘

·摇杆２
  ↖↑↗
  ←Ｎ→
  ↙↓↘

      摇杆１←＋摇杆２→：跳跃
跳跃中摇杆１←＋摇杆２→：二段跳
飞行中摇杆１→＋摇杆２←：跳跃取消
近敌时摇杆１→＋摇杆２←：防御
      摇杆１↑＋摇杆２↓：顺时针转向
      摇杆１↓＋摇杆２↑：逆时针转向

◎按键
            左扳机：左武器
            右扳机：右武器
        左＋右扳机：中武器
        左or右按钮：冲刺(需配合摇杆使用、空中可)
冲刺中发动反向冲刺：冲刺取消
      冲刺中左扳机：冲刺左武器攻击
      冲刺中右扳机：冲刺右武器攻击
  冲刺中左＋右扳机：冲刺中武器攻击
摇杆１→＋摇杆２←＋任意攻击键：下蹲射击

面朝敌人时，画面中央会出现一个准星，
武器攻击会附带追踪性能。
靠近敌人一定距离时，三种武器会陆续变为黄色，
只要有一种武器变为黄色，就表示进入黄色武器
的近敌模式。近战模式下，标记敌人的准星也会
变为双准星。
近敌模式下武器伤害更高，但也可发动防御技能。
防御住攻击，只会损失很少体力。

◎角色特殊技
[ Temjin ]
○摇杆１↑＋摇杆２↓＋中武器：顺时针旋风剑
○摇杆１↓＋摇杆２↑＋中武器：逆时针旋风剑
○空中前冲中＋中武器：飞行剑
○快速摇动左操纵杆＋右扳机连打：机枪连射

[ Grys-vok ]
○下蹲射击＋开始键：重磅炸弹
  速度很慢、每回合限定一次

[ Raiden ]
○快速摇动左操纵杆＋右扳机连打：机枪连射
○二段跳中开始键：致命速度
  损失90%体力换来机动性的极大提升，每回合限定一次

[ Bal-Bados ]
中武器把所有部件发射出去后，武器能量100%时，
○站立中，右冲刺键＋中武器：黑洞(吸收远程攻击)
○下蹲中，右冲刺键＋中武器：金字塔陷阱(限制敌人移动)
○跳跃中，右冲刺键＋中武器：反射激光

[ Angelan ]
○下蹲射击＋开始键：天使之翼
  15秒内速度提升、损失一半体力、
  近战攻击有吸血效果

[ Specineff ]
○站立中开始键·开始键：死亡模式
  13秒内无敌，之后自爆、每回合限定一次
○快速摇动右操纵杆＋右扳机连打：机枪连射
○摇杆１↑＋摇杆２↓＋左武器：鬼火
○摇杆１↓＋摇杆２↑＋左武器：鬼火
○前冲中，蹲＋中武器：滑行攻击

[ Apharmd B/S ]
○中武器能量满时，空中前冲中＋中武器：空袭
○二段跳中＋中武器：性能强化
  每回合限定一次

[ Cypher ]
○三种武器能量均满时，空中前冲中＋中武器：S.L.C
○二段跳中开始键：喷气战机
○快速摇动左操纵杆＋右扳机连打：无限飞刀

[ Fei-Yen Kn ]
体力低于50%时自动进入金色暴走模式
武器威力提升

[ Dordray ]
○站立中开始键：巨大化(每回合限定一次)
○中武器能量满时，摇杆１↑＋摇杆２↓＋中武器：回旋钻头
