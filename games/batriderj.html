[ Rank系统 ]¶

[ Rank系统 ]
───────────────────
Rank系统即根据玩家表现而自动调整难度的
系统，决定了敌机进攻性、弹速、子弹数、
坚固度、道具下落速度。

rank值增加方法：射击、吃各种道具(通过
大型火力道具升到同样等级增加的rank值少
)、使用保险

[ 僚机特殊攻击形态 ]¶

[ 僚机特殊攻击形态 ]
───────────────────
仅对来自《空战之路》的机体有效。当同时
满足两个及以上攻击形态的条件时，优先顺
序为：
搜索型--尾随型--跟踪型--环绕型--范围型


○范围型
  不吃5个小型主火力道具，然后吃一个僚
  机道具。僚机将变成大范围型。

○搜索型
  不吃5个僚机道具，然后吃一个僚机道具
  僚机将变成自动瞄准型。

○跟踪型
  不吃5个保险道具，然后吃一个僚机道具
  僚机将变成自动跟踪型。

○尾随型
  不吃5个加分勋章道具，然后吃一个僚机
  道具。僚机将尾随玩家机体。

○环绕型
  不吃5个大型主火力道具，然后吃一个僚
  机道具。僚机将围绕机体旋转并不断向前
  射击。

[ 奖命/主副火力升级特殊方法 ]¶

[ 奖命/主副火力升级特殊方法 ]
───────────────────
[ 奖命特殊方法 ]
默认每150W分会奖命，日版是出现奖命道具
，其它版本是直接奖命。但还有三种特殊方
法：
1)不吃三个大型保险道具。
2)不吃三个奖命道具，吃第4个奖命道具会
激发特别奖励，玩家命数将补满。
3)Highway关，在蓝色气垫船停下来后摧毁
它。蓝色气垫船起初在高速路上出现，不要
摧毁它，让它行驶到尽头，进入建筑物里。
待进入建筑物内部后，道路尽头就停着蓝色
气垫船。

[ 主火力升级特殊方法 ]
不吃三个主火力升级道具，然后再吃一个主
火力升级道具。主火力会升至最高。
需要注意，在准备升至第5阶段之前，自机
的当前升级状态最多只可以升到第3阶段。
如果你已经升到了第4阶段，那么你是无论
如何都升不到特殊的第5阶段了。

[ 辅火力升级特殊方法 ]
不吃三个副火力升级道具，然后再吃一个副
火力升级道具。副火力会升至最高。前提条
件也是子机的当前升级状态最多只可以升到
第3阶段，升到第4阶段不可。

取得特殊升级时机体附近会出现字体提示，
而且旁白语音也会说出“SPECIAL POWER UP
！”。

[ 自机简介 ]¶

[ 自机简介 ]
───────────────────
游戏背景：为了对付那些法律无法制裁的大
奸大恶，政府就招揽了一些犯罪者作为特工
组织起来，并将其命名为ZERO COP（零号警
队）

○前警员队
[Ｄ．Ｄ (FLYING KILLER)]
年龄：２３
罪行：偷贩毒品以及斡旋（冤案）
速度：５
僚机：广范围炮
保险：超级重力
评析：性能标准，没有明显缺点，相当实用
的主角机。主火力攻击范围比较宽广，而子
机则是向两边发散攻击。这种配置在攻关过
程中可说是颇有优势的。爆弹的攻击范围和
持续时间都属一般。移动速度属于偏快，适
合进行Ｃ强化（主火力+子机）总之，无论
单打双打都是可以发挥得不错的机体。

[ＳＨＯＲＴＹ (Carry Carry)]
年龄：２２
罪行：偷贩毒品以及斡旋（冤案）
速度：３
僚机：跟踪导弹
保险：雷电之刃
评析：综合性能非常优秀的机体！主火力攻
击范围适中，而子机的跟踪导弹发射迅速，
可以对敌人进行全方位的追击。爆弹的攻击
范围表面上看似乎有些狭窄，但胜在放出后
机体可以自由移动而且持续时间长。如果是
双打时，那么放出前最好提醒一下同伴。
进行Ｃ强化后则毫无疑问成为队伍里的主力
机体。

[ＪＹＵＪＩ (Sangerea 5280)]
年龄：２８
罪行：警局内大打出手，导致轻重伤者多名
速度：８

保险：火焰凤凰
评析：主火力攻击范围偏窄，而且子机的攻
击还是属于直线型。虽然移动速度较快，但
就算经过Ｃ强化在中后期的关卡里还是很容
易陷入应接不暇的境地。另外，爆弹的攻击
范围一般且固定。火凤凰前进的速度也不算
快，如果遇到一些移动迅捷，体积较小的敌
人，那么还很可能会让他们躲避过去，有时
候如果被迫在版边放出就更是可惜。单打时
比较难用，而双打时作用有所提高，因为有
同伴分担版面，而且集中型的火力也利于攻
坚。

贴近敌人攻击很快就造成巨大的伤害。爆弹
的性能也很不错，持续时间一般，打击范围
宽广，就算双打时也能轻松保护到同伴。

[咒术师ＢＯＲＮＮＡＭ (GOLGODIAN)]
速度：４
僚机：地狱亡魂
保险：恶兆
评析：主火力打击范围宽广，子机的攻击属
于追踪型。按理说是很棒的配置，但是此机
的追踪攻击却有点特别，就是无法打击后方
的敌人，因此比不上其他有追踪攻击的机体
。爆弹的打击范围和持续时间都是一般，总
体上性能还是不错的。

[ＣＡＲＰＥＴ (MAGIC CARPET)]
速度：６
僚机：小女巫
保险：阿拉伯跳跃
评析：主火力打击范围比较狭窄，再加上移
动速度不算很快，因此最好还是进行Ａ强化
。子机会尾随主机移动，非常特别，而且其
攻击还带有穿透性。另外，如果蓄力放出后
，主机保持不动，那么子机也会停留在该位
置。有时候可以利用这一点让子机分担版面
。爆弹的持续时间较长，而且快要结束时，
自机会慢慢变小。这时候我们就可以从容地
移动到安全位置。总的来说，是既好用又有
趣的角色。

[ 隐藏飞机Galaxy Valhallyzer ]¶

[ 隐藏飞机Galaxy Valhallyzer ]
───────────────────
这架飞机来自魔法大作战，是Gain满足特定
条件后变形的机体。

不吃奖命道具(日版才有奖命道具设定)，然
后再吃一个僚机道具。玩家机体就会变成
Galaxy Valhallyzer。

[ 隐藏BOSS ]¶

[ 隐藏BOSS ]
───────────────────
蝙蝠骑警许多BOSS都来自Raizing之前的游
戏，且大多数都是隐藏的。在特定关满足特
定条件即可对战隐藏BOSS。除了第一关，其
余关都有至少一个隐藏BOSS。

  关卡名：  BOSS名
○Sky High：Bashinet 
摧毁关卡中段出现的所有大型蓝色三角翼飞
机(共7架)。
在摧毁BOSS Boredom后，队伍中至少有一个
来自魔法大作战的活着的角色。


○Sewage Sytem：Black Heart
摧毁关卡前段那一整排的大型镭射翻板炮台
(共5座)，这些炮台只有在开启时才能被攻
击到。
击毁BOSS Deviate时，队伍中至少有一个来
自空战之路的活着的角色。


○Zenovia Airport：Gob-Robo
摧毁喷气式客机中段大型炮台的两端辅助火
力(即带红竖道的部分)。
队伍中至少有一个来自魔法大作战的活着的
角色。
 

○Highway：Bashinet MkII
第1关，先摧毁画面中间大型广告牌后的大
房子，再把大型广告牌击倒，会看到飞起一
大群鸟。
在清完第4关后，队伍中至少有一个来自魔
法大作战的活着的角色。

○Highway：Tsumuji-Maru、Hayate-Maru
若两个BOSS的条件都符合，会两个BOSS都打
1)Tsumuji-Maru
三名角色必须副火力的范围性状不同。
队伍中至少有一个来自魔法大作战的活着的
角色。
2)Hayate-Maru
三名角色必须副火力的火力属性不同。
队伍中至少有一个来自魔法大作战的活着的
角色。

○Highway：Sobut、Blunt、Envy
第2关决定了这一关要对战的是哪一个随机
BOSS。比如，若第二关是Sewage System，
则本关将对战Sobut。
对应关系如下：
Sobut (绿色) - Sewage System 
Blunt (黄色) - Sky High 
Envy  (红色) - Zenovia Airport
这三个BOSS，玩家默认会见到至少一个。
若想见更多BOSS，必须队伍中至少有一个活
着的蝙蝠骑警独有角色，并满足以下条件：
第2关和第3关BOSS - 
使用过主火力升级特殊方法
第2关和第4关BOSS - 
使用过辅火力升级特殊方法
2、3、4三关BOSS  - 
使用过主火力升级特殊方法和辅火力升级特
殊方法


○Zenovia City：Black Heart MkII
三名角色必须拥有至少两种不同的机体类型
(主火力型、副火力型、速度型等)。
Highway关清完时，队伍中至少有一个来自
空战之路的活着的角色。


○最终关：Bashinet-R & Glow Squid
若两个BOSS的条件都符合，则两个BOSS都打
1)Bashinet-R
Zenovia City关，关卡中段，摧毁3架地下
导弹发射器。当发射器在地下时，只有保险
能打到。
摧毁BOSS Grubby后，队伍中至少有一个来
自魔法大作战的活着的角色。
满足以上条件会在最终关前，见Bashinet-R
2)Glow Squid
Zenovia City关，摧毁四座大型旋转炮台
(即那种一圈小炮旋转发射的组合炮台。关
卡前段两座放紫色小子弹的，关底一座放紫
色小子弹，一座放绿镭射)。
摧毁BOSS Grubby后，队伍中至少有一个来自
空战之路的活着的角色。
满足以上条件进最终关前，见Glow Squid。

[ 关卡隐藏加分道具 ]¶

[ 关卡隐藏加分道具 ]
───────────────────
○第1关
1)开头的大洞里会驶过火车，放保险摧毁它
可打出大量勋章。
2)关底，画面左边有一些警车。如果让BOSS
把他们毁掉，会出现金色加分勋章。
不需要留太多警车，不然很容易漏吃勋章。

○Sewage System关
1)在会向下翻转隐藏的镭射炮台的场景处，
当炮台翻转隐藏起来时，放保险摧毁他们
2)放保险摧毁会驶出坦克的绿色四方建筑物。

○Airport关
放保险摧毁电梯

除了以上较大分值的勋章，其它分值较小的
隐藏勋章都藏在只有放保险才能打出的建筑
物内。
比如：第1关的玻璃房顶；Airport关的飞机
尾部、Sewage System街道

[ 勋章系统 ]¶

[ 勋章系统 ]
───────────────────
连续吃勋章道具(金色的那种道具)时，每个
道具的分数会逐渐递增，最高增加到1W分/
个。如果漏吃了打出的勋章道具，下一个道
具又从100开始计算。部分场景需要用保险
才能破坏，打出勋章，比如
Zenovia Airport的喷气式客机尾部。

[ 版本区别 ]¶

[ 版本区别 ]
───────────────────
1998版开启了之前版本隐藏的BOSS战模式、
隐藏飞机，关卡编辑。
日版到奖命分数是出现奖命道具，其它版本
是直接奖命。
改变子弹颜色和子弹速度、显示分数，这三
个隐藏功能只在日版可用。
