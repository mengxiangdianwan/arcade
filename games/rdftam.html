[ 勋章系统 ]¶

[ 勋章系统 ]
────────────────
○银色勋章：10-90分。
  没有slave(副机)时可获得。
○蓝色勋章：100-900分。
  有一架slave(副机)时可获得。 
○绿色勋章：1000-9000分。
  有两架slave(副机)时可获得。 
○金色勋章：10000-100000分。
  和slave(副机)无关。
收集完银色、蓝色、绿色勋章后，
才会出现金色勋章。

勋章的分数增加：连续吃同样的勋章
，每个勋章的分值不会改变。摧毁一
个敌人，吃一个勋章；摧毁一个敌人
，吃一个勋章，如此循环，勋章分值
才会递增。

金色勋章模式：当画面上同时出现9
枚金色勋章时，若他们碰到一起会爆
炸，之后会进入金色勋章模式，该模
式下才能增加金色勋章分值。金色勋
章模式中，如果打出的勋章有漏吃的
，则勋章分会重新从10000开始。

[ 擦弹加分 ]
────────────────

[ 僚机攻击方式 ]¶

[ 僚机攻击方式 ]
────────────────
当获得两架僚机后，再吃一架僚机会
改变副机的蓄力攻击方式。
剩余生命数和B的数量之和决定了攻
击方式类型。比如剩余生命数0与B的
数量3之和为3，则再吃一个僚机，蓄
力攻击方式会变为3。

攻击方式类型：
0、4、8  [前置两向火力射击]
1、5、9  [定点射击]
2、6、10 [扩散火力扫射]
3、7、11 [跟踪攻击]

[ 隐藏飞机 ]¶

[ 隐藏飞机 ]
────────────────
○Judge Spear
飞机选择画面游标移动至Chaser，
按住↓到？处，再按ＡorＢ。

○Raiden MKII
飞机选择画面游标先移动至Aegis或
Beast Arrow(不要移动至Chaser)，
再移动至Endeavor，
按住→到？处，再按ＡorＢ。

○Slave
飞机选择画面游标移动至任一飞机，
如果当前飞机是上一行的，就按住↑
；如果是下一行的，就按住↓，
再同时按Ａ、Ｂ和开始键。
Slave的速度由选择使用秘笈的飞机
决定。

[ 关卡选择 ]¶

[ 关卡选择 ]
────────────────
一般情况下，第１、２关和第４、５
关是随机的，但可以使用特殊方法确
定具体的关卡。
投币后，同时按住开始键和一个方向键进
入飞机选择画面，当确定飞机后，关
卡也就确定了。

开始键＋↑…森林→基地、冰洋→列车
开始键＋↓…森林→基地、列车→冰洋
开始键＋→…基地→森林、冰洋→列车
开始键＋←…基地→森林、列车→冰洋

[ 隐藏的Miclus／小精灵 ]¶

[ 隐藏的Miclus／小精灵 ]
────────────────
Miclus：Miclus出现后，迅速攻击他
会爆许多勋章。
森林：经过水面后，停在最右边的那
栋东西向的房子上方。
森林：BOSS前会出现一架中型敌机，
不要急着摧毁它，等他的尾翼也飞过
BOSS的主炮后再摧毁。Miclus从左方
出现。

基地：先摧毁左边的油桶堆，再摧毁
右边的油桶堆。Miclus会从右边停靠
的飞机下出现。
基地：经过油桶堆后的画面右方，斜
方向停放的飞机的尾部。

海洋：两辆气垫船中间的海边。

冰洋：刚进入冰洋的画面，独立于海
中的一大块冰与右边冰块之间的水面
(那块水面的右边靠近冰块的位置还
有一小块冰)。

火车站：停在画面右边一栋南北向的
棕色房子的最下边。 
火车站：停在画面左边一栋东西向的
棕色房子的最右边。

最终关：游戏中段有一辆巨型坦克车
，在他之前的画面右边那个炮台。
最终关：经过巨型坦克车后，画面中
有五个炮台的画面会出现一架中型战
机，等他飞到和后一个画面上两座炮
台同一水平线再摧毁它。Miclus从左
边那个炮台下出现。

小精灵：发现小精灵后，等她放出
BOMB，再决定是吃掉(100000分)或杀
掉(10分)。
森林：在连续8架中型敌机射击前，
击毁他们(获得 "Quick Shot"评价)
，右边的树丛会飞出一个小精灵。
森林：BOSS战处，保护房屋不被BOSS
炮击，共要保护四所房屋。

沙漠：摧毁中型坦克的中央炮口，
再等待几秒钟。

海洋：BOSS战处，第一次见军舰最下
方的那门巨型加农炮时就摧毁他。
海洋：BOSS战处，第一次见两口紧挨
着的向右射击的加农炮时就摧毁他们。
冰洋：刚进冰洋见到的两艘潜艇，先
摧毁两端，再摧毁中间。后一艘潜艇
会爆出小精灵。

天空：游戏刚开始时连续击毁那5架
同样的中型敌机(获得 "Quick Shot
"评价)。云层段过去后，会出现
小精灵。

最终关：该关共4辆中型坦克，先摧
毁主炮，等几秒，放出小精灵。

[ 特殊分 ]¶

[ 特殊分 ]
────────────────
海洋：关卡开始的海滩有两艘气垫船
。在两辆小坦克进到左边的气垫船里

[ 加快倒数９９９ ]¶

[ 加快倒数９９９ ]
────────────────
首次载入游戏时，画面会显示倒数
９９９秒。
按一次Ｆ１０键可加快倒数速度，
倒数结束时再按一次Ｆ１０恢复正常
速度，最后按Ｆ３重新载入即可，
以后就不需要倒数９９９了。
如删除nvram文件夹里同rom名文件，
则载入又需要一次倒数９９９。

[ 关于Miclus ]¶

[ 关于Miclus ]
────────────────
Miclus是Seibu的吉祥物，样子是一条
蓝色小飞龙，在公司多款游戏中作为
BOSS、道具、角色出现过。
最早出场于1985年的Wiz的终BOSS，
在雷电战机2和jet中作为可选角色登
场，也是雷电系列和雷电战机系列的
加分道具，仅在日本本土发行过毛绒
玩具。
