[ 武器系统 ]¶

[ 武器系统 ]
──────────────────
主火力：吃小或大子弹道具升级。
每级所需道具数量，1级(默认)、2级需1
个、3级需2个、4级需4个、5级需8个

保险：吃40个小炸弹换取一个保险

僚机：吃小飞机道具得到僚机。
僚机有5种攻击形态，按Ｃ可调整。
攻击形态改变顺序为：散射、向后、向前
、旋转、自我调整。
当满足特定条件时，僚机攻击形态会有其
它变化。

[ Rank系统 ]¶

[ Rank系统 ]
──────────────────
Rank系统即根据玩家表现而自动调整难度
的系统，决定了敌机进攻性、弹速、子弹
数、坚固度、道具下落速度。

本游戏的rank值受以下因素影响：
玩家射击出的弹药数和弹药类型、吃道具
、使用保险、进入敌人射击死角攻击。

弹药数和弹药类型对rank值的作用说明：
主火力中间部分 +３
主火力其它部分 +１
僚机火力       +２

每次使用保险和吃道具都会使rank值增加。

当贴近敌人攻击时（对于绝大多数地面敌
人，则是停在正上方），就会使贴近的敌
人不攻击你，就相当于进入敌人死角射击
区。这种方法也会增加rank值。但随着难
度的提升，敌人的死角射击区也越来越小。

当rank值达到一定数量即会提升难度。死
命能明显降低rank值。死命时，残留的命
数越少，对于rank值降低的幅度越大。
Rank难度第1级的数值在1W分，第2级在5W
，……。
当难度提升后，就无法降低了难度级别了。

[ 僚机特殊攻击形态 ]¶

[ 僚机特殊攻击形态 ]
──────────────────
○搜索型
  不吃5个僚机道具，然后再吃一个僚机
  道具。

○尾随型
  不吃5个加分勋章道具，然后吃一个僚
  机道具。

○跟踪型
  不吃5个小炸弹，然后吃一个僚机道具。

○范围型
  不吃5个小型的主火力升级道具，然后
  吃一个僚机道具。

○连锁型
  只在双人模式下可用。一名玩家必须满
  足一种僚机特殊攻击形态的条件，此时
  有50%几率发动。

如果同时满足以上几种模式的条件，
则按以下优先顺序确定攻击形态：
搜索型、尾随型、跟踪型、范围型

[ 游戏秘笈 ]¶

[ 游戏秘笈 ]
──────────────────
○机体选择
　在飞机选择画面
　1)正常火力型：按Ａ选飞机
　2)速度提升型：按Ｂ选飞机
　3)中弹判定减小型：按Ｃ选飞机
　4)速度提升&中弹判定减小型：
　　按Ａ＋Ｂ＋Ｃ选飞机

○开启隐藏飞机
  投币后，↑(2)↓(2)←→←→ＡＢＣ开始键

  扩展模式：投币后，按住Ａ，再按开始键 
            (Type-2版和新版本中无效)

  困难模式：投币后，按住Ｂ，再按开始键 
            (Type-2版和新版本中无效)

  关卡编辑：投币后，按住Ｃ，再按开始键
            (Type-2版和新版本中无效)

○特别模式
  (扩展/困难/关卡编辑三种模式的合集)
  投币后，按住Ａ＋Ｂ＋Ｃ，再按开始键 
  (type-2版中无效)

○提示
  破坏桥梁和建筑，他们经常隐藏有加分
  道具。

[ 隐藏的1-UP ]¶

[ 隐藏的1-UP ]
──────────────────
第3关中段会见到一辆棕色的大型坦克，
当他移动到终点时再摧毁它，它会把终点
炸出1-UP。
该坦克中央一座炮台、两侧各有一座炮台
和火箭发射器。当这些火力单元收起来时
是无法被攻击到的。摧毁火力单元后，才
能攻击到主体。必须先摧毁两侧的火力单
元，最后摧毁大型坦克才会炸出奖命道具
。

[ 勋章系统 ]¶

[ 勋章系统 ]
──────────────────
连续吃勋章道具(金色的那种道具)时，每
个道具的分数会逐渐递增，最高增加到1W
分/个。如果漏吃了打出的勋章道具，下
一个道具又从100开始计算。

[ 版本区别 ]¶

[ 版本区别 ]
──────────────────
   游戏的版本可以在标题画面中看到，
而不同的版本也有一点差异。
VER. Sat Feb 3 1996：
最初的版本，主要在日本、美国、欧洲和
亚洲地区发行，而且该版本还有三个隐藏
模式。
VER. Sat Mar 2 1996：
第二个版本，主要在澳大利亚和香港发行
，该版本没有隐藏模式。
VER. Tue Apr 2 1996：
第三个版本，主要在丹麦和中国发行，同
样没有隐藏模式。
1945二代：
D版商出的版本，标题画面中还可以看到
“CHINA VERSION”的字样，游戏经过大幅
修改，Ｃ键不再是原来改变副机的攻击方
式，变成了更换飞机，飞机一出来就是最
大火力，而且每架飞机还能承受3次中弹，
局数和BOSS也作出了修改，每100W奖机一
次，贪分成为了这个游戏的乐趣。
