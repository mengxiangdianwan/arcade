  Ａ：吐泡泡

[ 隐藏要素 ]¶

[ 隐藏要素 ]
────────────────────
★ ORIGINAL GAME
正常情况下，中途不死命，才能在20/30关
/40关可进入秘密房间“银之门”。
输入密码后则无需这个条件，20/30/40关
必定可进入秘密房间“银之门”。
50关会出现可传送至70关的“金之门”。

投币前，标题画面输入：ＡＢＡＢＡＢ→开始键

关于银之门内，画面下方的字符含义：
画面下方由一横线分割成上下两部分字符。
横线上方的26个字符由左至右代表A-Z。
横线下方的字符则是一句话，每个字符可以
对照横线上方字符所代表的字母拼成一句话。
三句话的核心就是要你双人合作通关。

★ POWER UP！
初始就获得连射/速射/加速的3大效果、
即使死命，效果也不会消失。

投币前，标题画面输入：←Ｂ←开始键←Ａ←开始键

★ SUPER BUBBLE BOBBLE
标题会变化，多出一个SUPER模式。
SUPER模式下关卡颜色以及敌兵出现的顺序都
做了变动。

投币前，标题画面输入：开始键ＢＡ←→Ｂ开始键→

[ 结局简介 ]¶

[ 结局简介 ]
────────────────────
共三种结局：

结局１条件：通关时只存活一名玩家。
BOSS关上面有两个气泡，打败BOSS后，两个气
泡破掉，下来两个人。但是只有一人能与玩家
结合。游戏虽然看上去结束了，但是其实游戏
还没有结束。滚动字幕过去后，游戏从第85关
重新开始。

结局２条件：通关时两名玩家都存活。
此时，上面的两人下来后，各自与一只小恐龙
结合，小恐龙变回人形。结合后画面上显示
Happy End!同时出现滚动字幕。
滚动字幕过去后，画面上显示
But it was not a true ending!
同时下方出现了一行密码，这行密码就是通向
完美结局的钥匙。

结局３条件（完美结局）：
投币前，标题画面输入SUPER模式密码。
SUPER模式中，双人通关即可看完美结局。
BOSS关(必须是双人)，BOSS被气泡包住后弄破
后不会像前两种情况那样直接消失，而是变成
一只大恐龙，随后变成两个在一起的人。上方
的气泡随后破掉，两人下来与小恐龙结合，小
恐龙变成人形。结合完毕后显示Happy End!
