mmaulers 究极战队 (C) Konami 1993¶

究极战队 (C) Konami 1993

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：mmaulers

[ 基本操作 ]¶

[ 基本操作 ]
────────────────────
·摇杆
    防御：输入与敌人相反的方向
    跳跃：↑
    下蹲：↓
    滑铲：←←or→→
  三角跳：空中版边输入反方向
　　　　·高位置输入反方向则弹跳距离近
　　　　·低位置输入反方向则弹跳距离远

·按键
  轻攻击：Ａ
  中攻击：Ｂ
  重攻击：Ｃ
普通投技：地面or空中近敌正面时ＢorＣ
        ·对于大体型敌人，地面抓住敌人时
          需连打ＢorＣ才能投技成功。
  合体技：两玩家站在一起，同时输入相同合
体技指令。合体技类型由双方共同输入的指令
决定。

[ 指令符号说明 ]¶

[ 指令符号说明 ]
────────────────────
●：特殊投技
○：特殊技
◎：必杀技
☆：合体技

[ 隐藏结局 ]¶

[ 隐藏结局 ]
────────────────────
１币通关即可看到真结局。
显示工作人员表时是反派打主角的片段；
最后是邪恶三人组的特写。
续币通关显示的是主角打反派的片段；
最后是主角三人的特写。

[ 版本区别 ]¶

[ 版本区别 ]
────────────────────
日版(JAA)第一关能听到主题歌，而在欧版
(EAA)是最后一关。日版(JAA)每场战斗前都会
用日语报出'战斗场景'名字。Eagle的国籍在
日版(JAA)中是不详，而在欧版(EAA)中是美国
。
