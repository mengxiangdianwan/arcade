msword 魔法剑 (C) Capcom 1990¶

魔法剑 (C) Capcom 1990

枫枫的游戏机 粉丝专属街机游戏秘籍 街机游戏名称代码：msword

[ 基本操作 ]¶

[ 基本操作 ]
──────────────────────────────
·方向
　←→：左右移动
    ↓：下蹲
    ↑：开门

·按键
　　Ａ：攻击
        短时不攻击会蓄满能量槽，此时武器闪光可发动强化攻击
　　Ｂ：跳跃
Ａ＋Ｂ：保险(放保险有瞬间无敌，且期间的攻击有暴击)

[ 盾的作用 ]¶

[ 盾的作用 ]
──────────────────────────────
装备盾(游戏初始的盾不算在内)后，正面比背面防御高。玩家站着时
，正面对上路和中路远端物理攻击免疫。比如蜥蜴人站着放飞刀无法
伤害到玩家，但蹲着放飞刀时，最下面那把飞刀对玩家有效(此时，
玩家蹲着才会不受伤害)。不同的盾的耐久度不同。当盾牌只能再使
用一次的时候，花边会变成橘红色。
盾牌共4级，使用次数分别是2，3，4，5。换盾牌时，得分等于原盾
牌的剩余使用次数×万。

[ 牢房锁的含义 ]¶

[ 牢房锁的含义 ]
──────────────────────────────
金锁门关的是巨人、法师和蜥蜴；银锁门关的是小偷、牧师和忍者；
粉红锁关的是女战士、骑士和陷阱。牢房的门一般是随机的，但哪
一层关的囚犯是不会变的(33层和48层的牢房和囚犯都是固定的)。

[ 装备道具说明 ]¶

[ 装备道具说明 ]
──────────────────────────────
1) 王冠：提高防御力
2) 护手：增加攻击力
3) 金色罐子：加快助手回血速度
4) 水晶球：探测隐藏宝箱，隐藏宝箱会一闪一闪
5) 项链：杀死敌人和吃加分道具时，得分加倍
6) 魔法药(蓝色小瓶)：加快魔法槽积蓄速度
7) 钻戒：招降蜥蜴人和盗贼
招降蜥蜴人的方法参见特殊道具“铃铛”。
招降盗贼的方法：捡到了钻戒后就要想想随后在什么地方会有宝箱，
而且这宝箱还要装有加分或者加血的道具才行，装着钥匙和魔杖的也
不可以。戴着钻戒打开宝箱，里面的好东西露出来后就要静等一会，
不久盗贼就会从天而降来偷东西，当他接触到宝物后立刻打他一下。
有没有钻戒的区别就会显现出来了。有钻戒的话，盗贼挨打后就会投
降在原地不动，而且还会掉出一个随机道具给你。
8) 卷轴：带来好运，大幅提高敌人掉道具的几率
9)火焰石：提升终极武器的蓄力攻击威力

[ 特殊道具说明 ]¶

[ 特殊道具说明 ]
──────────────────────────────
1)铃铛：召唤一只敌对的蜥蜴人。如果你装备了钻戒，打败他后，会
  跪下加入你
2)镞石(银白色像钻石的道具)：召唤一只小精灵，带来三个补血道具
  鸡腿(一格血)、葡萄(一格血)、牛奶(半格血)。追打一下小精灵，
  可以多掉落一个绿苹果(半格血)。
3)剪刀：获得60000分；取消和当前助手的契约(助手会消失)。
4)红心：得3000分，助手升一级
5)蓝心：得100分，助手补血一格
6)黑心：得8000分，助手降一级
7)酒罐：得500分，助手血补满
8)魔杖：60秒内增加蓄力攻击射程以及威力。
9)圣杯：当前场景5秒内本体无敌并秒杀近身敌人。
10)魔法书：自动放保险一次，秒杀当前画面任何敌人，不耗血。
  被杀死的敌人只会爆一个900分的金币，不会爆其它道具
  (例外：空中飞的法师召唤的石头；亡灵巫师召唤的骷髅头；
  也不影响石头人死后躯干碎块爆出的加分道具的大小，但头颅受此
  规律影响只爆900分金币)。
  使用保险(Ａ＋Ｂ)杀死敌人也是以上情况。
11)毒蘑菇：5秒内玩家本体无法攻击。
12)草药：得2060分，解除中毒状态。

[ 助手说明 ]¶

[ 助手说明 ]
──────────────────────────────
若牢房中救出的囚犯与当前助手相同，采用其当助手，助手升两级。
1) 巨人：不能下蹲攻击。回血速度最快，攻击力强但攻击速度最慢
2) 忍者：攻击方式为扔出能在墙壁和天花板间反弹的镖。
   回血速度中，攻击速度快，范围大但威力小。
3) 亚马逊女战士：用弩持续射击。回血速度慢，攻速快但威力不大
4) 盗贼：能发现隐藏宝箱和陷阱(陷阱是红色一闪一闪)。
   回血速度中，攻击速度快，炸弹攻击距离近但威力大。
5) 骑士：回血速度快，攻击速度快、攻击带穿透效果。
6) 牧师：回血速度中，攻击需蓄力、蓄力满时攻击会触发２秒护盾
         圣光弹能对不死系生物造成双倍伤害。
7) 魔法师：攻击力最强，回血速度中，攻击需蓄力。
8) 蜥蜴人：有钻戒才会加入。回血速度中，攻击速度快，攻击力强

[ 隐藏门 ]¶

[ 隐藏门 ]
──────────────────────────────
第1层  -  在黄蜂出现的第一层阶梯的左边往左上跳起打。
          通向第4层。
第7层  – 进门后，向左走并攻击柱子。
          通向11层。
第14层 – 在左边上楼梯的位置，2层楼梯上面那层，蹲下往右边打
          通向19层。
第17层 – 攻击最后一个拱门的左边。
          通向20层。
第21层 – 在关着骑士的牢房和这一层的出口之间，
          在台子上向左跳并攻击。通向24层
第28层 – 蹲在第一个酸液池上方的平台上，攻击中间。
          另一条29-32层路线
第34层 – 跳起攻击第四个悬在空中的柱子的左边。
          通向36层
第39层 – 经过第一个火坑后，攻击遇到的第一个门左边突出的角。
          通向42层
第40层 – 蹲在第一个刺坑前的平台上的最右边，攻击。
          另一条41-46路线

[ 隐藏道具 ]¶

[ 隐藏道具 ]
──────────────────────────────
蓝宝石：第14层，到第2个门型雕像边，雕像上方像个人头。
跳起攻击雕像上方的“眼睛”几次，会掉出两个各值8万分的蓝宝石。
钻石  ：第49层画面最左边尖刺中的一个隐藏宝箱
五角星：4格补血道具。取得地点在第48层上方火焰池的最左边一个
        隐藏宝箱内。
