[ 系统介绍 ]¶

[ 系统介绍 ]
─────────────────
[ 骷髅槽系统 ]
  相当于怒首领蜂系列的连击槽设定，
决定了杀敌得分倍率。短时间内，连续
攻击敌人，画面顶端的骷髅会增加。当
有5个骷髅时，骷髅槽就会闪烁金光。
骷髅槽闪烁金光时，普通攻击(非式神)
击杀敌人和破坏坛子得到的金币值更大。

[ 金币系统 ]
  普通攻击(非式神)击杀敌人可打出金
币。式神攻击杀死敌人时若骷髅槽是满
的，会爆出大量金币。每枚金币都会增
加1到画面顶端的金币计数内，但攻击
结束后，金币总数会消失。下一次再吃
到金币，又会从1开始算起。

[ 1000模式 ]
  当金币计数达到1000时，只要骷髅槽
在闪烁金光，则普通攻击(非式神)攻击
任何敌人都会产生水流一般的金币。

[ 关于自机 ]¶

[ 关于自机 ]
─────────────────
[  鬼使   & 式神 力王]
  正常射击火力略分为两股。

[ 柊小雨  & 式神 八飞车(八尾妖狐)]
　前方集中型火力。 
　移动速度是三个角色中最快的。

[贺茂源助 & 式神 麒麟丸]
　前方广范围射击。

[ 隐藏补血道具 ]¶

[ 隐藏补血道具 ]
─────────────────
1)第3关
满血道具：当画面移动由水准变为垂直
时，右下角有四个雕像。其中一尊(比
较肥)和其它三个不同，用式神先摧毁
这个特殊的雕像，再摧毁其它雕像。
2)第4关
满血道具：用式神击杀第4关小BOSS
(屋里伸出的蜗牛头似的怪)，只要杀死
的那一击是式神造成的即可。
3)第6关
50%血道具：在画面水准移动时，一面
石墙后有一些蜘蛛。用式神杀死第4个
出现的蜘蛛。
满血道具：50%血道具的场景，石墙里
最后一个出现的蜘蛛，用式神攻击才能
打出。
满血道具：第一辆牛拉的车。
满血道具：BOSS第一形态时，左边的绿
色火焰，用式神攻击才能打出。

[ 隐藏BOMB ]¶

[ 隐藏BOMB ]
─────────────────
第5关，BOSS战前的那个大房子正中的
装饰物，用式神攻击才能打出。
第6关：BOSS第一形态时，右边的绿色
火焰，用式神攻击才能打出。

[ 箱子怪 ]¶

[ 箱子怪 ]
─────────────────
第3关，右下角有四个雕像的场景之后
，画面左上方会从一个箱子里弹出一个
女人头的妖怪。
用式神杀死她，会立即把骷髅槽蓄满。

[ 部分隐藏金币的坛子 ]¶

[ 部分隐藏金币的坛子 ]
─────────────────
只有本体走过藏坛子的地点，坛子才会
出现。

第1关
5个：第一座桥前面的右下角(河边的石
     灯前)。
3个：过桥之后，桥的右下角。
3个：过桥后，紧挨着画面左边樱花树
     的右上方。

第2关
2个：第一座桥的前面最右边以及后面
     最右边。
6个：第二座桥上第一座石灯后，沿着
     桥栏杆一线。
3个：过第二座桥后，画面最左边，也
     即蜘蛛场景入口的左边。
13个：每棵趴着蜘蛛的树的前面岩石
      一圈。

第3关
2个：关卡开头，画面右边大石头的上
     部边缘。
2个：枯树上方，画面最左边那里。
6个：上台阶后，画面右下角。
8个：右下角有四个雕像的场景，画面
     左上方的房子前和右边靠下。
2个：画面右边竹林前。

[ 版本区别 ]¶

[ 版本区别 ]
─────────────────
特别版(又称青版)，连击限制上限突破
1000；式神减速后的子弹脱离减速区域
后仍然比原始子弹速度稍微慢一点。

原版进入机台设置需按住Ｆ２不放；
特别版则是按住机台服务键不放，
预设是主键盘区的９
