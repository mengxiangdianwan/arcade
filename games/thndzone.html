[ 武器简介 ]¶

[ 武器简介 ]
─────────────────────────
●轻机枪
连射速度快，按住射击键可自动连射。
威力低、攻击范围窄。

●重机枪
连射速度快，威力比轻机枪高、攻击范围略大。

●火箭筒
单发威力大、连射速度慢。

●跟踪导弹
跟踪性能、威力低、连射速度慢。

●散弹枪
广角射击、射程近。

●火焰喷射器
攻击力很高，按住射击键可自动连射，攻击距离极近。

●反坦克步枪
连射时威力较弱，不射击时会自动蓄力。
随着蓄力时间的增加，威力也增加。
最大蓄力时，威力极强。

[ 协力设定 ]¶

[ 协力设定 ]
─────────────────────────
    人物血槽有蓝黄红三色，当血槽变红时，移动速度降
至最低。
    此时如果有同伴走到身边，红血的人会暂时放弃行动
控制权，改由同伴搀扶协助战斗，但搀扶人的速度会降低
。被搀扶者按Ｂ是脱离搀扶。

[ 疯狂得炸弹BUG ]¶

[ 疯狂得炸弹BUG ]
─────────────────────────
    整体游戏全部打小兵得到的炸弹最多只有8个左右，
可利用bug使每个小兵身上掉下的炸弹数达到3倍甚至6倍。
    步骤如下：首先知道哪一个小兵打后会掉下炸弹，然
后尽量靠近他，靠近后，开枪马上站在他的位置上，务求
他一开始掉下炸弹就接触到炸弹，换句说话就是他掉下的
炸弹一瞬间接触的话就开始计入你得到的。炸弹在他身上
出现的一瞬间你已经接触到了，然后站着别动，等炸弹从
空中落至地面瞬间，你就往上跑开，(注：我一般都往上跑
，大家可以试试往其它方向)，然后你就会发觉炸弹数目不
断转动了，最后会有3个甚至更多。
    得到N倍补血品的原理也是一样的。

[ 武器发射速度提高 ]¶

[ 武器发射速度提高 ]
─────────────────────────
    选用带红围巾那位，将按钮调至连发，手持散弹枪按
着连发不放去重复地拿散弹枪，如此重复几次，散弹枪的
连发速度就会快几倍，打第二关开始不久的六个导弹发射
台很轻松。前提是自己不能中弹，否则恢复起初。其它武
器提高速度的原理一样。

[ 小游戏介绍 ]¶

[ 小游戏介绍 ]
─────────────────────────
第一关后：
    爬铁丝网时两键快速按可以加速人物爬行，但只能持
续加一段且加速后要休息一会才能再加速，一开始先加速
爬一点但不要把电脑拉远了，掌握控制距离差，差距太大
他们也会加速，不容易追赶，当终点在望时可突然加速，
第一名十拿九稳！
    比赛过铁丝网，第一名可获3颗雷，第二名2颗，第三
名1颗，第四名0颗。
    4P合作时可以不用争抢第一名，倒计时60秒后会每人
加3颗雷。
    如果和CPU合作，可以在开始时往上拉住摇杆不要按键
，靠在通道上方会拖住CPU所有角色，倒计时60秒过后也是
每人加3颗雷。

第三关后：
    注意千万不要伤到美女。如果不小心误伤的话系统会
自动锁定准星数秒作为惩罚。
    比赛结果第一名可获3颗雷，第二名2颗，第三名1颗，
第四名0颗。
    注意：如果4P中没有CPU，队员们可以不费一枪一弹，
大家订立攻守同盟0分过关，赛后相同的分数当然同等待
遇，每人加雷3颗。

[ 版本区别 ]¶

[ 版本区别 ]
─────────────────────────
日版主要在画面细节上有许多和其它版本不同之处：
1、第一关击毁BOSS后直接过关，而其它版本之后还有进入
房间救人质的内容。
2、演示动画最后有一只鹰飞过四名主角。
3、第二关场景变为冰山，其实只是颜色变为蓝白色了。
   破坏拦路的大冰块可打出补血道具。
4、第二关后的过场剧情动画背景变为冰山。
5、敌弹少，弹速慢。
