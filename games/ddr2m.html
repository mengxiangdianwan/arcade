[ 隐藏要素 ]¶

[ 隐藏要素 ]
─────────────────────────
[ 专家模式 ]
↑↓↑↓↑↓后选择技能等级。
该模式下一半的箭头都会消失

[ 另一个隐藏模式 ]
歌曲选择画面，↓↓↓←←→→

[ 较简单模式 ]
歌曲选择画面←↓→↓←↓→↓↑

[ 疯狂模式 ]
↓↓↓→↑↑←or↓↓←←←→→→←→，
再选择技能等级。歌曲选择界面↑(8)可取消该模式。

[ 使用Socket Man ]
按住左选择键，再按开始键

[ 曳步舞模式 ]
歌曲选择界面输入↑↓←→↓↑→←

[ 开启2P模式 ]
按1P开始键开始，再按住2P的左选择键和右选择键，按2P开始键

[ 舞者面朝右 ]
歌曲选择界面输入→(8)

[ 舞者面朝左 ]
歌曲选择界面输入←(8)

[ 全歌曲 ]
游标选中Hard mode(困难模式)，再按右选择键4下
