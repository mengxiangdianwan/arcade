[ 自机选择以及差别 ]¶

[ 自机选择以及差别 ]
─────────────────
Type A：：移动速度较快，射击面窄
Type B：移动速度慢，射击面宽
选定飞机之后还有DOLL选择 　
　
散弹强化SHOTIA：散弹射击范围宽，
自机移动速度慢，初始有3个BOMB，
最大有6个BOMB 　　

激光强化LEINIAN：激光射击不会被小
怪阻挡(秒杀小怪)，初始有2个BOMB，
最大有4个BOMB 　　

双强化EXY：散弹与激光均强化，
初始有1个BOMB，最大有2个BOMB

[ 判定点 ]¶

[ 判定点 ]
─────────────────
子弹只有击中飞机正中心才算做中弹。
这个点大约在机舱下方。

[ 连击 ]¶

[ 连击 ]
─────────────────
打敌人时连击数和连击槽都会增加，
连击槽会随时间减少，减完则连击数从
0重新计算，连击数与得分有直接关系。
用激光击打大型敌人时连击槽会维持在
最低限度。

[ Hyper ]¶

[ Hyper ]
─────────────────
    左上角有hyper槽，若处在hyper未
发动的状态，则hyper槽满了以后屏幕
上会掉出hyper道具。最多可以攒5个。
BOSS战时，如果你还有BOMB，则不会掉
出hyper道具。吃了hyper道具之后按Ｂ
键可以发动hyper。发动以后敌人弹速
变快，自机火力变强，连击槽减少速度
降低，打敌人时获得的连击数增加。
Hyper发动时连击槽会回满，并且有短
暂的无敌时间。

[ 蜂道具 ]¶

[ 蜂道具 ]
─────────────────
　　每关有10个，位置固定，需要用激
光头击打(激光条穿过道具)才会出现。
吃了以后会得到道具分数*当前连击数
的分，并且HYPER槽会增加，增加量和
当前Hit数有关。如果处在HYPER发动状
态，则吃道具时HYPER槽不增加。如果
某一关吃齐10个，最后一个蜜蜂分数X2
，并且以后的蜜蜂基本分数＋1000。
    白版增加量：大约是200Hit增加
1/4，每个道具最多增加1/4。如果当前
Hyper槽超过3/4，吃了1/4后，会得到
一个hyper道具和一个完全空的新hyper
槽。黑版增加量：大约是200Hit增加1/2。

[ 二周目 ]¶

[ 二周目 ]
─────────────────
不续关且以下三个条件达成任意一个即
可进入2周目：
1、死命少于2次 　　
2、BOMB使用少于2次 　　
3、5关中有3关吃齐10个蜂道具
(从关卡开始到吃完最后一个过程中不
能死命) 
　　
    2周目子弹数量和密度变多，连击
槽下降速度降低。最后有真BOSS绯蜂，
BOMB可以无敌消弹但不能伤绯蜂的血，
难度极大。

[ Rank与MAXMIUM ]¶

[ Rank与MAXMIUM ]
─────────────────
Rank：本作有这样一个设定，如果你一
直没有死命或者Bomb，那么弹速会越来
越快。 
MAXMIUM：如果Bomb满的情况下吃到
Bomb道具会进入MAXMIUM状态，此时分
数会自动增加，积攒的Bomb道具越多，
增加得越快。

[ 奖命道具1-UP ]¶

[ 奖命道具1-UP ]
─────────────────
第4关：关卡中段的第一个巨炮，其两
边各有一个小点的炮。先摧毁小炮，
再摧毁大炮。
