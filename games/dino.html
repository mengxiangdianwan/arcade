[ 角色特长 ]¶

[ 角色特长 ]
──────────────────────────────
杰克：奔跑中Ａ的攻击能有效击中正面敌人，比如头撞过来的恐龙。
汉娜：射击速度最快、匕首可连击三下、奔跑跳Ａ的攻击时间长、
      最大的优势是Ａ＋Ｂ时间最短，收招特别快。可以故意不打到
      敌人，而充分利用大招结束之后的1秒无敌时间去攻打敌人。
      比如幻影刀手BOSS。
穆斯塔法：攻击距离最远、流星踢可前跳或后跳中发动且攻击范围更
      大。
迈斯：奔跑跳跃的速度最快但枪械射击速度最慢、投技的无敌时间长
      、抱摔距离近所以可以连打投解决敌人，比如第2关BOSS。

[ 隐藏8万分 ]¶

[ 隐藏8万分 ]
──────────────────────────────
击杀从版边滚出来放置炸药的黄衣小个子(俗称猴子)且该敌人的尸体
在画面中消失，会获得8万分。若尸体是被打出画面消失的是不给8万
分的。
出现该敌人的地点：

第３关：不要上车。打完第一关BOSS样子的敌人后（默认难度下出现
3个），就会从右边出现4个黄衣小个子。

第４关：BOSS处，不要太快击杀BOSS，过一段时间，版面两边会各出
现1个。

第５关：蜥蜴人至第一关BOSS这一段内，共4个。

[ 奔跑出拳 ]¶

[ 奔跑出拳 ]
──────────────────────────────
任意角色和杰克站在一起触发合作无敌，必须是杰克把玩家推出去这
一种合作无敌方式。在被杰克推出去奔跑的过程中，按住→不放就可
保持奔跑状态，该状态下按Ａ是出拳攻击，而不是常规的奔跑攻击。

[ 无敌ＢＵＧ ]¶

[ 无敌ＢＵＧ ]
──────────────────────────────
当分数足够奖命，但是奖命动作还没有做出来之前，持续跑动，不要
停，一直到奖命动作出现，角色就变成无敌状态。
放ＡＢ、抓住敌人扔出去、捡道具、切换场景，无敌状态就解除。
打死BOSS后如果是无敌状态，会卡一会儿才能过关，所以最好在BOSS
死之前故意解除无敌。
