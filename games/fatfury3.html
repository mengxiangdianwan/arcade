[ BOSS使用方法 ]¶

[ BOSS使用方法 ]
───────────────────────
○一般方法
  家用模式下，先开启记忆卡功能。
  游戏中ＡＢＣＤ四键同时按，可进入
  MEMORY CARD UTILITY(记忆卡管理)。
  或者Tab菜单中的记忆卡选项载入记忆卡，
  一个游戏可以创建 0-1000号 编号的记忆卡文件。
  创建记忆卡后会在MAME模拟器目录下生成一个叫
  memcard的文件夹。

  不接关的情况下直至打败秦氏兄弟完美爆机后，
  保存数据。
  再次开始游戏会问你是否载入游戏进度
  (DATA LOAD OK？)，选YES。

  之后选择NEW GAME

  角色选择画面，依次在特瑞、红虎、不知火舞、
  基斯、鲍伯、望月双角、安迪、佛朗哥、东丈、
  玛莉处各按一次Ｂ，按的时候会听到特殊音效。

  成功的话，会出现HERE COME NEW FIGHTERS字样，
  之后画面变黑。等画面再亮起来时，就多了三位
  BOSS头像可选。

○调试模式方法
  调试模式(DEBUG DIP SETTINGS)中将DIP2的第5项
  设为1。角色选择界面即可选择。

[ 隐藏要素 ]¶

[ 隐藏要素 ]
───────────────────────
○Easy Command Mode
  超必杀技＆潜在能力指令简化模式。
  调试模式(DEBUG DIP SETTINGS)中将DIP2的第1项
  设为1。指令简化如下，后续追加技是无法简化的
  ，仍然要和正常模式一样输入。
  超必杀技↓↓＋Ａ(基本发动指令)
  潜在能力↓↓＋Ｃ(基本发动指令)

○使用潜在能力
  比赛前按住ＡＢＣＤ不放，直到出现“Go”时，
  按开始键。此时角色的名字会变为绿色，可以使用潜在
  能力了。

○加速模式
  角色选择时，按住开始键再选定角色。
  该模式攻击判定时间缩短。
  挑战方才可用该模式。

○专家模式
  操作说明画面←→←→Ｎ→←→←。
  若成功，原来显示难度的地方会出现EXPERT字样。
