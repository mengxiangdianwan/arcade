unsquad 战区88 (C) Capcom 1989¶

战区88 (C) Capcom 1989

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：unsquad

[ 基本操作 ]

  Ａ：射击

[ 关于自机 ]¶

[ 关于自机 ]
──────────────────────────────
风间·真
国籍：日本
座机：F-20 虎鲨
主火力：20毫米口径火神式机关炮
特性：“虎鲨”拥有高速度与破坏性的威力。
      几乎可以摧毁任何目标。
简介：他曾在所属部队的所有飞行员中展现过最高的射击命中率。
      座机F-20在对地对空的攻击中均表现优越。

米奇·西蒙
国籍：美国
座机：F-14 雄猫
主火力：20毫米口径火神式机关炮
特性：“雄猫”以空中格斗见长，
      拥有同时攻击一片广大范围的可能。
简介：前美国海军飞行员（据说越战后因患“战争症候群”无法和平
      过日子而继续当佣兵）。
      他能够以无以伦比的技巧随心所欲地操纵F-14进行空中格斗。

葛列格·盖茨
国籍：丹麦
座机：A-10 
主火力：“加特林”机关炮
特性：特长是对地攻击，其对地面倾泻的诸多弹药拥有极强的破坏力
简介：曾经在整个欧洲以帮助人质逃离为业谋生（据说是因为曾护送
      一对母女到达伦敦而遭到苏联特务的追杀）。

[ 故事背景 ]¶

[ 故事背景 ]
──────────────────────────────
    Kutal，一个位于中东的国家，多年来一直处于一种内战持续不
断的可怕状态。一个由军火商们聚集形成的组织“计画 4”随即出现
，以便他们最大限度地获取利润。
    当时这种军火利润非常可观，然而“计画 4”竟贪得无厌，很快
开始了在和平随时受到威胁的 Kutal 国内煽动战争的勾当。通过使
用他们的非法资金，他们已研发了很多革命性的武器装备，现在正企
图征服全世界。
    为了在他们的全部计画完成之前试图反抗这一威胁并恢复Kutal
长久以来所寻求的和平，一支国际化的特别战斗小队于那时组成了。
它就是 U.N.Squadron 航空兵小分队。

[ 双打初始更多钱 ]¶

[ 双打初始更多钱 ]
──────────────────────
同时开始双人游戏时，每个玩家会得到3000金。
如果第2个玩家是后加入的，他将获得12000金，
首先玩的玩家会得到9000金。

[ 隐藏道具 ]¶

[ 隐藏道具 ]
──────────────────────────────
吃到隐藏道具会有提示音。

第3关：
1)独角兽---关卡一开始立刻向左下飞，破坏关卡所有的树木，
  独角兽会在BOSS前出现。
2)弥七---要塞最后一座炮台右边。

第4关：
摩比君(1000 pts)---第5根石柱后面底部。

第5关：
摩比君(获得隐藏武器三向镭射)---破坏所有高高突出来的石柱，
BOSS战前最后一根石柱会爆出。

第6关：
摩比君(10000 pts)---路分叉为上下两层后，走上面一层，
第一根石柱的底部。

第8关：
摩比君(1000 pts)---关卡开始不久，左边会出现两艘相邻的船
(前面的一艘会加速驶出画面)，在这艘加速行驶的船的快驶出画面的
位置最上方。
弥七---BOSS第2座红色炮台

第9关：
摩比君(1000 pts)---第2栋建筑(履带输送喷火和导弹机器人的灰白
色建筑)后面。
