osman 奥斯曼战士 (C) Mitchell 1996¶

奥斯曼战士 (C) Mitchell 1996

$info=osman

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────────
Ａ：攻击   Ｂ：跳跃   Ｃ：保险

·方向指令
    ↓：下蹲
    ↑：攀上平台 (抓住天花板时)

·按键指令
    Ａ：攻击 (组合指令可打出乱舞技)
    Ｂ：跳跃／降落 (抓住天花板or墙壁上时)
空中近敌时Ｂ：空中投技
空中↑：抓住天花板
↓＋Ｂ：滑步
    ├滑步中→：奔跑
    ├滑步中Ａ：滑铲
    └滑步中近敌时Ｂ：投技
            ├投技中←or→：向左or右投
            └投技中↑    ：向上投
    Ｃ：麒麟星祭 (需要[Ｓ]、空中可)

[麒麟乱舞]
  四段连续技
  Ａ·←or↓or→＋Ａ·Ａ·←or→＋Ａ

[麒麟星祭]
  发动中完全无敌
  BOSS战中可打出高伤害
  (１～４段直线命中判定、５／６段范围攻击)

[ 道具说明 ]¶

[ 道具说明 ]
───────────────────────
绿色：恢复１格生命
蓝色：生命全恢复
黄色：增加生命上限(最大４格)
红色：提升攻击力(最高升４级)
受伤一次就会降一级攻击力
伤害由低到高的裤子颜色不同：蓝、紫、红、白、黑

[ ３-５关BOSS出场顺序 ]¶

[ ３-５关BOSS出场顺序 ]
───────────────────────
第3关中段和关底的路线选择决定了后面BOSS的出场
顺序。这两次路线选择的地点都是右上斜坡的场景。

上、上：Cannons → Tianon → Willf
上、下：Cannons → Willf  → Tianon

中、上：Willf → Cannons → Tianon
中、下：Willf → Tianon  → Cannons

下、上：Tianon → Willf   → Cannons
下、下：Tianon → Cannons → Willf

[ 背景设定 ]¶

[ 背景设定 ]
───────────────────────
战斗集团「狄」的战士·麒麟应联邦司法部长杰克·
雷森的邀请去夺回恐怖思想集团阿卜杜拉占据的都市
但在麒麟解放了这座都市后，因为惧怕麒麟的强大力
量，杰克和「狄」背弃了他，要赶尽杀绝。
在联邦政府、阿卜杜拉集团、战斗集团「狄」的三方
绞杀下，麒麟逆袭复仇。

麒麟：
主人公、体能非凡、一路赤手空拳、所向披靡。

阿卜杜拉：
恐怖思想集团阿卜杜拉信奉的神，也是最终BOSS。
拥有复活亡者的能力，最终关复活前三关的BOSS参战
以神自居、曾经诱惑麒麟臣服于他。

「狄」Cannons
为「狄」命名的人。
麒麟的长辈，爱好和平，
使用自制的机械兵器如臂指使。

「狄」Tianon
「狄」组织的战士。
华丽空中体术、使用斗篷发动空斩裂脚压制麒麟。
麒麟的恋人。

「狄」Willf
「狄」组织的战士。双手持铁爪挑战麒麟。
他的目标是击败「狄」组织其他战士，
自己成为最强兵器。
