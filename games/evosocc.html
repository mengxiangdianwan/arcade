[ 特殊技 ]¶

[ 特殊技 ]
─────────────────────────
特殊技消耗能量。
除了祈福和无敌，其它都只消耗一个能量。
特殊技的发动共四种条件：
控球时、球在空中(即传球中)、无球时、控制守门员时。

    祈福：无球时↑↑↑↑＋Ｄ
    无敌：控球时↑←→↓＋Ｄ
超级扑球：控制守门员时↑↓＋Ｄ
超级铲球：无球时←↙↓↘＋Ｄ

祈福：受伤和疲劳的队员状态全恢复。
无敌：当前队员获得防御罩、
      对方除守门员之外都无法攻击该球员。
超级扑球：守门员疲劳状态恢复、
          增加防御住特殊技的几率。
超级铲球：增加铲球的攻击距离。

[ 版本区别 ]¶

[ 版本区别 ]
─────────────────────────
预设为世界版，游戏无模式选择。
墨西哥版有国际和国内比赛模式选择。
切换为墨西哥版的方法如下：
机台设置(F2进入)-Evolution Soccer SOFT DIP SET UP-
-FOR MEXICO设置为YES
