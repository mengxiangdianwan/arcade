aof 龙虎之拳 (C) SNK 1992¶

龙虎之拳 (C) SNK 1992

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：aof

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────────
①操纵杆
  ↖↑↗  后跳/垂直跳/前跳
  ←Ｎ→  后退&站立防御/回中/前进
  ↙↓↘  下蹲&下蹲防御/下蹲/下蹲

  ←←后跃、→→前冲。

②按钮
  Ａ：轻拳(可破坏飞行道具)
  Ｂ：轻脚
  Ｃ：重攻击
  Ｄ：挑衅
ＡＣ：超重击(上段)
ＢＣ：超重击(下段)

ＡorＢ按住不放：蓄气

超必杀技：气力接近4/5时便可使用
(对CPU战刚开始，人物的气力槽并非全满状态)
隐藏必杀技：体力1/5一下，气力槽接近4/5时
便可使用。

[ 指令符号说明 ]¶

[ 指令符号说明 ]
───────────────────────
以1P角色为准(即游戏角色面向右)

●：投技
○：特殊技
◎：必杀技
☆：超必杀技(没修行成功就不能使用)
★：隐藏超必杀技(体力1/4以下、气力MAX)

[ BOSS角色使用方法 ]¶

[ BOSS角色使用方法 ]
───────────────────────
①比克
  选坂崎亮或者罗伯特对电脑的故事模式中，当遇到
  比克时，让另一人加入挑战，便可以选用。

②空手先生
  选坂崎亮或者罗伯特对电脑的故事模式中，当遇到
  空手先生时，让另一人加入挑战，便可以选用。

[ 迷你游戏一览 ]¶

[ 迷你游戏一览 ]
───────────────────────
  在第2、4、6关后会出现奖分关卡(BONUS GAME)，
而其中又共有3个迷你游戏供选择。

○切啤酒瓶
  这个考验的是玩家的反应能力。只需看准时机，在
黄色能量值达到或最接近MAX时按Ａ，便可一次性将
5个啤酒瓶的瓶口切掉。成功后气力值的上限会有所
上升，而气力值的上限最多可做3阶段提升。

○冰柱割
  这个考验的是玩家的按键速度。必须在限定时间内
快速连打Ａ，使黄色量值达到或最接近MAX才能通过
。成功后体力值的上限会有所上升，而体力值的上限
最多可做3阶段提升。

○超必杀传授
  在限定时间内输入一定次数的“霸王翔吼拳”指令
来成功发动“霸王翔吼拳”。要求成功的次数多寡取
决于游戏难度。成功后玩家便可在故事模式中发动超
必杀技“霸王翔吼拳”。
