skullfng 空牙外传 (C) Data East 1996¶

空牙外传 (C) Data East 1996

$info=skullfng

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────
  Ａ：射击

  Ｂ：射击(AUTO MODE) /
      飞行速度(CHASE MODE)
      飞行速度平时只有1-5级，快速连按
      Ｂ会出现6级(OVERBOOST状态，只能
      短时保持)。

  Ｃ：无敌翻滚(FIGHTER PILOT)
      保险(BOMBER PILOT)
      爆弹(吃了SUPER-UNIT道具后)

[ 特殊操作 ]¶

[ 特殊操作 ]
───────────────────
※选择“FIGHTER PILOT”类型的角色时，
可使用Thrust指令技

 翻转机身(向左)：←↓＋Ｃ (再按Ｃ恢复)

 翻转机身(向右)：→↓＋Ｃ (再按Ｃ恢复)

 翻转机身(左翻转至机头向后)：
           ↑↖←↙↓＋Ｃ (再按Ｃ恢复)

 翻转机身(右翻转至机头向后)：
           ↑↗→↘↓＋Ｃ (再按Ｃ恢复)

 强力翻滚攻击：←→or→←＋Ｃ

 高空翻转    ：↓↓＋Ｃ (短时无敌)

 360度旋转   ：↓↓↓↓↑↑↓＋Ｃ

[ BOSS战数位记号说明 ]¶

[ BOSS战数位记号说明 ]
───────────────────
BOSS战时，玩家飞机会被一圈刻度包围，
刻度上有一个红色标志，还有一个不断变化
的红色读数。刻度代表方向，红色标志表示
BOSS的位置，红色读数代表玩家与BOSS间的
距离。

[ 道具介绍 ]¶

[ 道具介绍 ]
───────────────────
主火力道具——攻击道具会改变道具火力类
型。除了V，其它火力都有按住Ａ与连按Ａ
两种不同形态。

V(Valcun Cannon) ：机炮
C(Cluster Bullet)：集束弹
L(Laser Gun)     ：镭射
M(Homing Missile)：跟踪导弹

Power(Power Up) ：升级火力
MNV(Maneuver Up)：提高移动速度
LIFE(Charge Up) ：补充SHIELD
S(SUPER-UNIT)   ：超级装备
拥有SUPER-UNIT时，若保持在6级飞行速度
，会变成H-UNIT(HYPER-UNIT)。

[ 版本区别 ]¶

[ 版本区别 ]
───────────────────
1、模式选择后的画面，世界版是大叔，每
关BOSS前有大叔出现并有英文语音；日版是
小mm，每关BOSS前有小mm出现并有日文语音
，每关开始和结束的时候显示几秒钟当前驾
驶员的头像并有日文语音，每被攻击三次就
会出现驾驶员的头像做发怒表情并有日文语
音。
2、世界版5条命，敌人攻击高；日版只有一
条命，敌人攻击低(默认设置下)。
3、机台设置中的GAME OPTIONS不同。日版
多了SHIELD、CHARGE、STAGE RECOVER三个
选项。
SHIELD是调整护盾强度的(EASY最强，
VERY HARD最弱)；CHARGE是调整蓄力速度
的(EASY最快，VERY HARD最慢)；STAGE 
RECOVER是调整过关后的护盾恢复量的
(EASY是全恢复，VERY HARD不恢复)。
