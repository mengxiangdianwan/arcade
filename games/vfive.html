vfive V-5战机 (C) Toaplan 1992¶

V-5战机 (C) Toaplan 1992

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：vfive

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────
Ａ射击  Ｂ升级

在没按住Ａ时，可以通过↑和↓调整辅机的
位置，当辅机在前方重合时，火力最大。

[ 升级项目表说明 ]¶

[ 升级项目表说明 ]
───────────────────
画面下方共6个数值，变成粉红色的是当前
可升级的项目，每吃一个升级道具会改变当
前可升级项目的位置。
S-UP：提升速度(最高升4次，之后图示就变
      成S-DOWN，即降低速度)
SHOT：普通火力(默认)
SEARCH：跟踪火力
MISSILE：导弹
POWER：加强火力(最高升3次)
SHIELD：防御罩(可挡住一次子弹，在防御
罩消失后的2秒内玩家飞机处于无敌状态)

[ 版本区别 ]¶

[ 版本区别 ]
───────────────────
日版名V-Five，美版名Grind Stormer
日版画面下方为待升级的属性，击毁送道具
的敌机得到的是不针对特定项目的升级道具
。美版画面下方为保险，击毁送道具的敌机
得到的是特定项目的升级道具和保险。

[ 秘笈 ]¶

[ 秘笈 ]
───────────────────
隐藏的无敌功能---如果开启机台设置中的
Invulnerability(mame plus中名字为耐久
力)，玩家无敌，可以按2P的开始键暂停游戏，
按1P的开始键重启游戏。
轰炸机模式(日版可用)---投币后，在标题
画面输入→←Ｂ←↓Ａ↑←。画面显示
'BOMBER. Ver' ，则成功，此时游戏模式
变为美版Grind Stormer的。
