spf2t 作者: たの¶

作者: たの

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：spf2t,spf2td

[ 基本操作 ]¶

[ 基本操作 ]
──────────────────────────────
·方向
　←→：左右移动
    ↓：加速下落

·按键
  Ａ：左回转
  Ｂ：右回转
  开始键：挑衅(Dan可无限挑衅，其余角色每回合限发动一次)

[ 系统说明 ]¶

[ 系统说明 ]
──────────────────────────────
[  规则  ]
胜利规则：对手宝石堆积到顶。
失败规则：我方宝石堆积到顶。

[场地介绍]
场地容量为：１３行Ｘ６列
每次下落两颗宝石。
右上角NEXT显示下一对要出现的宝石。
NEXT左边的小框显示的是对手攻击过来的计时宝石数量。
计时宝石数量框旁边的大框会在对手攻击过来时提示小心。

[宝石介绍]
普通宝石：宝石基本单位，外表为四方固体状态宝石。分红绿蓝黄
          四种。
爆炸宝石：消除接触到的颜色宝石的单位，外表为包裹一个内核的
          圆形宝石。
超级宝石：POWER GEM。大数量同颜色宝石堆放在一起形成的体积
          巨大的宝石。形成规则为2x2堆放。关底分数统计时会有
          加分奖励。
钻石宝石：用来消除场地中所有该颜色的宝石。
计时宝石：带有数位的宝石，当数位为0时会变为普通宝石。
          当对手攻击你时，会出现该宝石。
计时宝石样式：当你攻击对手时，计时宝石在对方场地出现的样式。
              每名角色都有自己的计时宝石攻击样式，选择角色
              时会显示，表示的是１３行Ｘ６列落下的宝石颜色
              分布。

[攻击方法]
   炸掉宝石会给对手场地掉落计时宝石。
1、使用爆炸宝石：
   爆炸宝石炸掉接触的同颜色宝石，同时也会炸掉相邻的同色宝石。
   爆炸宝石不能炸掉直接接触的计时宝石，但当相邻的同色宝石被
   消掉时，计时宝石宝石也会被炸掉。
   每炸掉1个普通宝石就给对手1个计时宝石；炸掉超级宝石造成的
   计时宝石数翻倍。
2、使用钻石宝石：
   钻石宝石会炸掉场地内所有该颜色宝石，但对敌人的计时宝石攻
   击数会降低一半。

[特别技巧]   
连锁爆炸：CHAIN。当炸掉一堆宝石导致另一堆宝石爆炸时，就触发
          了CHAIN。关底分数统计时会有加分奖励。
SUPER FINISH：３０个以上计时宝石掉落对手场中KO对手。

[ 隐藏角色 ]¶

[ 隐藏角色 ]
──────────────────────────────
[ 豪鬼 ]
按住开始键不要放手
1P将选人框移到蝙蝠女处输入←↓←↓←↓，
接着按住←不要放手，按ＡorＢ。
2P将选人框移到猫女处输入→↓→↓→↓，
接着按住→不要放手，按ＡorＢ。

[ 火引弹 ]
按住开始键不要放手，1P移到蝙蝠女处，2P移到猫女处，输入13次↓，
按ＡorＢ键。

[ Devilot ]
按住开始键不要放手，1P移到蝙蝠女处，2P移到猫女处，输入13次↓，
等选人时间达到刚好10秒时，按ＡorＢ。

[ 隐藏要素 ]¶

[ 隐藏要素 ]
──────────────────────────────
·选择颜色
 选择角色时，Ａ、Ｂ分别对应不同颜色

·角色乱入
 [Devilot乱入条件]
 没有续币和玩家挑战，第6战结束时达成以下条件，第7战即会乱入
  ○最大CHAIN大于等于４
  ○最大POWER GEM大于等于２０
  ○完成过SUPER FINISH
  ○有１局６０秒以内胜利

SUPER FINISH：３０个以上计时宝石掉落对手场中KO对手。
