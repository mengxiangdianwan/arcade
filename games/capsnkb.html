○防御崩坏
  格斗攻击都会降低隐藏防御耐久值。
  当防守状态濒临崩溃时，防御的效果会逐渐呈现
  红色，体力外围也会出现闪动的危险讯号。
  当超过角色负荷时，防御状态强制解除，短时
  不能防御。
 
○CAPCOM GROOVE系统
  可通过使用必杀技或拳脚攻击来增加能量。
  能量槽分三段，能量超过第一个等级后攻击力自
  动上升。可使用三种不同等级的超必杀技。
  即输入指令后，按轻攻击键为LV1超杀；
  重攻击键为LV2超杀；轻＋重攻击键为LV3超杀。
 
○SNK GROOVE系统
  可以通过按住重拳＋重脚不放来增加能量。
  能量蓄满时，能量槽自动转为时间槽倒数能量
  MAX的状态。
  在倒数能量MAX状态中，攻击力大幅增加，还可
  使用超杀。体力低于1/4体力槽闪动时，可无限
  数用超杀。若配合能量MAX状态可使出MAX超必杀

○GROOVE POINT系统
  对CPU战的实时评分系统。
  当角色做出攻击或防御的动作时，评分系统会对
  该角色所做的动作做出D到S等五个不同的评价。
  S最高。评分直接影响每场战斗后GROOVE POINT
  成绩。
 
○RATIO系统
  每位登场角色都有一个RATIO值(1-4)。
  该值可看做衡量角色势力的标准，值越高的角色
  攻击、防御的能力也越高。战斗结束后体力恢复
  的比例也是RATIO高的角色有优势。
  例如隆、肯、草薙京等主角系RATIO值为2；
  春日野樱、坂崎由莉等角色值为1；
  BOSS角色如拜森(日版维嘉)、基斯值为3；
  隐藏角色杀意隆、暴走八神、豪鬼的值为4。

  RATIO系统中每支队伍的RATIO总值不能超过4。

[ 隐藏要素 ]¶

[ 隐藏要素 ]
──────────────────────
○PAIR MATCH MODE
  解锁PAIR MATCH MODE后，投币后在游戏标题画
  面中，按住轻拳＋重拳不放，再按开始键。
  该模式下所有角色的RAITO统一为２点，即限死
  为２vs２的对战模式。

○隐藏颜色
  每名角色均有四种基本颜色(对应四个攻击键)，
  但在解锁隐藏角色后，在选定角色时输入特定组
  合键能选到另外四种隐藏颜色。

·同时按两个拳
·同时按两个脚
·轻拳＋轻脚
·重拳＋重脚

○选择胜利动作
  KO对手后，马上按住任一攻击键不放，成功的话
  可看到与该键对应的胜利动作。

○莫莉卡乱入
  解锁莫莉卡乱入后，达成以下三个条件：
  1、选择CAPCOM GROOVE
  2、打完第三场比赛时GROOVE POINT在60以上
  3、第一至第三场比赛时使用超必杀击倒对手
     ４次以上

○娜可露露乱入
  解锁娜可露露乱入后，达成以下三个条件：
  1、选择SNK GROOVE
  2、打完第三场比赛时GROOVE POINT在60以上
  3、第一至第三场比赛时使用超必杀击倒对手
     ４次以上

○豪鬼乱入
  解锁豪鬼乱入后，打完第五场比赛时GROOVE 
  POINT在85以上

○使用隐藏人物
  解锁隐藏人物后，在拥有EX角色的头像上按住开始键
  选定角色。
