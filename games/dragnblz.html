[ 基本操作 ]¶

[ 基本操作 ]

①操纵杆
  ↖↑↗

②按钮
Ａ：攻击(可蓄力)    
Ｂ：保险    
Ｃ：放飞龙坐骑
    放出的坐骑不会移动，但该状态下
    的蓄力攻击也会改变。
    Ｃ的那一下是打出隐藏爆分宝箱的
    关键。

与敌机碰撞会有一瞬的硬直，且火力降
低1级。

普通射击杀死敌人得到银币，而用Ｃ放
飞龙坐骑的那一下杀死敌人得到的是金
币，分值是银币的两倍。

[ Rank系统 ]¶

[ Rank系统 ]
─────────────────
Rank影响敌机弹速、坚固度，共16级。
吃火力道具时，rank提升2。
随着关卡数推进，rank也会提升。
死命后，rank会在1分半内明显降低。
降低火力等级也可降低rank。

[ Technical Bonus说明 ]¶

[ Technical Bonus说明 ]
─────────────────
每关的大BOSS都有一个会展露给玩家几
秒钟的核心弱点(红色的圆球)。
当他展露给玩家时，用龙的分体攻击他
可一击必杀，这样能得到
Technical Bonus(技术加分)。
不要急于重创每关的大BOSS，重创情况
下他不会展露核心，但要小心在展露核
心时，会放出大量子弹。

[ 金币玩偶 ]¶

[ 金币玩偶 ]
─────────────────
每关都有一个隐藏金币玩偶(第7关是两
个)。
要打出金币玩偶必须用龙分身(即Ｃ放
出飞龙的那一下)攻击特定的敌人。
每攻击一下金币玩偶就得到一金币。

天空：关卡刚开始不久一波棕色蝙蝠飞
过后，会出现几个手拿弓箭的紫色敌人
。攻击左边第2个出现的该敌人。

巨型花：关卡刚开始是一波波的3个一
组的花状敌人。在由右上方垂直下来的
第一组该敌人后有一个孤单的花状敌人
，攻击他。

海洋：关卡刚开始时，在出现骑着四个
黄色鱼手拿枪的绿色怪后，会出现一波
波绿色的鱼怪，攻击画面最右边(几乎就
是贴着最右边画面了)出现的那一条绿鱼
怪。

沙漠：见到第2个大型炮台后(绿色蜥蜴
怪也在右边出现了)，画面右上方会出
现一群碟形敌机，而其中只有一架一直
都未改变飞行方向垂直向下方飞。

关卡5：在有四个乌龟坦克的画面中(注
意此时能完整看到大门处的乌龟坦克)，
攻击右上方飞出的第2架碟形敌机。

关卡6：骷髅小BOSS后会出现一波波的
骷髅怪，攻击左边第一波骷髅怪之后
那一个孤零零出现的骷髅怪。

最终关：用龙分身攻击击杀三头蟹小
BOSS，会显现两个宝箱。
左边的头一个，右边的头一个。
