[ 勋章系统 ]¶

[ 勋章系统 ]
────────────────
○银色勋章：10-90分。
  没有slave(副机)时可获得。
○蓝色勋章：100-900分。
  有一架slave(副机)时可获得。 
○绿色勋章：1000-9000分。
  有两架slave(副机)时可获得。 
○金色勋章：10000-100000分。
  和slave(副机)无关。
收集完银色、蓝色、绿色勋章后，
才会出现金色勋章。

勋章的分数增加：连续吃同样的勋章
，每个勋章的分值不会改变。摧毁一
个敌人，吃一个勋章；摧毁一个敌人
，吃一个勋章，如此循环，勋章分值
才会递增。比如铁路关，攻击迎面驶
来的火车能迅速提高分值。  

金色勋章模式：当画面上同时出现9
枚金色勋章时，若他们碰到一起会爆
炸，之后会进入金色勋章模式，该模
式下才能增加金色勋章分值。金色勋
章模式中，如果打出的勋章有漏吃的
，则勋章分会重新从10000开始。

[ 擦弹加分 ]¶

[ 擦弹加分 ]
────────────────
自机中弹判定点周围擦弹时会有白色
火花和刺啦声，同时有加分。

[ 关卡选择 ]¶

[ 关卡选择 ]
────────────────
一般情况下，第１、２关和第４、５
关是随机的，但可以使用特殊方法确
定具体的关卡。
投币后，同时按住开始键和一个方向键进
入飞机选择画面，当确定飞机后，关
卡也就确定了。

开始键＋↑…基地→沼泽地、空→海
开始键＋↓…基地→沼泽地、海→空
开始键＋→…沼泽地→飞机场、空→海
开始键＋←…沼泽地→飞机场、海→空

[ 僚机攻击方式 ]¶

[ 僚机攻击方式 ]
────────────────
当获得两架僚机后，再吃一架僚机会
改变副机的蓄力攻击方式。
剩余生命数和Ｂ的数量之和决定了攻
击方式类型。比如剩余生命数0与Ｂ的
数量3之和为3，则再吃一个僚机，蓄
力攻击方式会变为3。

攻击方式类型：
0、4、8  [前置两向火力射击]
1、5、9  [定点射击]
2、6、10 [扩散火力扫射]
3、7、11 [跟踪攻击]

[ 隐藏的Miclus／小精灵 ]¶

[ 隐藏的Miclus／小精灵 ]
────────────────
Miclus：Miclus出现后，迅速攻击他
会爆许多勋章。
基地(3个)——
1)关卡刚开始在画面左边的房子底部
，停1秒。

2)基地内在画面右方停着上下两排小
飞机的地方(紧挨着小飞机的北边还
停着一架中型飞机)，不要攻击碾压
上一排飞机的坦克，也不要攻击从左
边开过来的卡车。当坦克和卡车相撞
爆炸后，Miclus会从北边停着的中型
飞机机腹下出来。

3)关底有一辆中型的坦克，他会攻击
他面前的3辆小型坦克，总共三次攻
击。用保险抵消攻击。若成功，画面
会3次提示Defended the tank。此时
一个miclus会从右边停着的轰炸机机
腹下出现。


沼泽地(3个)——
1)关卡刚开始时，摧毁第7辆坦克。

2)在第2架大型轰炸机(击毁后爆S道
具的)右边，有一个灰色小洞，停在
上面。

3)BOSS战前，有一个被击毁后会掉
BOMB道具的轰炸机。别急着击毁他
，在他要向右飞出画面的时候再击
毁。


铁路(6个)——
1)刚见到铁路的时候，画面右边有一
栋南北向的棕色的房子。停在房子的
最下方，会提示Miclus出现。

2)见BOSS前，画面右边有一个东西向
的银色房子。停在房子正上方，会提
示Miclus出现。

3)火车BOSS共四节，每一节都有一些
坦克。在所有坦克上都停一下，每一
节都有一个Miclus。


海滩(3个)——
1)关卡开头，停在深入海中的陆地和
小岛之间的那个地方。

2)海上的所有船只，先击毁两端，再
击毁中间。在海滩岸边出现Miclus。

3)BOSS前出现的两辆中型战斗坦克，
先击毁右边出来的，再击毁左边出来
的。在海滩岸边出现Miclus。


沙漠(2个)——
1)BOSS处，在标志2的地方，会升起
一辆坦克，停在坦克上方。

2)BOSS处，在标志1的地方，会升起
一辆坦克，停在坦克上方。


最终关(3个)——
1)关卡开始的时候，有一段的画面
推进非常快。在这段时期到尾声的
时候，会看到画面左边有一个孤零
零的三角形的房子。飞到房子的中
线上。(注意看画面提示
DISCOVERED THE MICLUS，Miclus
出来后仍然在阴影中看不到，但
能攻击到)

2)快到关底时会进入敌人的堡垒，
有一些成对出现的炮台。飞到第2
对出现的炮台的右边那一个上面。

3)快到关底时会进入敌人的堡垒，
有一些成对出现的炮台。飞到最
后一对出现的炮台的右边那一个
(在画面最右边，BOSS战前)上面。


小精灵：发现小精灵后，等她放出
BOMB，再决定是吃掉(100000分)或杀
掉(10分)。
基地——摧毁BOSS战前那个中型坦克
的主炮，稍等一会，他会放出精灵。

沼泽地——关卡中段，会连续出现8
架中型敌机，迅速击毁他们(获得 
"Quick Shot")，左边的树丛会飞出
一个小精灵。

铁路——BOSS有两节车体有若干小炮
台，用保险一下全部摧毁。若成功，
会看到3个Destroyed at a time。
一个小精灵会飞出来。

海滩——BOSS前出现的两辆中型战斗
坦克，先把小炮摧毁，等一会，他们
会放出小精灵。

天空——摧毁所有大型敌机，
BOSS战前会出现一些小精灵。

沙漠——BOSS上有两个靠的很近的巨
型加农炮台。第一次看到他们时，迅
速摧毁下方的那一个，比如扔几个保
险。

[ 隐藏模式 ]¶

[ 隐藏模式 ]
────────────────
游戏内部计时每天超过75分，就算作
一天。
游戏时间超过6周，会开启模式
[NORMAL]和[SPECIAL]
游戏时间超过10周，会开启模式
[NORMAL]、[SPECIAL]和[EXPERT]

[ 真BOSS出现条件 ]¶

[ 真BOSS出现条件 ]
────────────────
1命击破前6关，其中第3关和第6关目
标破坏率100%。(第7关后可续币)

[ 加快倒数９９９ ]¶

[ 加快倒数９９９ ]
────────────────
首次载入游戏时，画面会显示倒数
９９９秒。
按一次Ｆ１０键可加快倒数速度，
倒数结束时再按一次Ｆ１０恢复正常
速度，最后按Ｆ３重新载入即可，
以后就不需要倒数９９９了。
如删除nvram文件夹里同rom名文件，
则载入又需要一次倒数９９９。

[ 关于Miclus ]¶

[ 关于Miclus ]
────────────────
Miclus是Seibu的吉祥物，样子是一条
蓝色小飞龙，在公司多款游戏中作为
BOSS、道具、角色出现过。
最早出场于1985年的Wiz的终BOSS，
在雷电战机2和jet中作为可选角色登
场，也是雷电系列和雷电战机系列的
加分道具，仅在日本本土发行过毛绒
玩具。
