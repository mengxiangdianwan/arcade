(可以连3个循环)：

…………(打轻BOSS，其中的Ｂ  ↓↘→Ａ  可以换成→→Ａ等)
(追击 ↓↓Ａ  ←→Ａ  ＡＡ  ←→Ａ)*3
…………(打重量级BOSS)  
(追击 ↓↓Ａ  ←→Ａ  ↓↘→Ａ ↑↓Ａ)*3       
…………(爆气状态，打轻BOSS；↑↓Ａ可用←→Ａ代替 )

每个循环最好连完整若是只连4招，踩的那一脚就会算是最后
一式，需要再踩一脚才可以继续连，而一共是只能踩三脚的。

关于倒地追击的次数问题：
1，在一个连招里面最多只可以3次

2，如果某个BOSS本来可以进行3次追击 ，
你实际进行的追击次数是0 / 1 / 2 / 3
那么下一个BOSS可以追击的次数是3 / 2 / 1 / 3

3，当BOSS被你追击的次数达到极限，他每次倒地将会快速
起身无法追击；解除这种状况的方法是→→＋Ａ撞他一次
(你喜欢的话多撞几次也可以)，但是撞了不要连
再次找机会连招，你会发现，又可3次追击了

◎第三关夏侯惇出现的地方之前有三部投石车。不要把它们打烂，也不要向前走，
否则会引爆所有车。又可以无限贪分了。

◎第二关定军山上出蒙面探子的地方，只要不把滚石引下来就可以无限打探子贪分。

◎第二关干掉彻里吉进入升降台后走左边的路，干掉所有小兵后就可以无限打铁甲
兵。

◎诸葛亮爆气用→→＋Ａ可以无限连某些BOSS贪分，比如孙姬、彻里吉、夏侯惇。

◎黄忠的→＋Ａ射箭没有攻击力，但分数照加，利用这个BUG可无限贪分。
  注意：只适用于V119版本。

◎黄忠引发定军山斩夏侯的剧情后，只要不走到夏侯渊面前，夏侯渊就不会醒。
这样可以贪这里小兵的分。

◎在小兵濒死或已经死了的时候使用鞭尸技，比如诸葛亮的爆气超杀、赵云的
→＋Ａ和跳起↑＋Ａ、马超的跳起↑＋Ａ，打小兵的尸体。每打1HIT，系统就会认
定杀了一个小兵，在过关计算分数时就会多加杀一个小兵的分数。

[ 有趣的BUG ]¶

◎双人无限天书BUG

张陵剑、张角巾、神仙笔、太清丹经、招安状、傀儡娃娃、援军令)，然后1P使用
天书，2P同时使用特殊道具。成功的话，就会解除放天书时不能使用道具的限制。
接着马上使用雨书，成功的话1P的天书效果就不会消失。
  注意：雨书可以定住除地书之外的所有天书，火书可以被任何书定住。

    电书           否    否    否    否    是
    风书           否    否    否    否    是
    地书           是    是    是    是    是
    雨书           否    否    否    否    否

◎张辽无限天书BUG
  站在张辽出来的地方，清掉最后一批小兵。在张辽出来之前使用天书，张辽就会
用风卷残云将你打出天书，这样天书的效果就不会消失。
  注意：使用电书时的位置要比火书的位置稍微向左一点。

◎夏侯渊无限天书BUG
  双打以上，有黄忠在场，引发定军斩夏侯的剧情。将小兵清光后，走到夏侯渊前
面，在出现对话的瞬间使用天书。之后不要急着按Ａ，等黄忠刀劈夏侯渊，这样火
书的效果就不会消失。

◎卡吕蒙
  在吕蒙出来之前先选好任一特殊道具，只有张陵剑、张角巾、太清丹经、神仙笔
、傀儡娃娃有效。等吕蒙出现讲完话后，马上使用刚才选好的道具。成功的话，吕
蒙就会被困在升降台中出不来。

◎卡魏延
  放走魏延进入水道后，背景下方第2棵大树右边的场景(跳出来几个铁甲兵和刀兵
)，不要把左上方的水草打掉，此时把小兵都干掉后马上跑到画面的最左边(大树那
里)。等到GO字样出现后马上向右跑，来到画面最右边。等魏延出现后马上跑回画
面最左边，将魏延打进左边屏幕。站到画面左上方的草丛右边，成功的话就会看到
魏延被卡在草丛里了。

◎无限援军令
  首先玩家只剩下一条命，然后使用援军令，在魏延退场之前玩家被打死，魏延会
因为玩家死亡那个准备退场，这时马上接关出来，魏延就会回场继续援助，而且不
退场了，直到转换画面或过关为止。如果中途玩家被打死，魏延就会退场。

◎不打死左慈也可以过关
  第一次遇到沙摩柯时放走他，当沙摩柯在八阵图中再次出现时不要打死他，先将
陆逊干掉。等左慈出现后再将沙摩柯干掉，此时地上就会爆出沙摩柯和左慈的道具
，接着就计分过关了，但左慈还没死。
  注意：从幻境出来后快速将每个阵的小兵打死就会破阵，这样沙摩柯就不会再出
现。

◎定军山定书的BUG
  进入定军山，在出现弓箭手射箭的剧情之前的瞬间可以随意活动。此时如果使用
火书，弓箭手的箭就可以把你从火书中射出来，火书的效果就不会消失。向前走的
话，整个画面会变的通红，而且不能使用任何道具，放超必杀就会使画面恢复原状。

◎赵云恐怖的超必杀
  多人打时，赵云使用爆气超必杀倒转昆仑。在超必杀击中敌人时，其它人在赵云
身边爆气，这样超必杀的威力会大增，甚至可以秒杀BOSS。
  攻击力增加的幅度由爆气的人数决定。同理可用于关羽的亢龙有悔。

◎天书保命
  1、首先玩家1不停向前跑或一直蹲下或打开道具菜单不动，之后玩家2使用任意
一本天书，但玩家1不加入。当天书结束后，玩家1就会无敌。
  2、在TIME OVER的瞬间使用天书，人就不会死，但血会降低到0。单打时使用这
个BUG会死机。

◎陷害大法
  当小兵和玩家站在一起时，此时有弓箭打过来或BOSS攻击你，你马上防御，就会
发现身旁的小兵被击中了。

◎死而复生
  在雷道最后的场景中，如果刚好在可以跳下桥的时候死亡，而且尸体刚好倒在桥
的缺口，那么人就会复活跳下桥，但没有血。

◎沙摩柯不动了
  在八阵图中，如果沙摩柯出场时候还有水兵在场，沙摩柯就会定住不动。

◎场景BUG
  第一关孙姬出现的地方，玩家先站在船的最上面凹处，然后把小兵干掉，在孙姬
出现的瞬间向凹位的右边前跳一下。成功的话，人就会卡在船栏中。

◎无名火烧书
  身上有金匮要略或史记残页，然后捡无名火，就会发现金匮要略和史记残页不见
了。

◎貂蝉吃道具
  貂蝉蹲下捡自己不能捡的道具，就会发现该道具不见了。

◎赵云吃黄石公
  在第三关金匮要略密室前，赵云先将黄石公捡到密室门口，接着打烂灯台。
  赵云在进入密室时将黄石公吃掉。张飞、马超和黄忠也可使用此BUG。

◎天师符的BUG
  1、先使用天师符，然后将七星灯或金匮要略扔出来，这样天师符的冰就会消失。
  2、赵云先使用天师符，然后低空使用三分天下，这样天师符的火会跑到赵云的
脚下。
  3、魔法张飞低空使用↓↘→＋Ａ也会使天师符跑到脚下。

◎魔法张飞变身
  雷道最后的场景中，将小兵都干掉后，魔法张飞站在左上角用↑↓＋Ａ跳下桥就
会变成吕布。
  如果是多人打时，变的吕布可活动，但是按Ｂ、Ｄ会死机。

◎太清丹经变色BUG
  1、用太清丹经将路选变成动物。当他恢复原装时衣服会变成橙色，脸变成紫色
，用毒镖或毒筒打他会变回原来的颜色，但一解除中毒状态就会变回橙色。
  2、用同样的办法可以使孟优和熊变色。
  3、在智破八阵图这一关中，用太清丹经将水兵变成动物。当水兵恢复原状时就
会变色。在雷道中也可以。
※神仙笔中的“变”也有相同效果。

◎主角变色
  在第二关开始的两座升降台移动的时候，除张飞之外的任何角色接关或中途加入
，该角色就会变色。

◎兔子不走了
  第四关有兔子出现的地方，先将前面的四个刀兵干掉，然后站在画面的最上面
(挨着两个栅栏的竹子的正上方)或最下面(竹子左边对应上方两个栅栏中间的位置)
，兔子出现后就不会走。这时可以打烂木栅栏前进，把小兵干掉后再打兔子。

◎两把铁剑
  马超在八阵图幻境中打死吕布后捡电剑，接着在出幻境的瞬间切换剑，出来后会
发现马超身上多了一把铁剑。

◎胖子变色
  第三关落石的地方之前有两个推火箭车的胖子。此时不要向前走引爆火箭车，先
慢慢向前移动将两只老鹰引出来，干掉两只老鹰之后再干掉一批炸药兵，这时再向
前走就会发现胖子都变色了。

◎人不见了
  第一关开始，走下面的路线。干掉两批小兵后，人站在画面最左边不动，这时船
就会慢慢向前移动，而画面也会跟着向前移动，一直移动到有小兵出的地方才停下
。这时会发现人已经不见了，只剩下小兵在画面中。

◎诸葛亮的浮空BUG
  诸葛亮打小兵时用ＡＡ·←→＋Ａ(←→＋Ａ要出得很快)，诸葛亮就会空中使出
霹雳火。

◎貂蝉变色
  貂蝉爆气后，等到气槽快完的时候↓↑＋Ａ。成功的话，貂蝉的颜色会变成爆气
后的颜色。

◎补血错位
  第三关落石的地方有两个推火箭车的胖子。在双打或多人打的时候，利用变色
BUG使两个胖子变色，然后去撞下面那个胖子，就会发现前面弓箭手身上的小包子
从胖子身上撞出来了。同理，如果打陆逊的时候没有将他身上的包子撞出来，就可
在左慈身上装出来。

◎幽灵
  多人打的时候，一人开着道具菜单过版，其它人先不过版，站在原地等一会，就
会发现刚才已经过版的那人的道具菜单从画面中飘过，而且是不断循环的。

◎定军山道具错位BUG
  定军山BOSS前的台阶上丢出无名火，会发现无名火错位了，在地上留下了一个黑
影，也不能把它捡回来。如果在这里放天师符，冰火会出现在脚底。

◎道具菜单BUG
  一直按着Ｃ不放，然后摇动摇杆上下左右移动，就可以开着道具菜单随意走动了
。如果同时按↓＋Ｂ＋Ｃ，就可以蹲下开道具菜单，而且还可以随意走动。

◎文字错位
  张辽死命下来的时候迅速连打Ａ＋Ｂ，就会发现头顶上的“1P”跑到了脚下。

◎火人
  在貂蝉出场、小兵开始退场的瞬间使用炸弹将小兵炸倒，会发现小兵站起来继续
撤退而且身上带着火焰。如果小兵死了，会在地上留下一团火。
