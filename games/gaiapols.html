gaiapols 黄金鹰之剑 (C) Konami 1993¶

黄金鹰之剑 (C) Konami 1993

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：gaiapols

[ 基本操作 ]¶

[ 基本操作 ]
─────────────────
Ａ：攻击
Ｂ：随从
Ｃ：魔法
开始键：确定选项/跳过剧情

·玩家指令
防御    ：按住Ａ
奔跑攻击：→→or←←or↑↑or↓↓
回旋斩  ：
顺时针整圆 (妖精为逆时针整圆)。
只需按出上下左右四个方向即可。

·随从指令
有随从时，按Ｂ可在主动进攻(受攻击)
和跟随主人(不受攻击)状态间切换。
主动进攻可配合方向，8方向攻击。
跟随主人状态会自动恢复生命。
随从的攻击力比主人低，但攻击获得的
经验一样。

·关于魔法
道具为魔法宝石，魔法最高为７级。
由低到高依次为
雷电、冰狼(无敌时间最长)、黑洞、
魔法阵、圣光、火龙、黄金鹰。
１-５级时，只需要吃一颗魔法宝石，
即可升级魔法。６级时需吃３颗，
才能升级为７级的黄金鹰。

[ 水晶寄居蟹 ]¶

[ 水晶寄居蟹 ]
─────────────────
在时间个位数是0/4/8时发动魔法，一
段时间后水晶寄居蟹就会从地面出现。
寄居蟹会跟着玩家缓慢移动，长时间
不杀它就会钻回地下。

不包括关卡7(BLANCMUTE)在内之前每关
最多可启动两个，之后每关只能启动一
个。

不同的时间个位数、击杀寄居蟹时是否
有魔法石、不同血量、爆出的道具是不
同的。血量是2格以下时，会爆出补血
道具。

[ 无限经验 ]¶

[ 无限经验 ]
─────────────────
关卡5(MAHARISHI)：
开始不久，画面左侧有一个水果摊。随
后会出现一条狗撞翻一篓水果，两个士
兵会出来。杀死他们后，别急着前进。
站在被狗撞翻的水果篓的位置向左攻击
水果摊会不断获得经验。成功得分的关
键是不能攻击到香蕉，一旦攻击香蕉则
再也无法无限得分。

[ 升级设定 ]
─────────────────

经验值，升级时提升攻击力、补血、

[ 隐藏道具 ]¶

[ 隐藏道具 ]
──────────────────────
关卡1(AVALON)：
补血道具--刚进场景的画面右下那个水洼。

关卡2(NEOMOSC)：
1级武器--村庄入口左下方的墙壁。
(入口右下方墙壁也可打破，但无物品)
补血道具--经过坦克车后，画面左侧横排灌木丛。
补血道具--BOSS战场景，画面最右边由上向下数，
第2个向里凹的位置。
加分道具--BOSS处那扇大门。

关卡3(TURAN)：
加分道具--上楼时，楼梯交接处挂画像的地方。

关卡5(MAHARISHI)：
3级护具--一开始那条路的尽头有三个摞在一起的
蓝道箱子，破坏后有一个隐藏房间。

隐藏关(NEOMOSC)
补血道具--开始场景，上小桥处的右边矮墙
加分道具--村庄入口右下方墙壁

关卡7(BLANCMUTE)：
补血道具--在出现一波金甲敌人的斜向路段，
有很短一段向南的断桥(两块木板)。

关卡9(SECOND KEY)：
补血道具--围墙左上角，即上方出口左边。

关卡10(ALEISTA)：
补血道具--开始场景地图左上那面山墙的中间。

关卡11(ATLANTICA)：
加分道具--场景右下那颗完整显示的树。
加分道具--道路最右方台子的左上那棵树。
加分道具--道路最左方的那棵树(树前有块墓碑)。

关卡14(GAIAPOLIS)：
加分道具--刚开始的场景左下角。
加分道具--BOSS战场景的最下方那个角。

最终关(LAST BATTLE)：
黄金鹰之剑--BOSS的王座。

[ 隐藏关卡 ]¶

[ 隐藏关卡 ]
──────────────────────
·关卡KOBEN：
关卡MAHARISHI取得３尊石像后摧毁喷水池中的塔
第一尊石像：关卡入口的四个水果篓的里面
第二尊石像(随机出)：获得盾的密室入口左边黑色
坛子里。必须进密室前打，如先进密室，出来后就
没坛子了。
第三尊石像(随机出)：铁栅栏最左边角落
只有取得３尊石像才能摧毁塔。
过关获得“有翼之石”可进隐藏关KOBEN。

·关卡NEOMOSC：
关卡MAHARISHI结束时，等级达到12级，
过关获得“剑之水晶”，进隐藏关NEOMOSC。
关卡NEOMOSC会得到3级水晶武器。

以上条件若同时满足，只会走KOBEN关。
若以上条件均不满足，则是得到魔法护符，
进关卡BLANCMUTE。
