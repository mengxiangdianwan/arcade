batsugun 疯狂枪支 (C) Toaplan 1993¶

疯狂枪支 (C) Toaplan 1993

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：batsugun

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────────
 Ａ：射击
     按住Ａ与连按Ａ是两种不同火力。
     每级吃P道具最多可提升4次火力。
 
 Ｂ：保险
     火力升级到LEVEL3后，每满一次升级槽，
     得一个保险。 

男一号是异域战将的主角，使用其通关后会有交代。
按游戏年代算，本作在异域战将之后；但如果按游戏
剧情算，异域战将的故事发生在本作之后。

[ 隐藏的猪 ]¶

[ 隐藏的猪 ]
───────────────────────
有桃色猪和紫色猪两种，猪的效果发动时他的耳朵会
竖起来。
靠近桃色猪，每1/60秒获得110分。
靠近紫色猪，每1/60秒获得11火力升级点数。

第2关：岛上机场最右边的塔楼，按住→击毁它会出
现桃色猪。特别版为紫色猪。

第3关：按住↓击毁中央炮台，在BOSS战前的瀑布旁
会出现桃色猪。特别版为紫色猪。

第5关：
关卡开始不久画面左侧的房子(普通版为绿色，特别
版为蓝色)，摧毁房子后里面出现两辆坦克，按住←
摧毁它们，出现两只桃色猪。特别版为紫色猪。

特别版，圆形坦克场景最后有一整排圆形坦克，
用保险会把它们全炸成桃色猪。

[ 隐藏的59630分 ]¶

[ 隐藏的59630分 ]
───────────────────────
第2关：岛上机场机库(普通版为红色、特别版为蓝色
)中驶出的飞机，当它在跑道上滑行准备起飞时再摧
毁它。跑道被炸毁出现59630分。

第4关：BOSS战时，两侧镭射炮台被击毁后会不断出
现小型坦克，当他们填满轨道后会变成红色，此时用
保险炸毁坦克，会把每辆坦克都炸成59630分。

第5关：
普通版，让黑黄边移动平台移动至终点。若把平台上
的两个机器人都击毁，平台会停止移动。
特别版，方法和普通版稍有不同。在移动平台移动至
终点前，将终点两侧台子总共10个箱子，打开至少8
个。

特别版，圆形坦克用保险能炸出59630分。

[ 版本部分区别 ]¶

[ 版本部分区别 ]
───────────────────────
特别版中
1)所有关卡颜色不同
2)一些图形颜色不同
3)标题画面，演示动画和机体选择画面颜色不同
4)火力升级效果不同，特别版升级时有无敌效果。
5)机体撞毁设定范围比原版小
6)单机保险即是普通版中的合体保险。
7)双厢卡车被击毁后会爆出桃色猪，而不是普通V字
道具。

[ 隐藏功能 ]¶

[ 隐藏功能 ]
───────────────────────
机台设置中有“无敌”功能可开启。若开启无敌，
可按2P 开始键暂停，1P 开始键继续。
