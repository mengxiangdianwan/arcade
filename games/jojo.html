○本体模式
  游戏刚开始人物的初始状态即是本体模式，本体模式下防御对
  手的替身攻击是会消耗体力的，另外按下Ｄ会转为替身模式
 
○替身模式
  本体模式时按下Ｄ即是替身模式(显示“STAND ON”)。
  替身模式基本的攻击力和防御力提升，部分角色可二段跳。
  防御对手的替身攻击会消耗替身槽。
  替身槽一旦耗光，人物就会进入完全无防备的防御崩坏状态。
  另外替身模式下按下Ｄ键会再次转回本体模式。
 
○替身连击
  替身模式下专用连锁技，以通常技取消通常技的特殊技巧，
  基本的按键顺序是：Ａ·Ｂ·Ｃ。
 
○替身出现攻击
  本体模式下的专用技巧，指令是：↓↘→＋Ｄ
  注意可远操作替身的人物使用这招后替身会自动处于分离状态。
 
○串联攻击
  会消耗一条能量棒的特殊技巧，又细分为程序类和原创连击类
  两种不同的类型。
  两次串联攻击发动间有２～３秒的间隔限定。无敌时间长。
  串联攻击中，本体可协同攻击。
 
○程序类串联攻击    指令是：↓↙←＋Ｄ(可蓄)
  发动后敌我双方都会定住(不能移动)，而发动者此时可在
  限定时间内自由输入各种不同的攻击，在限定时间完结后，
  发动方就会依照刚才输入的指令向对手展开攻击。注意程序
  类串联攻击在输入↓↙←＋Ｄ后是可以按住Ｄ不放来延长
  串联攻击指令输入时间的，不过延长的时间段内对手是可以
  自由活动的，所以意义不大。
  对应程序类串联攻击类的人物如下：
  空条承太郎、花京院典明、阿布德尔、波鲁那雷夫、伊奇、
  米德拉、迪波、维利拉艾斯、迪奥、克服恐惧的花京院典明
 
○原创连击串联攻击  指令是：↓↙←＋Ｄ
  和《街头霸王ZERO》系列原创连续技非常相似的系统，
  发动后角色会自动向前快速移动，此时可自由输入各种技巧
  向对手展开进攻。
  对应原创连击串联攻击的人物如下：乔瑟夫、阿雷希、恰卡

○相杀特写

  特定人物对战时，替身模式下部分技能在一定距离内同时发出触发
  判定构成相杀可形成相杀特写。相杀中，按键连打快的一方会把对
  方打飞，对手吹飞中可追打。
  必须是下列人物之间对战时使用相应技才可达成：

[ 关于“时间停顿” ]¶

[ 关于“时间停顿” ]
──────────────────────────────
 　游戏中共有三个人物可以使用这一技巧分别是空条承太郎 、迪奥
   和影子迪奥。这一招超必杀技不但不能防御，而且具有全屏攻击
   判定，所以对手是避无可避的。不过这一招只有在能量槽达到
   LV.3以上方可启动，注意能量槽级数越多停顿的时间也就越久。
   (能量槽级数=秒数，即LV.3时使出时间会停顿3秒，而在能量槽处
   于MAX(10级)时使出时间就会停顿10秒。)

   虽然“时间停顿”同时具有全屏判定和防御不能两种特性，不过
   要破解这一招却是非常简单的，原因是这一招出招奇慢，即使在
   对手倒地情况下使出，对手也能在起身后从容反击。另外空条承
   太郎、迪奥和影子迪奥这三人之间进行对战时，这一招的实用性
   就更小了，因为任一方停止时间后，另一方只要输入停止时间的
   指令，就可以在对方停止的时间中恢复活动。
   (当然前提是被定住一方此时的能量足以启动“时间停顿”)
   不过这一点倒是非常忠实原着的设定。

[ 隐藏要素 ]¶

[ 隐藏要素 ]
──────────────────────────────
○颜色选择
  选择人物时，ＡＢＣＤ开始键各自对应一种颜色。

○迪奥 使用方法
  角色选择画面，顺序在以下角色头像上输入指令：
　·阿雷希，按１次开始键。
　·迪波  ，按１次开始键。
　·恰卡  ，按１次开始键。
　·米德拉，按３次开始键。
　成功的话，画面右下角就会多出迪奥。

○邪恶的化身 迪奥!! 使用方法
　·先选出迪奥。
　·将游标移到“迪奥”上，按4次或4次以上开始键。
　成功的话，迪奥的头像会有变化。
  ·直接选“迪奥”是普通版；按住开始键来选是
  “邪恶的化身 迪奥”

○夸高血统的乔瑟夫 使用方法
  角色选择画面，顺序在以下角色头像上输入指令：
　·承太郎　　，按１次开始键。
　·波鲁那雷夫，按１次开始键。
　·乔瑟夫  　，按１次开始键。
　·花京院　　，按１次开始键。
　·伊奇　　  ，按１次开始键。
　·阿布德尔  ，按１次开始键。
　·乔瑟夫  　，按住开始键１秒以上。

○所有隐藏人物开启
   进入テストメニュー / TEST MENU(测试画面)，
   用1P摇杆选择第7项的“设定 / CONFIGURATION”，
   再选择第2项“ゲーム设定 / GAME”进入，
   然后在“ゲーム设定 / GAME”画面中顺序输入
   ↑、↓、→、←、Ａ、Ｂ、Ｃ、Ｄ，
   成功的话会听到一声怪叫。
   之后进入角色选择画面，便会发现角色选择画面多了迪奥。
   将游标停在空条承太郎头像上，再按←，可选到夸高血统
   的乔瑟夫；将游标停在花京院典明头像上，再按→，可选
   邪恶的化身-迪奥。

○“死神13”乱入
   1.选用“空条承太郎”、“乔瑟夫”、“花京院典明”
    “阿布德尔”、“波鲁那雷夫”、“伊奇”这几个
     漫画中的主角人物来进行STOTY MODE。

   2.没有续关。

   3.没有输过一个回合。

   4.中途没有人乱入向你挑机。

   5.所有对手均用超必杀技K.O.掉。

  达成以上条件，“死神13”便会在第5～6个人乱入。
