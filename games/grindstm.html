grindstm 飞鹰风暴 (C) Toaplan 1992¶

飞鹰风暴 (C) Toaplan 1992

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：grindstm

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────
Ａ：射击  Ｂ：保险

在没按住Ａ时，可以通过↑和↓调整辅机的
位置，当辅机在前方重合时，火力最大。

[ 升级道具说明 ]¶

[ 升级道具说明 ]
───────────────────
S-UP：提升速度(最高升4次，之后图示就变
成S-DOWN，即降低速度)
SHOT：普通火力(默认)
SEARCH：跟踪火力
MISSILE：导弹
POWER：加强火力(最高升3次)
SHIELD：防御罩(可挡住一次子弹，在防御
罩消失后的2秒内玩家飞机处于无敌状态)

[ 版本区别 ]¶

[ 版本区别 ]
───────────────────
日版名V-Five，美版名Grind Stormer
日版画面下方为待升级的属性，击毁送道具
的敌机得到的是不针对特定项目的升级道具
。美版画面下方为保险，击毁送道具的敌机
得到的是特定项目的升级道具和保险。

[ 隐藏的无敌功能 ]¶

[ 隐藏的无敌功能 ]
───────────────────
如果开启机台设置中的Invulnerability
(mame plus中名字为耐久力)，玩家无敌。
可按2P的开始键暂停游戏，按1P的开始键重启游戏。
