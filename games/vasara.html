[ 擦弹加分 ]¶

[ 擦弹加分 ]
───────────────────
自机中弹判定点周围擦弹时会有金色火花显
现，同时有加分。

[ 全斩加分 ]¶

[ 全斩加分 ]
───────────────────
每关合计有３０个有姓名的武将，
杀死他们过关时有额外加分。

当３０武将全斩杀时，过关会显示
“全PERFECT斩”，奖励１５０００分。

[ 蓄力/婆裟罗攻击 ]¶

[ 蓄力/婆裟罗攻击 ]
───────────────────
蓄力攻击可消敌弹。

☆红色宝石道具会增加婆裟罗槽。

☆婆裟罗(VASARA)槽满时蓄力攻击会
  发动婆裟罗攻击。

☆Ａ连打可使持续时间略有延长。

☆婆袈罗发动下的蓄力攻击范围和威力
  都明显提升。

[ 自机简介 ]¶

[ 自机简介 ]
───────────────────
·真田 幸村 [武器：村正]

移动速度：第２
攻击力　：第２

主火力：广范围直进型
副火力：广范围跟踪型

蓄力攻击：“烈风鬼斩”
　        攻击和消弹范围广
　

·杂贺 孙市 [武器：时雨]

移动速度：第１
攻击力  ：第３

主火力：范围广、威力分散
副火力：浮游追踪型扇
　　　　速度慢

蓄力攻击：“红莲扇风舞”
　        攻击和消弹范围为自机周围前方
          攻击距离近
☆注：普通模式下，快速输入任意方向后再
      立刻输入反方向可将扇风舞甩出去。

·岛 左近 [武器：墨炎]

移动速度：第３
攻击力　：第１

主火力：窄范围集中型
副火力：前方攻击用枪
　　　　击中敌人后会缓慢直行穿透

蓄力攻击：“铁甲枪·无尽”
　        攻击和消弹范围窄
          蓄力攻击远端范围大一些
