○防御崩坏
  体力槽下方是防御槽，防御攻击时会减少。
  防御槽耗尽时防御状态强制解除，处于硬直中的
  完全无防备状态。

○ROLLING CANCEL
  简称RC，指用回避来取消必杀技
  (特指C/A/N GROOVE的回避)
  作用是使必杀技附带ROLL的无敌时间。
  基本上任何必杀技都能RC，一些特殊技也可。
  要破解RC，通常只有LV3的超杀或者对用RC才可。

  例1：↓↘→＋拳 = ↓↘→＋轻拳轻脚·→＋拳
  例2：←蓄→＋拳 = ←蓄↓＋轻拳轻脚·→＋拳
  例3：↓蓄↑＋拳 = ↓蓄↘＋轻拳轻脚·↑＋拳

○COUNTER HIT
  在对手出招时(指令成立至招式完结前)击中对手
  ，就会造成COUNTER HIT。

○FIRST ATTACK
  即每回合开始后率先击中对手

○SUPER COMBO FINISH
  使用C-GROOVE / A-GROOVE / P-GROOVE的
  超杀KO对手。

○SUPER SPECIAL MOVE FINISH
  使用S-GROOVE / N-GROOVE / K-GROOVE的
  超杀KO对手。

○ORIGINAL COMBO FIHISH
  使用A-GROOVE的原创连续技KO对手。

○FINISH KO
  以下两种情况会被判定为FINISH KO
  1、用超杀击中发动必杀技中的对手形成
     COUNTER HIT且刚好KO对手。
  2、用必杀、超杀击中发动超必杀技中的对手形
     成COUNTER HIT且刚好KO对手。

○SPECIAL KO
  以下四种情况
  1、SUPER COMBO FINISH
  2、SUPER SPECIAL MOVE FINISH
  3、ORIGINAL COMBO FIHISH
  4、FINISH KO
 
○C GROOVE系统
  攻击力可提高三级。
  能量槽分三段，可使用三种不同等级的超杀。
  即输入指令后，按轻攻击键为LV1超杀；
  中攻击键为LV2超杀；重攻击键为LV3超杀。

  特别之处是LV2超杀在途中可用必杀或超杀取消，
  形成独特连击。

      疾进：→→
  空中防御：空中↖or←or↙
  紧急回避：轻拳＋轻脚
  延迟起身：倒地时同时按住两个拳不放
  防御反击：防御对手攻击时→＋中拳中脚
            (消耗LV1能量)

○A GROOVE系统
  源自少年街霸的系统。
  能量槽分两段，蓄满一段即可使用LV1超杀，能
  量全满后可发动原创连续技模式，缺点是不能使
  用LV3超杀。

  原创连续技发动时有瞬间无敌，发动后能量槽开
  始下降。耗光前，可随意取消任何招式、蓄力攻
  击的蓄力时间几乎为零。该状态下使用超杀或被
  对手击中即会强制退出原创连续技模式。

      疾进：→→
  紧急回避：轻拳＋轻脚
  倒地受身：倒地瞬间同时按两个拳
  防御反击：防御对手攻击时→＋中拳中脚
            (消耗LV1能量)
  原创连续技模式：能量槽全满时重拳＋重脚

○P GROOVE系统
  源自街霸3的系统。
  能量槽只有一段，但蓄满即可使用LV3超杀。

  BLOCKING格挡：源自街霸3。
  成功的话不会消耗体力，也没有防御后的硬直。
  此时可对进入收招硬直时间的对手进行反击。

      疾进：→→
    小跳跃：瞬间输入↖or↑or↗
  延迟起身：倒地时同时按住两个拳不放

  中上段BLOCKING：
  在对手的中上段攻击命中自己之前的瞬间→

  下段BLOCKING：
  在对手的下段攻击命中自己之前的瞬间↓

  空中BLOCKING：
  空中时，在对手的攻击命中自己之前的瞬间→

○S GROOVE系统
  源自拳皇95的系统，唯一可自己蓄能量的模式。
  能量槽只有一段，蓄满后可使用LV1超杀，最大
  优点是体力槽仅剩15%时会变红，此时可无限使
  用LV1超杀。如果体力槽见红且能量槽满了，可
  使用LV3超杀。

      奔跑：→→(第2次→按住不放)
    小跳跃：瞬间输入↖or↑or↗
    蓄能量：按住重拳＋重脚不放
      闪避：轻拳＋轻脚
  闪避攻击：闪避中，拳or脚
  延迟起身：倒地时同时按住两个拳不放
  防御反击：防御对手攻击时→＋中拳中脚
  (能量槽满时才可用，使用后全部能量消失)

○N GROOVE系统
  源自拳皇98的系统，蓄满一次能量即可获得一颗
  能量珠。最多蓄三颗。
  一颗能量珠可发动一次LV1超杀；MAX状态下可使
  用LV3超杀。

      奔跑：→→(第2次→按住不放)
    小跳跃：瞬间输入↖or↑or↗
  紧急回避：轻拳＋轻脚
  闪避回避：紧急回避中，拳or脚
  延迟起身：倒地时同时按住两个拳不放
  防御移动：防御对手攻击时←or→＋轻拳轻脚
            (消耗一颗能量珠)
  防御反击：防御对手攻击时→＋中拳中脚
            (消耗一颗能量珠)
   MAX状态：至少有一颗能量珠时重拳＋重脚
            (消耗一颗能量珠)

○K GROOVE系统
  结合了侍魂系列和饿狼传说-狼之印记的系统，
  只有一段能量，且只有被击中或使用及时防御时
  才会增加能量。蓄满后可用LV3超杀，怒状态下
  攻击力增加1.35倍，防御力增加1.3倍。

      奔跑：→→(第2次→按住不放)
    小跳跃：瞬间输入↖or↑or↗
  倒地受身：倒地瞬间同时按两个拳
  及时防御：在对手的攻击命中自己之前瞬间输入
            防御指令。不会扣体力，还能恢复一
            些体力(恢复量取决于对方所用招式)
            还可减少防御后的硬直，不会减少防
            御耐久值。空中也可使出。

●更换游戏模式 

  进入SYSTEM MENU - GAME TEST MODE -
  GAME CONFIGURATION - PASSWORD

  输入YAMAOOUT或MORIKAWA

  再进入GAME CONFIGURATION，追加BATTLE YYPE
  可将之自由切换为RATIO MATCH、3on3 match、
  single match。

  RATIO MATCH：游戏默认系统。只可选择1-3人
  出战。RATIO值和前两作一样，一共4点。
  选完角色后，进入RATIO值分配画面，可自由分配
  个角色的RATIO值。值越高的角色能力也越强。

  3on3 match：拳皇式的3对3模式

  single match：一对一模式。

●BOSS BATTLE模式
  
  进入SYSTEM MENU - GAME TEST MODE -
  GAME CONFIGURATION - PASSWORD

  输入MORIKAWA

  投币后于游戏标题画面中，按住中拳中脚不放，
  再按开始键，即进入该模式。
  该模式下CPU角色均为BOSS级别。

●隐藏颜色 
  角色选择画面选人时，六个不同的攻击键对应
  六种基本颜色，同时按轻拳中拳重拳可选到第
  7种颜色，同时按轻脚中脚重脚可选到第8种颜
  色。

●CPU拜森(日版维嘉) 或 CPU基斯 乱入
  CPU第4战结束前，满足如下条件，会发生乱入。
  1、没出现过TIME OVER
  2、没被CPU取得６次以上的FIRST ATTACK
  3、没给CPU SPECIAL KO ３次以上

●普通BOSS战
  CPU第5战结束前，满足如下2个以上条件。
  1、GP取得1000点以上
  2、取得４次以上的SPECIAL KO
  3、CPU拜森(日版维嘉) 或 CPU基斯乱入
  4、取得２次以上的LV3 SUPER COMBO FINISH
     胜利，即用LV3超必杀技KO对手２次以上。

●真BOSS战
  CPU第5战结束前，满足如下2个以上条件。
  1、GP取得1500点以上
  2、取得１次以上的SPECIAL KO
  3、CPU拜森(日版维嘉) 或 CPU基斯乱入
  
●乱入战·ＢＯＳＳ战　人物决定条件
·第４战结束前，挑衅次数０，
  乱入角色BOSS角色随机

·第４战结束前，挑衅次数1、5、9、13…，
  乱入角色拜森(日版维嘉)、BOSS角色豪鬼

·第４战结束前，挑衅次数2、6、10、14…，
  乱入角色基斯、BOSS角色豪鬼

·第４战结束前，挑衅次数3、7、11、15…，
  乱入角色拜森(日版维嘉)、BOSS角色卢卡尔

·第４战结束前，挑衅次数4、8、12、16…，
  乱入角色基斯、BOSS角色卢卡尔

  便可看到对应的一款胜利对白。
