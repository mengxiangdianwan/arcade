[ 难度rank ]¶

[ 难度rank ]
─────────────────────────
游戏关卡难度和主火力(对空)等级直接相关。
每关都有一个默认难度，当前主火力等级≤关卡默认
难对应的火力等级时，当前关的难度为默认难度。
主火力超过对应等级时，会提升当前关的难度等级。
关卡默认难度对应的主火力等级如下
关卡 1：02/14 
关卡 2：06/14 
关卡 3：09/14 
关卡 4：11/14 
关卡 5：12/14 
关卡 6：13/14 
关卡 7：14/14 
比如关卡1时主火力达到Lv2(超过了02/14)，rank难度
就会提升到关卡2的水平(06/14)。

除了主火力等级，还有以下因素会增加rank值：
敌机编队全灭、击毁BOSS的部件、各种道具取得、
按一次射击键

[ 捕获小BOSS ]¶

[ 捕获小BOSS ]
─────────────────────────
在游戏中会有一些头上带有一个闪光圆球的强敌出现。
攻击圆球一会会打掉它，在圆球自毁之前赶紧得到就
可捕获该强敌为己用。注意：圆球离体后会很快自毁
该系统设定被续作G-Darius改进使用。
击毁小BOSS得分20000，捕获小BOSS得分30000

[ 特殊道具 ]¶

[ 特殊道具 ]
─────────────────────────
三种特殊道具：奖命、全屏灭敌(金)、奖分(银)
全屏灭敌(金)和奖分(银)是隐藏道具，每关都有。
射击特定地点即可打出，3级及以上有贯穿地形性能的
主火力比较容易打出。黑洞爆弹无法打出隐藏道具。
奖分(银)：随机50-51200分之间共11档
奖命：样子是一个透明小气泡里的小飞机、
      第3关和第6关可击毁敌兵得到

[ 关于特别版 ]¶

[ 关于特别版 ]
─────────────────────────
特别版为官方hack版，关卡改动、玩家火力发射频率
大幅提高，新增加暂停键。
单人游戏时，使用2P无法在关底选择关卡分支，这样
就可以打全部28关。双打时，先选2P，再让1P加入，
则该方法同样有效。
