1942 1942 (C) Capcom 1984¶

1942 (C) Capcom 1984

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：1942

[ 基本操作 ]¶

[ 基本操作 ]
─────────────────
  Ａ：射击
  Ｂ：空翻
      每关３次

[ 隐藏要素 ]¶

[ 隐藏要素 ]
─────────────────
[ 弥七 ]
是从画面边端出现，向上直接飞过去的
敌机，射击弥七使它发生变化。
每次击落数200架时出现。
5000分

[ 特别攻击奖励 ]
用旁边的副机击坠大型敌机，和显示的
分数无关，可得10000分。
中型敌机则是5000分。
如果失去副机，自机仍然获得同样的分
数。
要点：大型敌机初次2000分，再度击落
则每次以500分递增，最高到8000分。
如果失误，则回到2000分重新开始。

[ ∑ ]
自机数达到10架以上的话，则10架部分
表示为 ∑。

[ 关于BUG ]¶

[ 关于BUG ]
─────────────────
不是 MAME 的原因，是实际街机引起的
BUG。

·10000分？
击坠率100%时，应获得50000分奖励，
但是，在 set1 和 set2 版中，却显
示10000分。实际上，还是得到50000
分的。
在 set3 版，已修复显示为50000分。

·Extend 死机
20000分、80000分、以后每个80000分
再到96000分，总计13次，这应评定为
Extend，然而一次性获得超过 Extend
点10000分以上的话，
于是，就出现 Extend 死机。
在 set3 版中，已修复此 BUG。

·冻机
在亚矢虎出现同时使用空翻，就会
冻机了。只限于 set2 版。

[ 自机杂记 ]¶

[ 自机杂记 ]
─────────────────
自机在太平洋战争中，以双头的恶魔之
称而驰名，击坠了山本五十六军官座机
的也是有名的 P-38 Lightning。
这架战机为美国陆军使用。实际上，并
没有装载在航空母舰上。但不知为何在
这个游戏中，它是从航母上起飞的。
在成为中途岛海战导火索的东京市初次
空袭中。如以 Dorit中校的部队从大黄
蜂号航空母舰上起飞 B-25 机的事实为
基础，从航母出发返回是可行的。
在这个游戏中出现的“88”号航空母舰
实际上并不存在。但应该是以大黄蜂号
航母和 CV-8型航母为原型而设定的。
