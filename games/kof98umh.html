  ADVANCED能量槽时消耗1颗能量珠；
  EXTRA能量槽时，普通状态下消耗1颗能量珠，
  MAX状态下发动则会强制终结MAX状态)

○援护攻击
  1.昏迷
  2.在画面中看到尚未败阵的同伴，
    且尚未败阵的同伴的相性与自己的相性不相克。

  角色间存在三种相性，分别为：
   1.良好 - 必定会出手相助。
   2.普通 - 只有一半机会出手相助。
   3.恶劣 - 一定不会出手相助。
  满足以上条件的情况下，同时按ＡＢＣ
 
○防御破坏
  和气绝十分相似，每个角色都有一定的防御耐久值
  ，受对手连续攻击时防御耐久值便会减低，当超过
  极限后，就出现角色防御不能状态。而当出现防御
  破坏时，不仅防御状态被强制解除，原来若具有
  MAX状态的能量也要被解除。但由于持续时间极短
  ，所以除非在版边，要不然都不容易形成追打。

○COUNTER HIT*CRITICAL HIT
  COUNTER HIT是击溃对手特殊技、必杀技、超必杀
  技、MAXIMUM状态下的通常技和奔跑(ADVANCED模式
  )，除攻击力加强之外，对手在倒地前会再次出现
  被攻击判定，此时能以任何方法作空中追打攻击。
  而以特定必杀技、其特定的动作击中对手及随机的
  情况下便会出现CRITICAL HIT，它除了将必杀技的
  攻击力加强之外，更会令对手出现瞬间硬直状态和
  被攻击判定，能于此时出招作追打攻击。

[ ADVANCED模式专用操作 ]

○打击取消紧急回避(消耗1颗能量珠)
  通常技、特殊技击中对手或被对手防住时ＡＢ

○ADVANCED能量槽
·靠攻击或者防御来积蓄能量，每蓄满1条能量槽可
  得到1颗能量珠。
·能量珠第1个人最多蓄3颗，第2人最多可蓄到4颗，
  第3人最多蓄到5颗。
·有能量珠的状态下，同时按ＡＢＣ可发动MAX状态
  (消耗1颗能量珠)。MAX状态下，攻击力是平时的
  1.25倍。
·消耗1颗能量珠可发动超必杀技。
·MAX状态下，发动的超必杀技(消耗1颗能量珠)会
  自动强化为MAX超必杀技(发动后MAX状态会强制终
  结)。
·普通状态下，输入超必杀技指令时最后改为同时
  按下两个相同属性的按键，也可发动MAX超必杀技
  ，但1次就会消耗掉3颗能量珠。
  比如，红丸的雷光拳指令由↓↘→↓↘→＋ＡorＣ
  改为↓↘→↓↘→＋ＡＣ

[ EXTRA模式专用操作 ]¶

[ EXTRA模式专用操作 ]
───────────────────────
前冲        →→
行走        行走的速度是ADVANCED模式(或者选择
            ADVANCED移动方式的ULTIMATE模式)的
            1.3倍
攻击回避    ＡＢ
回避攻击    攻击回避中ＡorＢorＣorＤ

○打击取消回避  
  通常技、特殊技击中对手或被对手防住时按ＡＢ。
  普通状态下发动会消耗1颗能量珠；MAX状态下发动
  会消耗1/2条MAX状态的时间槽。
  打击取消回避动作中，按ＡorＢorＣorＤ可使出回
  避攻击。

○打击取消MAX状态发动
  通常技、特殊技、必杀技击中对手或被对手防住时
  或自己防御中，按ＡＢＣ(消耗1颗能量珠)。
  作用是减少技的收招硬直(加快收招动作)，这样就
  可以使出一些平时不成立的连续技。比如比利的
  →＋Ａ原本不能接任何招，但只要在→＋Ａ第2段
  的时候用打击取消MAX状态发动，就可以使用其他
  招式来接续了。

○EXTRA能量槽
·被击中或防住对手的攻击可积蓄能量，能量槽蓄满
  可得到1颗能量珠。
·没有能量珠的情况下，按住ＡＢＣ不放可蓄能量，
  能量槽蓄满后得到1颗能量珠。
·拥有1颗能量珠的情况下，同时ＡＢＣ可发动MAX状
  态(通常技、特殊技、必杀技击中对手或被对手防
  住时，或自己防御中，都可以发动MAX状态)。
  MAX状态下，攻击力是平时的1.25倍。
·当第1个角色的体力低于1/4，体力槽便会出现不停
  闪烁的红血状态。第2人、第3人时，体力槽会比第
  1个角色更快进入红血状态。
·消耗1颗能量珠可发动超必杀技。
·MAX状态下，可发动超必杀技
  发动后MAX状态强制终结
·红血状态下，可发动超必杀技
  发动后红血状态会暂时消失一小段时间，
  必须稍等一会后红血状态才会重新出现。
·红血状态＋MAX状态下，发动超必杀技会自动强化
  为MAX超必杀技。
  发动后红血状态会暂时消失一小段时间，必须稍等
  一会后红血状态才会重新出现；MAX状态也会强制
  终结

[ ULTIMATE模式 ]¶

[ ULTIMATE模式 ]
───────────────────────
本模式是ADVANCED模式和EXTRA模式的融合，
在移动、回避、能量槽三方面任意选择是ADVANCED
模式还是EXTRA模式。

移动模式：可选择奔跑或者前冲
回避动作模式：可选择紧急回避或者攻击回避
能量槽模式：可选择ADVANCED能量槽或EXTRA能量槽
超必杀技的发动条件，ADVANCED能量槽和ADVANCED
模式一样，EXTRA能量槽和EXTRA模式一样。

[ 隐藏要素 ]

①随机选用角色 & 随机选用队伍
  角色选择画面，游标移到左边的？内，可进行随机
  队伍选择；将游标移到右边的？内，可进行随机角
  色选择。

②隐藏颜色
  本作中每位角色均有8种颜色可供选择，方法是分别
  按下以下按键来选定人物。
  Ａ、Ｂ、Ｃ、Ｄ、Ａ＋Ｂ、Ａ＋Ｄ、Ｂ＋Ｃ、Ｃ＋Ｄ
 
③选择胜利动作
  将对手K.O后马上按下ＡorＢorＣorＤ不放，
  成功的话就可以选到与该键相对应的胜利动作，
  你也来选出你最喜欢的胜利动作吧......
 
④蔡宝健的特殊胜利画面
  本秘技在对人战和对CPU战中均有效。
  只要达成以下条件，SNK日本电台两名人气角色
  便会骑单车出现。
 1. 队伍中必须要有“蔡宝健”。
 2. 战斗场景必须是在“日本街头(JAPAN STREET)”
 3. 第三回合时，蔡宝键在30秒内K.O对手，但不能
    是PERFECT完胜，且最后K.O对手的那一下必须
    是必杀技、超必杀技或MAX超必杀技。
    (非满血但回合结束时因补充体力而达到的体力
    全满不算在内，因那样的话已经不算PERFECT了)
 4. K.O对手后，按住Ｃ不放来选择胜利动作。

⑤NEOGEO博士现身
  本秘技在对人战和对CPU战中均有效。
  只要达成以下条件，NEOGEO博士便会骑单车出现。
 1. 队伍中必须要有“克拉克”。
 2. 战斗场景必须是在“日本街头(JAPAN STREET)”
 3. 第一回合，克拉克获胜
 4. 第二回合，克拉克最后必须以任意超必杀技K.O
    对手，而且战斗结束时，时间刚好停在44秒。
