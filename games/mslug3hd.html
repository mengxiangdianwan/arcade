[ 状态说明 ]¶

[ 状态说明 ]
───────────────────────
[胖子]
食物道具吃多后的状态。
移动速度和射击速率下降、火力的攻击范围变大。
“减肥药”道具可解除胖子状态。

[木乃伊]
被木乃伊攻击后的变身形态。
武器被限制为手枪。不能坐乘物。
移动速度大幅降低、手榴弹投掷速度也变慢。
一定概率下，扔手榴弹时会把头扔出去。
跳跃能力提高。
得到强化武器都会变成1000分。
“圣水”道具可解除该状态。

[僵尸]
被僵尸攻击后变成的形态。
武器被限制为手枪。不能坐乘物。
移动速度比木乃伊状态更慢。
不能下蹲，跳跃力几乎丧失。
但这个形态下角色不怕普通敌人的攻击
(丧尸、雪人除外)，
而副武器也会变成威力巨大的横扫角色
面前90度范围的喷血攻击。
“急救箱”道具可解除该状态。

[ 特殊设定 ]¶

[ 特殊设定 ]
───────────────────────
蹲着射击会提升射击速度。

攻击取消：蹲下和站起能取消射击动作。这个设定可
用于节省[H]枪和[L]枪弹药。

重合快刀：和敌人重合在一起时，蹲下左右快速摇方
向并不断攻击。主要运用在最后一关的复制人和最终
BOSS身上。

蹲跳快刀：等敌人贴近时，蹲下砍一刀后立刻同时按
下攻击和跳，就可以在原本砍出一刀的时间内出两刀
。主要运用在不会吐泡泡的寄居蟹身上。

跳跃刀：敌人近身后，射击和跳一起按（或先按射击
、后按跳）就是跳跃刀。该设定的好处是可以在攻击
敌人的同时躲避攻击。

手雷“熏”：就是利用手雷爆炸后的烟雾攻击敌人，
用这种方法可以多赚分。可“熏”的范围包括第一关
的寄居蟹、第二关的木乃伊、第三关的盾牌兵、第四
关的食人花等等。

从二代开始，乘物过关加分条件改为必须使用乘物击
毁BOSS，而在一代中只要击毁BOSS时场景中有乘物即
可。

从二代开始，用刀杀死敌人500分，过关后每名战俘
奖励有10000分，特殊人物（相川留美，龟波老头-
一文字百太郎）有30000加分，乘物有100000分，打
掉有捆绑战俘的绳索和打隐藏战俘点都也有加分，
所以能用刀砍尽量用刀，减少重型武器的使用是拿高
分的方法。此外打部分隐藏点有额外加成分，方法是
只能用手枪来打，会得到剩余隐藏东西成倍的加分。
比如隐藏点有3名战俘，用手枪打一枪可以得300分，
还剩2名战俘时打一枪有200分，以此类推。
可射击打破相川留美的背包，会掉出四个加分道具。
注意若长时间不去撞留美，她会跑掉。若打破背包，
放出四个道具一秒后也会跑掉，所以必须在打破背包
放出第三个加分道具后立刻撞她。

无隐藏点。最后可解救一个富豪。

第四关：
○走蛆洞路线
1)破坏沙袋掩体有一名龟波气功战俘。
2)射击最下面一排的酒桶底部可破坏，射击得分。
酒窖内摇起洞口后，进入蛆洞。
1)第一个从天而落一堆蛆的那个地点向上射击，
1只青蛙5万分。
2)第三个从天而落一堆蛆的那个地点向上射击，
1只公仔5万分。
3)得到钻头机的场景之后，6个顶部的洞都有物品，
前5个共计25万分，第6个是大便10分。
4)最后蛆王击杀后，立刻放弃乘物才能追上相川留美。
5)BOSS战能拖延5名战俘出来。
○山顶直行路线
1)山顶第二个沙袋掩体破坏，1名战俘（公仔5000分）。
2)雕像头顶随机出现鸟窝1000分或猫咪100分，
且都有5万分概率。
3)BOSS战能拖延5名战俘出来。
○山顶木乃伊路线左
上升路段右侧的洞里有两名战俘（[F]枪和BOMB）
○山顶木乃伊路线右
1)流沙上方的石像头右眼有1颗宝石。
2)站在下方的木乃伊房子上向上射击，有2个宝箱和
一坨大便，共60010分。在房子字母牌正对面的沙坡
稍微靠下一点的位置向左跳就可以跳到房顶。
○日本兵路线
1)画面里侧靠墙的两个石灯，各有1颗100分宝石。
2)神社入口靠里柱子的上方，1000分鸟窝。
3)5个道具箱子和1名战俘在一起的场景，破坏左上
角的小石头，一只青蛙5万分。

第五关：
无分支路线
1)坐直升机用子弹打死小兵只有100分，用螺旋桨划
死有500分。
2)地面树林的飞行场景，如果在没有飞机乘物的情
况下出现敌方直升机，则击毁敌方直升机会出现BOMB
补给，否则是5000分的小熊。
3)飞机小BOSS前，从画面右上方会飘下一辆挂着气球
的导弹车。在它刚刚出现时就迅速摧毁，会得到BOMB
补给。如果时间拖久了，即使摧毁了，也不会有任何
道具。
4)击败小BOSS后开始步行作战的场景，将履带车上的
报警器破坏有1只猫咪50000分。
5)进入外星人飞船内，刚进来的地方，背景第一根柱
子顶部，1名战俘（包子300分）；第二根柱子顶部，
两名战俘（螃蟹1000分，气功战俘）。
6)送[R]枪的生化脑机器人之后的第二根柱子顶部，1
名战俘（烧鸡1000分）。若有气功战俘，他会自动射
击有隐藏战俘的隐藏点。
7)送[R]枪的生化脑机器人之后的第三根柱子顶部，
1名战俘（螃蟹1000分）。
8)在门还没破坏之前跳上平台，向右上方开枪。
左边会掉下1个富豪（BOMB，奖章10分）。
9)XX军司令囚笼控制台的右侧第一根柱子上方，
有1名战俘（烧鸡1000分）。
10)射击背景第1、2、3、4以及最后一个圆球，
各有1名战俘。得到的道具依次为：白菜300分；
一串香蕉100分，有几率摔散成3根，每根10分；
猪1000分；鱼500分；包子300分。
11)大门前左上方那个凸出的机器，1名战俘（公仔
5000分）。
12)第一个封锁门前的台子上面中间上方房顶，
第四个封锁门前台子上面右边上方房顶各有1名战俘
（水果套餐1000分、猫咪5万分）。

[ 乘物简介 ]¶

[ 乘物简介 ]
───────────────────────
油桶可补充机器乘物的耐久力。

[SV-001/Ⅱ Metal Slug]
改进型万能战车。
可发射穿甲弹、移动力提升。
主炮：穿甲弹
副炮：巴尔干机枪

[SV-CAMEL]
骆驼。对玩家无保护。
主炮：无
副炮：巴尔干机枪

[SV-ELEPHANT]
大象。对玩家无保护。
主炮：若得到‘电池’或‘火辣椒’的强化，
主炮将变成从鼻子射出的镭射或火焰。
副炮：巴尔干机枪

[SV-ODTRICH]
鸵鸟。对玩家无保护。
移动速度快，跳跃能力高。缺点是左右转身时
有一段不能攻击的硬直时间，此时若遭到攻击
会死亡。
主炮：无
副炮：巴尔干机枪

[SVX-15D]
双足战车。
双手配备巴尔干机枪，跳跃力高，适应不平坦
的地形。
主炮：铁蜥蜴
副炮：巴尔干机枪

[SV-F07V]
垂直起降战机。
同万能战车，对玩家有保护。
主炮：导弹
副炮：火神机枪 (可180度范围内射击)

[SVX-17M]
潜艇。
主炮：鱼雷
副炮：机枪(抛物线子弹)

[SVX-22UG]
钻头车。
这个载具不能跳跃，但拥有驾驶舱抬升的能力。
移动时前方的钻头会自动启动凿穿敌人。
主炮：钻头
副炮：巴尔干机枪

[SLUG FLYER]
直升机。
主炮射程短。
主炮：爆雷
副炮：巴尔干机枪
螺旋桨可划死敌兵，得500分。

[Astro Slug]
火箭。
可配备左右副火力。Ｂ可加速移动。
主炮：导弹
副炮：巴尔干机枪。

[REBEL ARMOR]
革命武装、小型单兵动力装甲
可二段跳，按住跳不放会自动二段跳。
两臂分别为机枪和加农炮。
和‘Astro Slug’一样使用角色强化武器，
初始附带10发炮弹和200发重机枪[H]的子弹，
也可以使用[R]、[S]、[F]、[L]等强化武器，
但游戏中若无法得到弹药补给，
主武器子弹耗光后会无法使用。

[ 武器简介 ]¶

[ 武器简介 ]
───────────────────────
[-]手枪(弹药无限)：
基本武器。

[H](Heavy Machine Gun)：
机枪。可斜方向射击，火力威力为手枪两倍。
弹药量：200
补给量：150

[Ｈ](BIG Heavy Machine Gun)：
弹药量：200
补给量：150
连射速度提升、子弹体积变大。
射击死角变小。威力同[H]。

[S](Shotgun)：
散弹枪。射程为1/3画面，但是全武器中威力最大的
，可贯穿墙壁。
弹药量：30
补给量：10

[Ｓ](BIG Shotgun)：
弹药量：15
补给量：10
射程距离有半屏。威力同[S]。

[R](Rocket Launcher)：
火箭发射器。有跟踪性能和爆炸范围杀伤判定，
跟踪范围限制在射击方向45度内。威力比手榴弹稍小。
弹药量：30
补给量：20

[Ｒ](BIG Rocket Launcher)：
弹药量：30
补给量：20
子弹体积变大，判定范围也变的更大。威力同[R]。

[F](Flame Shot)：
火焰发射器。射程2/3画面。
火焰消失前可造成持续伤害。
弹药量：30
补给量：15

[Ｆ](BIG Flame Shot)：
弹药量：30
补给量：15
火力范围变得很大，火焰消失前可造成持续伤害。
威力同[F]。

[L](Laser Rifle)：
镭射枪。威力比机枪大，不能斜向射击，
对普通敌兵有贯穿攻击效果。
弹药量：200
补给量：150

[Ｌ](BIG Laser Rifle)：
弹药量：200
补给量：150
火力直径明显变大，对普通敌兵有贯穿攻击效果。
威力同[L]。

[C](Enemy Chaser)：
跟踪弹。跟踪性能好，发射速率比火箭快，
但威力比火箭小，爆炸火焰可贯通墙壁。
弹药量：40
补给量：15

[G](Super Grenade)：
榴弹枪。攻击方式类似火箭发射器，但威力更大，
威力同手榴弹，爆炸火焰可贯通墙壁。
弹药量：20
补给量：5

[D](Drop Shot)：
跳跳弹。碰到墙壁地面才会慢慢弹跳前进，
爆炸火焰可贯通墙壁。
弹药量：30
补给量：10

[I](Iron Lizard)：
铁蜥蜴。近似于地面导弹，爆炸火焰可贯通墙壁。
弹药量：30
补给量：10

[CLOUD](Thunder Cloud)：
雷暴云。玩家操纵的角色上方将会有一小片乌云
跟随杀敌，并随着主角手枪的攻击发出闪电攻击敌人
，遭受攻击的敌人将会有硬直状态，初始弹药30发。

[MOBS](Mobile Satellite/Torpedo Satellite)：
机动卫星。会像雷暴云一样跟随玩家并随主角手枪的
攻击而发射光束攻击敌人，初始弹药200发。

※手榴弹和石头威力相同，
  燃烧瓶威力小于以上两者，但范围更大
手榴弹：
图示为炸弹箱。
补给量：10

燃烧瓶：
将手榴弹变为燃烧瓶。
补给量：10

石头：
将手榴弹变为石头。
补给量：10

A.P.弹头
使合金战车的炮弹变为平射。
