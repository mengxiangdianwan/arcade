ket 绊地狱 (C) Cave 2002¶

绊地狱 (C) Cave 2002

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：ket

[ 基本操作 ]¶

[ 基本操作 ]

①操纵杆
  ↖↑↗
  ←Ｎ→
  ↙↓↘

②按钮
Ａ：射击(连打时散射；按住时锁定)
Ｂ：炸弹  
Ｃ：连射(需机台设置中开启)

锁定火力时自机的移动速度会变慢，但
击毁敌人时会爆出更多加分盒子，自机
有4个辅助火力。

散射火力时辅助火力和自机都会发子弹
，方向固定。

锁定火力时自机会发出一条镭射，碰到
敌人以后4个辅助火力会依次开始向碰
到的敌人发射子弹。离敌机越近，镭射
的攻击力越高，并且锁定的速度越快，
自机的镭射长度不固定，锁定敌人的辅
助火力越多，镭射射程越短。

[ 分数设定 ]¶

[ 分数设定 ]
─────────────────
当吃到盒子状道具时，画面上方会显示
三排盒子状的数据。从上到下依次是：
总倍率数、关卡倍率数、现倍数(盒子
数值)
总倍率数用于关底计算锁定火力分数。
关卡倍率数用于关卡中计算锁定火力分
数。现倍数是锁定火力能打出的数字。

总倍率数在BOSS战时不断减少，直至零
。BOSS战后，该数值越多，则奖分越多
。用普通散射杀敌时，会根据自机和敌
机的距离，放出适当的数位。最近的有
5，越远越小。太远的大型机就连一个
1都没有。用普通散射杀任何敌机只会
出一个数字，第一关的银色大飞机除外
。所有用普通散射打出的数位，要自机
接近时才能吃到，而且同一时间第三行
数位就会变成你刚才打出最大的数字，
吃不到的也计算。
用锁定火力杀敌时，就会出现和现倍数
相同的数字，数量和敌机大小有关。所
有用锁定火力打出来的数位会自动被吸
收，但需要少量时间。锁定能够在敌机
的残骸爆炸时抽取出数字，所以会比散
射火力多。

击杀普通敌人的分数计算方式：用散射
火力打的，只会得到敌机的分数。用锁
定火力打的，就得到敌机的分数x现倍
数(盒子数值)。
BOSS战后的分数计算方式：总倍率数x
生命数+50x炸弹数量

[ 自机选择以及差别 ]¶

[ 自机选择以及差别 ]
─────────────────
有两架机体可以选择，分别是AH－Y72
(TYPE A)和FH－X4(TYPE B)。
在移动速度上，TYPE A比TYPE B的速度
相对较慢些，锁定火力时也是如此。
在散射火力方面，TYPE A是“W”形的
散射，TYPE B是直线形散射。
如果全中敌人的话TypeA比TypeB威力
略大，但是因为TypeA范围大，所以一
般不能全中敌人，这时候TypeB看起来
威力大些。
在锁定火力方面，TYPE A的锁定速度比
TYPE B要快。

[ 判定 ]¶

[ 判定 ]
─────────────────
本作是款标准的弹幕STG，中弹判定极
小，位置大致在自机螺旋桨机轴处。
敌机子弹中间纹路是十字、颗粒很大的
圆球弹最外面一圈也是没有判定的。
若是自机与敌机发生相碰，这时判定则
是算整个机体但不包括子机。

[ 二周目条件 ]¶

[ 二周目条件 ]
─────────────────
表二周目--1周Bomb次数＋死命数≤6。
里二周目--1周不死命＋不用BOMB，
分数1.2亿以上，最终见真BOSS光翼。

[ 加分道具1-UP ]¶

[ 加分道具1-UP ]
─────────────────
第3关：不放BOMB，摧毁小BOSS的所有
武器后再击毁它。
第5关：小BOSS出场前的那个场景里，
会从左下方先后飞出两架移动速度很快
的和玩家飞机外形一样的小型直升机
(会放一堆蓝色导弹，第1架橙色，第2
架蓝色)。不放BOMB，摧毁这两架飞机
，第2架会爆1-UP。
