dondokod 木匠兄弟 (C) Taito 1989¶

木匠兄弟 (C) Taito 1989

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：dondokod

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────────
Ａ：攻击
Ｂ：跳跃(Ｂ按住的时间长短决定跳的高度)

[ 道具介绍 ]¶

[ 道具介绍 ]
───────────────────────
蓝色瓶：提高移动速度
棕色瓶：力量+1 
        力量值越高，被投掷的敌人可穿越的障碍物
        越多
    Ｐ：力量最大
毒苹果：力量-1
黑锤子：攻击可直接杀死敌人
蓝锤子：锤子可投掷
棕色书：炸晕所有敌人
黑色书：杀死所有敌人
魔法球：魔法攻击全屏
  国王：时间暂停

[ 里关密码 ]¶

[ 里关密码 ]
───────────────────────
第1关进入隐藏房间后，在左上角小台子处向左攻击
会得到另一把隐藏钥匙，可进入一个隐藏房间。
该隐藏房间内按特定顺序击破瓶子吃道具，
可直接跳到里关(第51关)。

密码：粉心、蓝桃、方片、梅花、
      方片、粉心、蓝桃、梅花
吃道具的顺序可简单记为3412 3412 
(两列道具皆从下到上依次标注为1、2、3、4)

[ 隐藏里关密码钥匙 ]¶

[ 隐藏里关密码钥匙 ]
───────────────────────
共5把隐藏里关密码钥匙，当攻击到隐藏钥匙的地点
时，会闪光。清完隐藏房间的敌人会出现宝箱，
攻击宝箱会飞出道具，其中有一个水晶球。
收集到4个水晶球后，若在第43关进入隐藏房间会
显示进入里关的密码。

 第1关：最上方的平台中间，跳起攻击。
第12关：最上方右边的平台中间。
第23关：最上方平台的左边。
第32关：平台组成的脸图形的左边那个向里凹的地方
需要跳到最顶端，在下落的过程中才能攻击到。
第43关：最下面一层右下角的左边一个身位。

[ 隐藏游戏关钥匙 ]¶

[ 隐藏游戏关钥匙 ]
───────────────────────
当攻击到隐藏钥匙的地点时，会闪光。
游戏就像下五子棋，只要成行就有成绩，
不同的成绩有不同奖励。

 第5关：右上角台子上向左攻击
第16关：画面右下角。
第27关：画面左上角那个敌人的上方。
第36关：画面右下角。
第47关：画面中央稍偏左那个小台子。

第52关：左上那个台子上，跳起来向右攻击。
第63关：画面左上方。
第72关：画面中央最上方。
第84关：2P得分下方的小台子处向左攻击。
第94关：画面中央。

[ 隐藏分 ]¶

[ 隐藏分 ]
───────────────────────
用锤子攻击同伴、宝箱、公主、20、30、40关的BOSS
每下是10分。

第20关和第70关BOSS被击败后落下的白帽子是1000分。

[ 红色泰迪熊 ]¶

[ 红色泰迪熊 ]
───────────────────────
关卡演示中时，按1P的Ａ(3次)，
画面右下角就会出现。
