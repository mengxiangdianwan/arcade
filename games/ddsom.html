[ 一招八式 ]¶

[ 一招八式 ]
───────────────────────────────
游戏连招设定为一招八式，由已故玩家“电光影里”提出。

[一招八式的相关名词和知识]
1秒钟 = 60帧，游戏里的计数单位是以帧为单位的。
用武器打中敌人时，会产生2个重要的参数：打击值 和 相应计时器。

1、打击值描述了敌人的承受力。敌人连续被打时，它就会不断累加，
   达到8时，敌人就会倒地，无法继续连段。
2、产生打击值的同时，游戏还会设置1.5秒钟的计时器，它会逐帧递减
   ，到0时，打击值就恢复为0。
3、非武器的攻击(道具类、空Ｄ、→→撞)，不产生打击值，也不影响
   计数器。

各攻击动作和方式对打击值和计时器的影响：
        攻击动作                  产生打击值(式)  相应计时器
站立时的起手攻击Ａ动作                 3          设为1.5秒
其他任何用武器的每一下攻击动作         1          设为1.5秒
道具类(飞刀、锤子、箭)                 0          无影响
非武器攻击(→→撞、脚踢、头顶、身体蹭) 0          无影响
特例：火剑/雷剑/冰剑的任何攻击         2          设为1.5秒

注意：多人围攻一个敌人，敌人打击值的叠加是同一个，不会分开计算
。比如，战士牧师精灵一起打特尔阿林，大家一起Ａ，特尔阿林挨了战
士起手Ａ(3式)，又挨牧师的起手Ａ(就变→式)，再挨上精灵的起手Ａ
(共9式)，直接就会倒地。 

突破8式限制的简明道理你只需要牢记2点，一切就尽在掌握了：
1、敌人最大承受打击值为8，满8就倒地。
2、让打击值恢复的唯一办法只有时间，撑过计时器的1.5秒，就赢了。
   破8式限制的要点就是撑过1秒的倒计时器，你只要有本事让这一秒
   撑过去，还要保证这1.5秒内敌人处于受创或无法动弹的状态中，一
   旦打击值回到了0，你就能继续连招打敌人。
   既要维持敌人继续受创，又要不能再增加打击值，最简单的办法就
   是丢道具，丢3-4把刀，丢几个锤子，这是最常用的手段；另一个办
   法就是使用空Ｄ和→→撞，想要Ｄ和→→连接紧密还需ＢＢ取消，
   这个比较复杂，请参考法师穿越连击。

对杂兵的破8式连招，要求可以放低很多，它们浮空落地后还会弹地，
所以你可以等它们弹地再起来时继续连，因为落地+弹地的过程，这时
1.5秒钟已经过了。 

[典型的破8式连招例子分析]

第一类 简易型：
1.战士↓↑＋Ａ无限：靠的就是↓↑Ａ第2段的高滞空，敌人下落时间
  足够长过1.5秒，于是就可以再次连招。
2.盗贼的ＡＢ/↓↑＋Ａ火烧：把敌人烧起来，等它落下时，那就1.5秒
  过了。
3.火剑无限连：靠的是火焰把敌人烧上去，再下落下来，这样一个时间
  段，过1.5秒。(不过这样无限实在是太慢啦!)
4.→→撞/空Ｄ 配合杂兵弹地：不要一次性打满8式，用→→撞一下敌
  人或者空Ｄ，然后等它在地上弹一下，这样时间就过1.5秒了，你再
  接着连就是了。
5.利用锤子的眩晕特性，让敌人晕过1.5秒。不过除了视频里的招，
  其他都不好使，因为敌人晕了，你打他会浮空。

第二类 复杂点的：
1、ＡＡＡ ＋ ＤＤＤ 循环：连续打击之后，投掷3-4把飞刀撑过1.5
  秒。丢锤子的话，需要偶数的打击hit。(理论上2刀就可以，但时间
  需要极为准确，很难很难。)
2、→→ＢＢ穿越+空Ｄ：主要用于法师的无限折返连击，用2轮Ｄ蹭
  BOSS撑过1.5秒。主要用于BOSS战，杂兵的话还是上面的空Ｄ＋→→
  省事省力。其他角色不建议这样。

[ 连击取消系统 ]¶

[ 连击取消系统 ]
───────────────────────────────
本游戏大胆地引入了格斗游戏的连击取消系统，令游戏的爽快感大幅上
升！只要操作得宜，一串连击取消就可以送一个小兵上路了。所以连击
取消是必须掌握的重要系统，使用过程要注意以下几点：

　　1、连击取消的等级：
　　第一级：“普通技”、“吹飞攻击”和“奔跑砍杀”
　　第二级：“重击”
　　第三级：“突进必杀招”
　　第四级：“倒地追击” 
　　第五级：“对空必杀技”
　　第六级：“特殊飞行道具”只限精灵

　　只有等级高的招式可以取消等级低的招式。“无敌技”可以取消第
一级，但“无敌技”不能被取消！

　　2、有时敌人会在连击取消过程中“晕厥”(不是有星星的“晕迷”
)，“晕厥”状态是不能追打的，要注意。不同敌人的“晕厥”点不同
。一旦敌人进入“晕厥”状态将全身无敌，你能做的事就是等他掉下来
进行“倒地追加攻击”。

[ 武器攻击力和伤害设定 ]¶

[ 武器攻击力和伤害设定 ]
───────────────────────────────
    每种武器都有自身的基础攻击力，这是固定不变的。然而影响最终
输出伤害的因素还有许多，比如牧师的加持祝福状态，带了手套手环，
以及出现暴击等等情况。
    每种武器有一个基础伤害，比如传说剑攻击力高，短剑攻击力低，
这个是从基础伤害的角度来说的，这里要引入另外一个全新的概念：武
器的攻击力上限。
    每种武器存在一个输出上限，你使用它，不论怎么加状态，带手套
也好，必杀技也好，最终都不能超出它的上限限制。有的武器基础伤害
不错，但是攻击力上限很低，所以注定是垃圾。

[整个伤害计算过程]：
1、武器基础伤害经过乱数程序，得出基础伤害+偏离(-3~+1)，但最小
   不会小于4。(暴击的话，伤害 +5)
2、经过牧师的强化术魔法，增加伤害的50%。
3、判断是不是必杀技(↓↘→Ａor↓↑Ａ)，增加伤害50%，再修正一些
   (减少一些，4P模式修正的很厉害，伤害最小为4）。
4、是否屠龙剑，对龙类(黑龙、绿龙、红龙、SYNN)加50%伤害。
5、是否神圣复仇者，对不死系杂兵512点伤害。
6、是否有牧师的祝福术魔法，是的话上一步中的伤害增加25%
7、有手套伤害+3，有手镯伤害+1
8、伤害封顶验证，各个武器自己限制。

    所以，重剑的上限为28，不管牧师怎么加持，矮人也发挥不出多少
伤害，打死了也就28。而矮人的斧子封顶上限是99，所以用自身斧子时
一旦加了祝福术和强化术，那效果就天差地别了。
    此外，在4P模式中，感觉敌人特别耐打，就是因为第3步的修正，
伤害很低，往往只能打出4点的伤害，这是系统最低的保证。
    当法师在打血很多的小兵时(超过100的)，出现了100%效果的会心
时，有时发现竟然死不了，原因就是受了他自身的杖上限99的限制。
会心的这个情况有时上限会限制，有时又不限制，可以一下戳死，暂时
不明。

[武器上限的解析]
一般的武器上限都是99，那么上面所说的神圣复仇者伤害512是怎么
回事呢。它其实是这样的，最终的伤害分成2段来看，(256的整数倍)
 + (0-255的部分)。只有0-255的部分会受上限限制，256的倍数不会，
所以拿着复仇者用脚踢也能靠512的加持伤害打死任何骷髅和食尸鬼。
而一般的其他武器，不管怎么搞，伤害都在0-255之间，也就无法脱离
上限的控制。

[ 魔法伤害的叠加计算公式 ]¶

[ 魔法伤害的叠加计算公式 ]
───────────────────────────────
    暗黑秘影中，有些法术在打到多个敌人时，威力会变的很大，签名
耳环的精灵在第3关人形魔蝎BOSS处，连小兵带BOSS一起电，就可以发
挥出可怕的破坏力，直接能把BOSS电残废。然而，有的时候明明电了很
多敌人，但这个电魔法造成的伤害又很小，到底是为什么呢？黑法师的
陨石破坏力惊人，但是装备了耳环和红杖之后，怎么伤害就很小了？

    魔法伤害的计算过程是：
1、判断是否有法师的杖类加持，如果有就进行对应的加成，具体如下
   ：巫术之杖(红杖)伤害+50%、魔力之杖(蓝杖)伤害+25%、元素之杖
   对法师5级魔法伤害+25%。
2、判断是否带耳环，如果有魔法伤害就+25%(增加基础魔法伤害的25%)。
3、判断是否持有短杖类魔法装备道具：
   火魔棒-火系伤害+5，电魔棒-电系伤害+5，冰魔棒-冰风暴伤害+5。
4、以上计算过程中，伤害叠加型的魔法，任意一步伤害超过255的话，
   就被减去256；如果伤害为0，那么就置为1。

    以上说的魔法伤害，包含纯魔法和可以释放对应魔法的戒指，但不
包括牧师变蛇魔法和虫灾魔法。注意杖和耳环虽然是分先后计算，但它
们都是以魔法基础伤害来做运算的。这个和物理伤害计算不同。
    假设法师持有巫术之杖、耳环、电魔棒，电基础伤害40。那么第一
次的伤害=40+40×50%+40×25%+5=75。但物理伤害不同，加持了Bless
和Sriking魔法，武器基础伤害20，使用Ａ攻击时：第一步Sriking加成
算20×1.5=30，然后少量修正，假设修正为26；Bless魔法加持就是
26×1.25接着运算，它不是以基础伤害作为运算基准的。这是物理和魔
法伤害计算中的很大的差异。
 
[只有耳环时的魔伤计算] 
    魔法伤害的计算这里造成伤害的关键是耳环，它的作用是魔法伤害
+25%。意思就是每一次的伤害都比上一次增加25%。当伤害超过255时，
则被减去256。(如果为0，则伤害置为1)
    在暗黑秘影中，每一个魔法均有2个伤害值，一个对应小兵，一个
对应BOSS，对BOSS的伤害要低一点。
    耳环的作用+25%，使得伤害变成了一个等比数列，公比是1.25，
下次的伤害是上次的1.25倍，超过255的话，则减少256。电魔法出手自
后，每电到一个敌人，游戏根据它是第几个敌人，把2组伤害计算出来
，然后再看它是小兵还是BOSS，决定伤害。决定伤害的关键是第几个电
到，而不是一次电到多少个。

[只有电魔棒时的魔伤计算] 
    电魔棒的作用是电系伤害+5。下一次的伤害比上一次的多5。

[同时持有电魔棒和耳环的魔伤计算] 
    游戏会优先计算法师的武器魔伤加成，然后耳环，最后是电魔棒。
    假设电魔法此时的基础伤害为30，那么第一个敌人受伤害就是
30 X 1.25 + 5 = 42，再下一个敌人的伤害就是 42 X 1.25 + 5= 55。
如此反复。运算中任何一步超过255，都会减去256。

[其他叠加魔法伤害计算]
    同一个魔法暂停时间内，按先后顺序打到敌人的这种方式法术，都
会有上面这种伤害叠加计算效果(流星雨除外)，比如火墙、火球、闪电
术、魔法飞弹(同一颗)、石化术。全屏同时打中敌人的魔法则不会依次
叠加。

    黑法师流星雨魔法最容易伤害溢出。黑法师流星雨的基础伤害160
，那么他手持巫术之杖、耳环、火魔棒时，释放流星雨魔法的伤害如下
：
    伤害第一步，160 + 160 X 50%(红杖)= 240
    伤害第二步，240 + 160 X 25%(耳环) = 280。超过255，伤害溢出
变为 280- 256 = 24。
    伤害第三步，24+5(火棒)=29。
    所以这时黑法师的陨石只能打出小伤害29。
    本游戏中所有魔法都有随等级变化的攻击力表，唯独2个魔法是特
别的，就是流星雨和律令：死魔法，固定伤害160和70。

    ※电秒龙是怎么回事呢？其实是一个道理，红龙和SYNN在那种眯眼
状态下会被电魔法持续的打到。电秒龙其实是打了N段的结果，每一段
的伤害均符合上面的计算公式。

[ 敌人状态解说 ]¶

[ 敌人状态解说 ]
───────────────────────────────
    “暴击”状态---打击敌人的时候，会闪出特别大的打击火花。出
现这种现象时，伤害会提高(伤害 +5)，但敌人会因此而浮空，有的连
技就被迫中断。我们一般不太希望出现这种现象。

    “浮空”状态---敌人在站立的状态被“重击”、“飞跃攻击”、
“突进必杀技”或“对空必杀技”命中即进入浮空状态。“浮空”敌
人处于可追打状态，此时可用各种必杀技和飞行道具来进行追加，但
不推荐用直接普通攻击追打，因为普通攻击追打易使敌人产生“晕厥”

　　“吹飞”状态---敌人在空中或被“浮空”后再被“重击”、“飞
跃攻击”、“突进必杀技”或“对空必杀技”命中即进入吹飞状态。
被“吹飞”的敌人会像一支箭似的横飞出去，期间可撞倒阻挡它“飞行
”的所有敌人，包括BOSS！“吹飞”也是实现多人连携的重要技巧！

　　“晕迷”状态---敌人被重创、中了“恒光术”魔法或“恒光戒指
”的魔法后有一定可能性出现CAPCOM的招牌小“星星”。此时，敌人
进入完全无防御状态，大可追打之。

　　“晕厥”状态---敌人连续被直接的普通攻击命中、浮空时被普通
攻击命中或浮空后被同一种必杀技连续命中后会出现“晕厥”，一旦
敌人进入“晕厥”状态将全身无敌，你能做的事就是等他掉下来进行
“倒地追加攻击”。

　　“倒地”状态---当敌人被“浮空”或“吹飞”后，掉倒在地上处
于无防御状态，此时可以进行“倒地追加”攻击。

[ 战士分析 ]¶

[ 战士分析 ]
───────────────────────────────
战士
    战士可以使用多数武器。不会使用魔法，但攻击力和防御力很高，
只要有武器你就仍然可以战斗。耐力很好，适合初学者。

　　战士可以使用你所能在游戏中找到的绝大多数武器。HP不如矮人，
但是他的耐久力很好。他的必杀技是前冲加重斩，可以接对空技 - 
升龙斩。在打出升龙斩的时候继续按攻击键可以在接一个空斩。击倒攻
击是一个向前的重踢。后踏回避是一个向后小跳，后跳过程中你可以连

※二刀流模式简介
战士使用双剑的时候很多招式都会自动出2剑，左右手各一剑。在伤害
计算上，它是非常特殊的。首先短剑的伤害算的是Lv2长剑的伤害（12点
），然后才是自身长剑的。这是因为游戏程序的错误，它没有读取战士
短剑的攻击力，而是读了2级长剑的伤害，所以初期使用短剑打怪，会
明显感到伤害高。
    战士双剑模式的攻击形态：动作外观上看，是先长剑，再出短剑；
但在攻击力的计算上，它总是第一下算短剑的伤害，第二下算长剑的伤
害。如果只有一个动作，那么都是按短剑伤害来算。
    因为游戏程序的错误：用短剑的基础伤害算LV2长剑（12点），用
长剑的算LV1长剑8点。也就是说哪怕你后期到了LV3和LV4长剑，你使用
双剑的话，依然是12+8的伤害。
    举些例子：
双剑的ＡＡ攻击：第一下12点基础伤害，第二下8点基础伤害。
战士双剑的↓↘→＋Ａ滑斩：
总是先按短剑算，伤害=12*1.5倍必杀技加成=18
战士双剑的↓↑＋Ａ二段：
第一段 12*1.5=18（按短剑出手算），
第二段 8*1.5=12（按长剑出手算）
所以，越玩到后期，感觉双剑越来越无力，原因就在于此，固定12+8的
伤害组合。
    在游戏中，双剑比较有用的地方是打第5关和第6关的几个BOSS。
过了lich之后，请把你的短剑的丢进垃圾桶去，双剑模式从第7关开始
就没有用了。

，甚至扫平一整队怪兽或把BOSS送西天。不过他的HP很短，而且他不能
防御，所以用他时要格外小心。魔法师的基本动作和别人有所不同。他

不要用常规攻击，而改用重击，有时可以立即干掉一个小喽啰，或把

的效率。什么魔法要事先准备好，什么魔法用在什么特定场合等。用魔
法师时你需要很冷静，否则你在手忙脚乱的找魔法时肯定会死翘翘的。

[ 神秘宝物的出现条件 ]¶

[ 神秘宝物的出现条件 ]
───────────────────────────────
即使不打败敌人也会在攻击的过程出现的宝物：
“眼魔之眼、曼帝考尔之皮、移位兽之皮、龙角、龙鳞”

宝物出现的机率计算方法：

　　32分之0的概率为最初设定，每1发必杀技能命中的话，便把宝物出
现32分之一的概率增加1个分子，同理命中2发必杀技宝物出现的概率
32分之2，以后每命中必杀技分子增加1。

　　因而如果命中32次必杀技绝对出现。32次碰不出是奇迹。

　　再者，所谓必杀技，除了通常攻击和投掷系武器以外的全部的攻
击好像都符合要求。总之以魔法，短跑冲撞也有效。如果敌人濒死持续
试着做冲撞也好吧。

　　魔法箭的攻击段数多，分子一口气增加几点，十分容易出现打出宝
物。

用掉的魔法点有两种方式可以恢复！！
　　１、通过挥斩怪物或打开宝箱获取魔法卷轴，一个卷轴可以恢复一
个魔法点。卷轴的颜色表示魔法的等级，从低到高依次顺序是：
蓝 -> 绿 -> 黄 -> 红。越强的魔法卷轴颜色越深，卷轴分1-8级，在
组队游戏时，考虑到谁去拿卷轴将是非常的重要。
战士，矮人和小偷不能拿卷轴，所以不用担心。
    ２、升级。当你升级后，你的属性会提高，也可能学会新的魔法，
增加武器等级或增加hp值，此时你所有的等级的魔法也会同时加满魔法
点。另外一点是恢复的魔法点是看你使用了多少来恢复的，不占用你在
游戏途中自己找到的魔法点。比如你是一个14级魔法师，开始时你有5
个1级魔法和4个3级魔法，途中你找到2个1级魔法，但用掉了所有的3级
魔法，那么当你升级至15级时，你会有7个1级魔法，4个3级魔法。精灵
不会进化到10级以上，但她在清关以后也会恢复魔法点。同一级别你能
拥有的最高魔法点数为9。
      

魔法另外的用途：
    魔法并不只是给怪兽带来伤害，有时可以让你避开攻击如
Deadfalls或火球或一些怪兽的特殊攻击，因为在释放魔法时你是处于
无敌状态的。最好的例子是暗影精灵法师和红龙的火球。
只要时间得当，一个小小的魔法箭可以让所有同伴们免遭涂炭。注意的
是不同的咒语持续的时间不同。不过你仍然得小心你的魔法可能失败，
最好的例子(也是唯一的情况)就是眼魔，没有任何魔法会对其中央的眼
睛产生效果。

　　有的时候你会发现你的Ｄ键没有用(不可施放魔法和使用道具)，那
是因为对方在施放魔法或在使用特技，但这种情况只在某些BOSS面前会
发生，也就是暗影精灵兄弟，红龙，纳加帕和席恩。
具体情况是暗影精灵兄弟施放火球、闪电，巫妖Lich放流星雨，Nagpa
使用闪电，和红龙/席恩使用龙息的时候。这种情况下，找个地方躲躲
比较好。在上述不能使用键的时候，牧师可以立即取消这个效果，方法
是使用无敌技Turn Undead (A+B)。所有队伍中的同伴就又可以使用键
了。在龙从中央喷火时，使用这招极为有效，让所有人都逃到一边，然
后牧师用turns undead，然后精灵/法师开始放魔法，即便龙在喷火也
会因被魔法击中而抵消对你带来的伤害。

要学会掌握终极魔法你需要首先具备以下3个条件：

1) 至少3个游戏者；
2) 达到总共200，000，000 XP；

BOSS：Dark Warrior I：

    2.会进入加速状态。
    3.躺地被追击受限制4次。
    感知：无法看破隐身，但可以感知正面或侧面靠近的隐形生物，

着。

可以了，有5个投锤左右就搞定了。

[BOSS：Man Scorpion(s)]

    1.前螯自带攻击判定，碰到即受伤。
    2.躺地被追击受限制4次，第4次被攻击立刻强制蹦起来。
    3.浮空之后，不可被打到。
    弱点：倒地起身时没有强制攻击能力，可连续追打。
    其它：会格挡，挡了后没有特定反击招式。
    注：屏幕下一满行HP为352。敌人出场HP与时间有关。

    如果你是单打，只有一个红色的Man Scorpion出现；否则还会有一
个绿色的Man Scorpion登场。同Man Scorpions一起登场的还有一些穴
居人，当所有的穴居人被杀后，会出现一批食尸鬼，如果它们又被消灭
后，穴居人会再次增援，如此反复。Man Scorpions一般是跳起来用尾
巴攻击你，它的镰刀会做回旋攻击，尾巴会抓住你然后用毒针扎你，另
外象冲刺，或发射石化射线，闪电波是它的特技。有时候它会向上跳起
，着陆时会引发爆炸。不过最具威胁的攻击还是它的回旋镰刀，可以四
处乱飞击伤所有的人。

打法：单打的话在BOSS登场的位置等候，BOSS出现就投锤飞刀之类，然
后放倒火瓶，只要BOSS不出回旋镰刀就没啥危险，BOSS起身一般ＡＡＡ
Ａ就可以打到，要是飞了起来就后退一点防御即可。穴居人(鱼人)的攻
击基本可以忽略，你只要和他上下错开一点点就不会被他打到，食尸鬼
会自己调整和你的相对位置，所以要注意。
    估计有绿色蝎子王登场的情况下先派一个人到预定位置投锤飞刀
ＡＡＡＡ弄死那只绿的，同时其余人对付红的，不求高伤害但求放倒。
千万不能出现两只蝎子王满屏乱窜的情况，如果两把镰刀飞舞那更是噩
梦。总之速度搞死那只绿的，他的血其实也很少。
    战士和矮人注意你们的对空技打不全，尤其是战士，没有无限连的
可能。 
    牧师注意恒光术晕住BOSS和补血，食尸鬼就超度掉，穴居人(鱼人)
就捆住，捆住不要打，就当不存在，可以很长时间无干扰打BOSS。
    法师注意全屏法术控场，还是放倒后棍子伺候，需要注意的是镰刀
飞舞的时候千万不要为了自己安全放魔法放倒BOSS，那样的话镰刀会一
直飞到BOSS站起来，很多队友就倒霉了，法术害人这就是一例。
    精灵还是上面的综合，法术放倒之后对地攻击，BOSS起身就挤住
ＡＡＡＡ，BOSS飞了就冰风雪或者防御。
    盗贼还是那句话，多偷吧，不过可能由于路线原因没咋遇敌，那就
看自己的本事加运气了。

特殊打法
    1、电魔法：本关BOSS之前有很多小蝎子，这就符合了电法术打出
高伤害的条件，所以精灵有耳环或者法师有电魔棒(谁说法师不能电，
那么多蝎子电棍累加的伤害也不小)，那么全员就不要动小蝎子，上去
后站下面箱子靠前一点的位置，面向左边，先等一回，小蝎子会在左边
排排队，之后引出BOSS，不管是一只BOSS还是兄弟俩集体出来，一个字
，电它，你站在右下角，先电左边然后电束上移，就打到BOSS了，有时
候一次电不绿的，那就转而面向右边再来一电，基本绿的就死翘翘了，
剩下的红的也就半血左右了，大伙上吧。
    需要说明的是精灵的耳环有可能打出极高伤害(高到两点秒杀)，也
有可能打出破255的极低伤害，关键就是电到几只小蝎子再电的BOSS，
如果不想赌这个博的话就适当的打死几只小蝎子。法师没这个烦恼，因
为他拼了命也就打大半行血。至于说精灵的电第一关全部扔在车子上了
怎么办，过关途中有巨多穴居人(鱼人)，那些家伙会概率掉1到4级法术
，你就期待着那本深绿色的卷轴吧。
    2、盗贼的隐身：这个打法适合单打，进入BOSS场景之后共有三个
箱子，这个就是本打法的关键，上面的箱子盗贼开会有一个隐身术戒指
，盗贼使用隐身术戒指隐身的情况下比较特殊，只要盗贼不做出解除隐
身的动作就会一直隐身下去，没有时间限制，而别的隐身术都有一定有
效时间。这个打法的运气成分就在下面的铁箱子，如果你看过去上面有
个红色骷髅，那么这个打法就没法实现了，如果没有，说明下面的箱子
不会被抬开，也就是可以无限次抬起来。
    具体做法就是用隐身术戒指隐身之后用箱子一直砸BOSS，砸到死，
抬起箱子和扔箱子不会解除隐身状态，你可以尽情欣赏BOSS那茫然的神
态，慢慢的用箱子砸死他。虽然每次伤害很低，但是积少成多总会砸死
的，此方法安全无比，就是比较费时间，需要耐心，如果你不小心空按
了一个Ａ，那还是准备投锤飞刀吧。另外，本BOSS有发现距离很近的隐
身者的能力，他会把你抱在怀里一顿咬，但是没事，不要慌张，受伤不
会解除隐身状态，咱们可以起来继续砸他。


★Stage 4 — Skies over Aengmore：安格莫尔的天空

    经过前几关后你可以有机会去趟商店了。多存点燃烧瓶、飞刀和投
锤，矮人应该多存点箭而不是飞刀。这关你会乘坐一座飞船，暗影精灵
首次登场并攻击你，当你跑到船的最右边时，去开一个宝箱(你需要有
盗贼或一把钥匙)， 你将得到一个药水，有时会是一个治疗戒指。
当一部分暗影精灵被消灭后，会出现一个骑在双足飞龙上的暗影精灵，
他会使用火球魔法。你可以通过施放魔法飞弹，或跳跃或侧滑来回避火
球。总共有两批暗影精灵，之后BOSS登场。

[中BOSS：Harpy]

    1.会空中受身，伴随招式免疫。
    2.追击攻击限制4次，第4次被打立刻起身。
    弱点：属于轻型BOSS，打击感类似杂兵，可以从地上捞起来打。
    注：屏幕下一满行HP为352。敌人出场HP与时间有关。

    当杀了最后一个暗影精灵时，上空出现Harpy，如果你离它降落的
地点太近，你会被攻击到。它基本上用挥棒来攻击，向你冲刺，或飞起
来发射羽镖，有时向你俯冲或把你举起来往地上砸。有时会飞起来，拍
打翅膀产生强大气流，使你步履艰难。

打法  一般选择在飞船右边遇BOSS，事先先把那个箱子抬起来扔到一边
，免得将来连招的时候出现蹲下开箱子的尴尬局面。没有刷分需求的话
就在BOSS飞下来之后投锤飞刀伺候，之后就是各种对空了，BOSS的大部
分攻击都能被防御，但是它高高飞起下来抓你这招防御不可。这个BOSS
还有个特点就是它被火瓶烧了之后从倒地状态变为浮空状态，可以继续
连招。
    鸟人空中受身之后，你不能使用相同的招式来打她，必须变换招式
。比如鸟人空中被玩家的跳Ａ打一次且受身后，就再也打不到了。交替
使用Ａ和跳Ａ攻击她就能避免这种打不到的情况。
    战士的升龙斩对BOSS很有效，矮人的升龙强制倒地之后可以蹲下扫
斧子，BOSS飞起来就继续升龙，平时注意防御即可。
    牧师注意辅助状态和恒光术，BOSS飞在空中牧师可以跳起来Ａ下来
继续连招。
    法师还是那样，注意控场和扫棍子。我一般开场蹲在BOSS上面一点
点，先扫8棍子，之后再扫，飞起来就先翻转重力拉下来，然后切换法
术，飞起来再控下来，一般冰风雪不用放完就死了。

★Stage 5-B — Forest of Despair：绝望森林

    选择这关的理由一般就是里面隐藏的一堆装备，眼魔身上的眼珠等
，当然这关也可以刷分，也可以速通。

    适合的角色：强烈建议精灵选这关，别的角色一般无所谓。

    这一关要么很短，要么很长，主要看你走哪条路了。在这里我提供
一条我们常走的线路，如果你有其它的选择，过程会有所不同。

　　在一开始就有很多巨型甲虫向你飞来，所以尽快向右边冲，跳下山
坡后进入山洞，当然你有耐心也可以蹲下来慢慢等虫子全都跑光也可以
。山洞里多数是哥布林，狗头人，豺狼人和一些枭熊怪。有一个宝箱如
果你是矮人/战士/盗贼/精灵，打开后会得到一把火剑，尽可能让战士
或盗贼来拿。

　　走到洞的出口时要蹲下，因为有大量的虫子会飞出来。附近有一个
超级治疗药水，不过拿的时候要小心，太靠右的话，会把BOSS引出来。
等虫子跑光后，进入一个看上去像石头一样的洞窟。这里有个宝箱里面
有把诅咒之剑2。让牧师反复捡7-8次，它就会变成神圣复仇者。在右下
方的箱子里有一个帽子，当你出了洞之后，继续向左走，然后跳上去，
你会看到一块大石头。

　　让除了魔法师以外的任何人去推这块石头，从左往右(连按键可以
推得快一点)然后会出现另一个洞。里面的宝箱里如果让魔法师去开会
有一个3级卷轴。这个洞里只有哥布林把守。

　　继续向下走回到游戏刚刚开始的地方，摇杆按住↓不放你会掉入一
个秘密通道，这个秘密通道里有很多哥布林，狗头人和豺狼人。在画面
最上方的宝箱里有一把雷剑(你必须让战士，精灵，矮人或盗贼去开才
会得到)，如果你没有拿火剑，或者你让错误的人来拿火焰剑的话，这
把剑就不会出现了。战士或盗贼必须拿着这两把剑中的一把，否则
BOSS会很难打。

　　出了出口后向左走，你会见到一个洞窟，里面有两个木箱被巨型甲
虫把守着。若甲虫被消灭了，骷髅兵会出现。等所有敌人全被消灭了后
，让牧师去开最上面的宝箱。里面有一枚驱邪戒指，也许整个游戏中你
只能找到一枚这样的戒指。

　　出了洞之后，一直向左走到底。那里有2个箱子。木头的装的是
超级治疗药水，铁的装的是风神指环。
好了，是时候结束这关了，你可以选择一直向右边走，直接打下面的
BOSS。或者从我们最后进的那个洞的洞口，从那里跳上去，那就是我们
最初进的那个洞。再向上跳，那就是有神圣复仇者的那个洞窟，向右走
BOSS就登场了。


[BOSS：Beholder]

    1.被打超过4次时，会爆圈圈。
    2.正面的大眼睛瞪视可以取消玩家的魔法。
    注：屏幕下一满行HP为352。敌人出场HP与时间有关。

    Beholder是一个漂浮在空中的大眼怪。当它那只大眼瞪着你的时候
，你不能用魔法的。它的基本攻击是咬和冲刺，有时它会射出一些奇怪
的射线来攻击你。那是石化魔法， 当心了。那些头上长的小眼睛会让
你催眠，其它没有什么用。另外它会射出360度的射线(威力较大)如果
它连续攻击，那些小眼睛也会发射一些小的火圈来攻击周围离得较近的
人。

打法 这个BOSS处于飞行状态，所以没有倒地一说，他另外一个重要特
点是连续被攻击5HIT之后会强制反击，所以要点就是每次只打4次以下
的攻击。从上面遇见BOSS的话它开场会释放一个石化法术，需要注意
。一般建议从下面见BOSS，下面你可以慢慢移动直到见到一个灰色的
BOSS，再往前走就开战了，所以你可以提前做好准备，一见面就是用
一个魔法道具(一般是火神瓶)把它困在角落，之后就是如何继续困住
它的问题了，只要控制的好这个BOSS就完全没有攻击力了。
    以下都是以下面路线说明如何控制BOSS的，困住BOSS之后每攻击
四次BOSS会后跳一下，方便我们调整来继续攻击。
    战士 可以用长剑或者魔法剑的ＡＡＡＡ，战士的Ａ起手对空性能
很好，基本上BOSS不能从你头上飞出来，如果装备了短剑的双持，那
么就更简单了，只需要ＡＡ，就是4刀，毫无操作压力。
    矮人 4HIT为前冲斩连↓→＋Ａ，滑铲或者ＢＢ滚回来，继续。
    盗贼 (↓→＋Ａ)*2，ＢＢ滚回来，继续。
    牧师 这个比较麻烦，需要一定操作，具体为前冲撞，↓→＋Ａ，
↓↑＋Ａ，滑铲回来，继续。撞的时候一定要勇敢。
    法师 基本无压力的Ａ，虽然只有一下，但是法师的棍子对空性能
也很好。
    精灵 这个压力就比较大了，必须是长剑或者魔法剑，Ａ，→＋Ａ
，Ａ，→＋Ａ。精灵的Ａ控制范围远不如战士，所以要小心，另外如果
HP多的话可以用Ａ＋Ｂ。

     如果BOSS跑了出来，那么就得换打法了，当然BOSS有时候会傻到
自己回去，那有另说了。要点还是不要超过4HIT，以盗贼为例，BOSS背
后Ａ一下立刻↓→＋Ａ穿过BOSS到它前面，这时候它眼睛是闭起来的，
稍微等一下Ａ之后↓→＋Ａ再穿回去，如此反复。
     也有一个较为安全的打法，就是蹲在BOSS上下附近，但是不要同
一水平线，偷空就打它几下，它则打不到你。注意用滑铲靠近BOSS，
与BOSS距离太远会被射线打到。
     对BOSS释放法术问题，尽量在BOSS背后放，或者在BOSS被攻击闭
眼的时候使用，如果BOSS自己闭眼你有在它面前施放，那么他会瞬间
睁开眼睛取消你的法术。法师如果有六七个魔法飞弹的话可以抓住一次
BOSS背对你的机会一口气放完，反正是4HIT不用担心引起反击误伤队友
，而且连续飞弹基本必出眼珠。
     BOSS眼珠掉落问题，把BOSS困在角落里四次攻击的打法基本上打
不出眼珠，除了精灵的Ａ＋Ｂ，要出眼珠就一定要持续攻击，当然这也
很危险，值不值得就自己考虑了。

特殊打法 速射 下面见BOSS的话场景比较长，可以把BOSS引到左边一堆

置，这个打法连加速都不需要开。 



★Stage 5-C — Forest Bridge：通过森林之桥

    选择这关的理由基本就是速通，因为这个路线会跳过下一个BOSS，

[BOSS：Green Dragon]
    这是游戏中最弱的龙。飞行状态没有倒地，龙类共有的三次龙息
，分别是在满血见面时、1/2的HP、1/4的HP时喷3次。龙息可被法术取
消。
    
打法  全员开场奔到右下角，有投锤就扔，没有就打，战士盗贼升龙，
矮人跳Ａ，精灵牧师Ａ，→＋Ａ，其实它的血很少，一会就第二次第

特殊打法
    1 跳射 精灵和盗贼用，如果精灵有火箭则更好，开场站在画面最

    2 召唤兽瞬杀 本游戏中死的最快的BOSS不是被精灵一个电秒杀的
车子，而是这只杯具的被瞬杀的龙，它登场的烟雾还没有散去自己就挂
了。
    首先要有奇迹之蛋，这里有奇迹之蛋只有一个可能，就是盗贼偷枭
熊怪的蛋在美女商店换的。做法是见BOSS之前，有蛋的人和有单体辅助
法术的人(牧师精灵)站在一起，蛋与辅助法术同时按，这时候辅助法术
会取消扔蛋的动作，身上蛋还在，但是已经扔出去了一只，如此反复可
以制造一只召唤兽军团，它们排好队只需要向着BOSS飞扑一次BOSS就挂
了。


★Stage 6 — The Battle of Strong Oak：大橡树要塞之战

    这关一上来就和BOSS对战了。如果之前选择了让精灵引路而通过了
毒雾森林，那么这里将跳过此BOSS直接到达Stage6的第三个关卡。

[中BOSS：Manticore]
    Manticore象长了翅膀的狮子，带着长刺的尾巴和一张人脸。基本
攻击有挥爪和冲刺。有时会飞起来发射尖刺(跳起回避)，有时会飞离屏

    尾巴带刺，转身时，会造成尾巴伤害判定。会格挡，挡了之后会必

    食人魔兄弟至少还有个石头可以打打逆向，曼蒂考尔这个悲剧的所
有攻击都可以被防御，也就是说只要你防御BOSS就完全拿你没办法。
唯一能看的就是霸体攻击，不过反应不是太慢都来得及处理。

打法 可以攻击加盾防，也可以小火瓶慢慢烧，BOSS起身可以用飞刀投

和飞扑就盾防，盗贼注意后滚翻，法师直接一个飞弹打下来。由于
BOSS身体较长，可以在估计BOSS起身的时候用空中↓＋Ａ起手攻击，
一般至少2HIT，盗贼可以做到4HIT甚至更多(再次怨念精灵那废柴招)。

特殊打法 牧师和矮人可以蹲下Ａ目压BOSS至死


    当你打死了Manticore后，一块巨石自动移开露出一个隐藏山洞。
剧情将在那继续发展。洞里是一条很长的通道，里面有两个宝箱。

5)Dead Fall 死亡落石：

[ 陷阱介绍 ]¶

[ 陷阱介绍 ]
───────────────────────────────
1、飞箭：当你开启箱子后，将会有4支飞箭从里面飞出来，安全的做法
是站在不会被箭射到的地方，比如正下方，也可以跳起来躲避。如果你
是搬开箱子的，一般都不能倖免……好在伤害值是各类陷阱中第二低的
。

2、火焰：当你开启箱子后会有多丛火焰从箱子中撒开来，躲避的方法
是立刻从火圈中跳出去……如果装备加速鞋的话，可以直接跑出去，
伤害值比飞箭要高。

3、落石：当你开启箱子后会有落石从天而降，伤害值排名第二。躲避
方法是立刻往上方或者下方移动。

4、毒气：当你开启箱子后会从箱子里冒出毒气，中毒者会蹲地昏迷3秒
钟左右。伤害值是各类箱子中最低的，然而却是很讨厌的陷阱，因为毒
气后会从地上跑出几个骷髅头怪物，不杀它们的话会跟着你一辈子，要
杀它们那个响噹噹的盾牌能郁闷晕你……躲避方法是用钥匙开了箱子后
立刻往下方移动，然后设法抢走骷髅头们的盾牌以提高杀怪效率……

5、石化：游戏里最具杀伤力，甚至比龙BOSS还要让新手恐惧。伤害值
高，如果被光线照到以后在限定时间内不挣脱，会立刻game over。
躲避方法：若搬开箱子的请往上移动，若用钥匙开箱子的保持不动或者
向下移动，反正远离那束光线……若被光线照到，请立刻狂左右左右狂
摇，或者可以用我的方法，←↓→←↓→狂摇

6、弓箭：当你走路的时候会在某些特定的地方射过来弓箭，单机时由
于豹皮袍的存在一般人都不会注意如何躲避，但是连线时豹皮衣一般
只有一件或者两件，希望大家注意一些这些陷阱，毕竟被打破了手套和
防火戒指之类的装备感觉太差。躲避方法是紧记有暗箭的地点，到了那
些地点跳过去，或者用全身无敌的闪避方法比如矮人的Ｂ↓→＋Ｂ盗贼
和法师ＢＢ的，连线时刻等待装备豹皮衣的同伴把弓箭全引出来了再慢
慢踱步走过去嘿嘿嘿~所以连线时有豹皮衣的兄弟姐妹就要注意一下自
己的义务了，多多利用这个美妙的装备嘛！在乱箭中安然地散步感觉也
不错

7、钉板：在某些地方会有钉板从天而降，伤害值挺高的，躲避方法是
紧记哪些地方有钉板，以后要快速通过去，如果连线的话比同伴走得慢
，要看清楚情况，不要跟着跑一不小心被同伴引下来的钉板压一下子

[ 关于诅咒之剑 ]¶

[ 关于诅咒之剑 ]
───────────────────────────────
咒剑有两把。

1、眼魔那关场景最下方有一个洞穴里一群食尸鬼守护的，这把剑解咒
之后成为圣剑，对不死系(LICH除外)一击杀之，效果类似牧师超度的解
离，但是坏处很明显，打食尸鬼的时候超度之后啥也没有，而用圣剑杀
死仍然掉道具。解咒方法，牧师捡八次即可。本剑不开封印的情况下攻
击力极其高，等同于传说之剑。

2、奖励关红龙上面洞里有一把咒剑。解咒方法：挥剑计数80次。

有几点说明
   1)80次是总计数，可以分成几次挥，也可以分成几个人挥(一个人挥
几下换到地上别人捡起接着挥次数累计)。出骷髅算两次，不出骷髅算
一次。
   2)不费血解咒剑2的方法一，跳起来(各种跳都可以)，在快落到地上
时挥剑，这时可以听到挥剑的声音，但是并没有挥剑动作，可以不费血
解剑。(此方法盗贼不可以用)
    不费血解咒剑2的方法二，红龙下一关选火元素或者冰元素(就是不
能选暗黑精灵兄弟)，这关BOSS处有箱子可以开出飞行鞋。然后1P拿剑
2P去开箱子飞(飞起的时候一直按住左上或者右上不松手)，1P保证咒剑
在手的情况下用法术或者魔法道具把BOSS打死，这样游戏会卡住，此时
1P就可以不费血挥剑解咒了，这是个游戏BUG。等解完咒2P就可以松开
左上或右上，就会落到地上，游戏就可以继续了。这里的1P2P指的是双
人玩家，4人的时候保证开剑的小于飞的即可，比如2P开剑，3P或者4P
飞。

[ 结局条件 ]¶

[ 结局条件 ]
───────────────────────────────
每名角色都有四种结局。其实就是通关动画里，四种不同的文字交代，
并无任何动画的区别。
由1至4，层层判定结局。

○精灵/战士
  1.若手持寒冰法杖、传说之剑、雷剑、火剑、冰剑，必进入结局一；
    若没有，则看6个道具栏，道具栏内有投石索、传说之剑、雷剑、
    火剑、冰剑，则必进入结局一。
    如果手上和身上都没有这些道具，则进入2.判断。
   2.若该角色击杀了Synn(Finish)，则进入结局二；否则进入3判断。
   3.身上所持的钱大于等于3000SP，进入结局三。否则进入4.。
   4.进入结局四。(身上的钱小于3000，即0–2999SP)

○矮人
  1.若手持战斧、传说之剑、雷剑、火剑、冰剑，必进入结局一；
    若没有，则看6个道具栏，道具栏内有战斧、传说之剑、雷剑、
    火剑、冰剑，则必进入结局一。
    如果手上和身上都没有这些道具，则进入2.判断。  
  2.若该角色击杀了Synn(Finish)，则进入结局二；否则进入3判断。
  3.满足某不明条件A，进入结局三。否则进入4.。
  4.进入结局四。

○法师
  1.若该角色击杀了Synn(Finish)，则进入结局一；否则进入2判断。
  2.满足不明条件A，进入结局二；否则进入3.的判断。
  3.身上所持的钱大于等于3000SP，进入结局三。否则进入4.。
  4.进入结局四。(身上的钱小于3000，即0–2999SP)

○牧师
  1.若该角色击杀了Synn(Finish)，则进入结局一；否则进入2判断。
  2.身上所持的钱大于等于3000SP，进入结局二。否则进入3判断。
  3.满足不明条件A，进入结局三；否则进入4.的判断。
  4.进入结局四。

○盗贼
  1.若该角色击杀了Synn(Finish)，则进入结局一；否则进入2判断。
  2.满足不明条件B，进入结局二；否则进入3判断。
  3.身上所持的钱大于等于3000SP，进入结局三。否则进入4.。
  4.进入结局四。(身上的钱小于3000，即0–2999SP)
