feversos 弹铳 (C) Cave 1998¶

弹铳 (C) Cave 1998

$info=dfeveron,feversos

[ 基本操作 ]¶

[ 基本操作 ]
───────────────────
  Ａ：普通火力
　　　按住可切换射至高威力火力
  Ｂ：保险
　Ｃ：普通连射 (需机台设置中打开)

·连射火力
　TYPE-A   １点集中型
　TYPE-B 　  ３方向型
　TYPE-C   范围扩散型

·强火力
　LOCK-ON 锁定攻击
　BOMB　　连环爆炸爆弹
　ROLL　　按住蓄力会出现一些绕自机旋转
          的光球。松开按键会释放出去。

[ 隐藏要素 ]¶

[ 隐藏要素 ]
───────────────────
[ 隐藏飞机鱼鱼 ]
投币后，快速输入↓↑→←↑↓←→开始键。
1P的名字是Uotaro，2P名字是Poko(81)。 
注意：续关的时候，需要重新输入一次。

[ 限时模式 ]
投币后，按住Ａ＋Ｂ，同时按开始键 

[ 真BOSS条件 ]
不死命。真BOSS战时，玩家始终是无敌状态。

[ 1-UP ]
当收集到2200个碟片后会在画面中央出现。
