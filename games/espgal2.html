  普通模式与觉圣模式的切换

○防御罩 
  按住会开启防御罩，同时屏幕下方的能量槽也会逐渐
  减少，消失时会放出巨型激光(威力随防御罩持续时
  间而增大)

○连射
  移动速度快、但威力相对小

[ 系统介绍 ]¶

[ 系统介绍 ]
─────────────────────────
○圣灵石
在游戏中自机每击破一名敌人都会得到数颗绿色的圣灵石
，这些圣灵石就是觉醒时所需要的消耗品。
灵石最多积累至500，受到攻击后，圣灵石会变为250；
若死亡，圣灵石会变为100。

○金块
觉圣模式下击破敌机可得，最多积累至999，
受到攻击时会减少一半。

○觉圣模式
有圣灵石时按下觉圣键(Ｂ)进入觉圣死界，所有敌弹就会
变成紫色，速度亦大幅减慢，降低回避的难度。在觉圣模
式中击破敌机，画面内所有该台敌机发射的子弹都会变化
成金块。当圣灵石耗尽时，自机仍在觉圣中的话，觉圣死
界就会消失，取而代之的是敌弹变为红色，速度比原本的
更快。
在该模式中，中弹时会自动发动防御罩，但相应的会损失
50%的防御能量。注意：该模式下，每击破一名敌人，都
会减少圣灵石。

○绝死模式
有圣灵石和金块时，长按住觉圣键(Ｂ)进入绝死模式。
敌人火力更猛，子弹速度更慢，但分数倍数提升为500。
圣灵石和金块会不断减少。金块耗尽会进入觉圣模式。
圣灵石耗尽，自机仍在觉圣中的话，觉圣死界就会消失，
取而代之的是敌弹变为红色，速度比原本的更快。
击破BOSS之外的敌机，画面内所有该台敌机发射的子弹
不会变化成金块，反而会产生更多子弹。
死尸弹形成有一个时间。如果此时退出绝死模式，
死尸弹会立刻实体化。在该模式中，中弹时会自动发动
防御罩，但相应的会损失所有防御能量。
注意：该模式下，每击破一名敌人，都会减少圣灵石。

○赤走模式
短按觉圣键，无石头时的状态，子弹为红色。
子弹变快。击破敌人，本体会爆出黄金。
在该模式中，中弹时会自动发动防御罩，
但相应的会损失50%的防御能量。

○绝走模式
长按觉圣键，无石头且有金块时的状态，子弹为红色。
子弹变快变密。击破敌人，本体会爆出大量黄金。
在该模式中，中弹时会自动发动防御罩，但相应的会损失
所有防御能量。
赤走/绝走状态下会自动增加赤走等级，最多3级。
一旦增加，除非续币，不会降回。
赤走等级增加后石头和黄金会获得更多，但是普通状态
以外的几种状态弹速都会更快。

[ 保险介绍 ]¶

[ 保险介绍 ]
─────────────────────────
    本作的保险延续了ESP系列的传统，并非是按个数施
放，而是消耗防御罩槽，可以通按住Ｂ键蓄力的时间调整
威力。在蓄力时，自机会一直处于无敌状态，对于自机
周围的敌人有小幅攻击力。
    最小的保险，也就是按一下就放出的保险，攻击力
非常微弱，消耗25%的防御罩槽。不过如果剩下的防御罩
槽小于25%，哪怕只有一丝，也是可以放出小保险。
蓄到30%时放出三条光柱，35%时是四条，40%时是五条，
再蓄力则不会增加保险威力。保险的威力可通过听蓄力的
声音把握，声音循环一次约15帧。6次以下是两条光柱，
6至11次是三条，12至18次是四条，19次以上是五条。
    普通以外的四种状态中弹都会自动放保险，觉圣/赤
走模式消耗多半管防御罩槽，绝死/绝走模式直接消耗所
有防御罩槽。绝死/绝走模式中弹后如果没有死，会自动
进入觉圣/赤走模式，要想维持绝死/绝走模式的状态，
需要在中弹时长按住觉圣键。

[ 道具介绍 ]¶

[ 道具介绍 ]
─────────────────────────
圣灵石：上限为500，普通模式击破敌人可得，
用于维持觉圣模式和绝死模式。
近距离击杀敌人会获得更多灵石。
  Ｐ：升级火力
ＭＰ：火力达到最大
  Ｅ：恢复50%防御罩能量
  Ｌ：额外生命，第3关取得
金块：分数，发动绝死模式。

[ 奖命道具1-UP条件 ]¶

[ 奖命道具1-UP条件 ]
────────────────────────
关卡3：需满足以下三个条件。
1、摧毁第3关小BOSS的所有部件(左边三个，右边三个)
2、小BOSS战至打出1-UP这段时间内不能使用防御罩。
3、关底BOSS战前，得到Ｅ道具的地方，绝死模式攻击
画面左边的两个圆锥体中间的地方一段时间。

[ 真BOSS条件 ]¶

[ 真BOSS条件 ]
───────────────────
在见第5关BOSS前无死命，且赤走等级为3。
