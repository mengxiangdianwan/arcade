hotd 死亡之屋 (C) Sega 1996¶

死亡之屋 (C) Sega 1996

抖音《梦想电玩》粉丝专属街机秘籍 抖音橱窗、直播间均可下单 街机游戏名称代码：hotd

[ 光枪校验 ]¶

[ 光枪校验 ]
─────────────────────────
1)进入游戏后按test键(即机台菜单键，预设F2)进入test menu，
用service键将游标移至gun setting处按test键进入二级菜单，
同样方法进入gun mark check
2)将准心对准左上角蓝色圆圈中心，按一下扳机即可，之后退回
到游戏即可

[ 角色选择秘笈 ]¶

[ 角色选择秘笈 ]
─────────────────────────
此处的←指1P射击键，→指2P射击键。
1P使用Sophie---投币后，依次按→←→(2)←(2)开始键。
2P使用Sophie---投币后，依次按←→←(2)→(2)开始键。
1/2P都使用Sophie---投币后，依次按→←(2)→(2)←
1P使用Super G---投币后，依次按←(2)→←(2)开始键。
2P使用Super G---投币后，依次按→(2)←→(2)开始键。
1/2P都使用Super G---投币后，依次按←→←→←→，
演示动画开始前按开始键

[ 小提示 ]¶

[ 小提示 ]
─────────────────────────
如果解救了所有需要救助的人，最终关进入石灰岩洞穴后，
玩家会进入一个隐藏房间获取道具。

[ 显示玩家的分数 ]¶

[ 显示玩家的分数 ]
─────────────────────────
投币后，快速输入←(3次)→(2次)开始键。
此处的←指1P射击键，→指2P射击键。

[ 结局说明 ]¶

[ 结局说明 ]
─────────────────────────
1)好结局：通关分数至少62000分
2)坏结局：通关分数在62000以下且分数的最后一位是0
3)一般结局：通关分数在62000以下但分数的最后一位不是0
