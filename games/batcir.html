[ 送补血道具的金色飞碟 ]¶

[ 送补血道具的金色飞碟 ]
──────────────────────────────
  关卡２－７的每一个连续的大场景，当前玩家中最高的那个分数
  达到特定值，即会在当前画面中部的位置，从右至左缓缓飞出一
  个闪光的金色飞碟。
  一个连续大场景，指的是没有剧情转换/黑屏切换的一组场景。
  比如关卡５转场5次，可以触发最多5个飞碟。

  ①每一个大场景都能且只能触发一次飞碟。
  ②关卡中和关底BOSS战均可触发飞碟。
  ③关卡１和关卡８无金色飞碟。 
  ④关卡３追逐博士环节和关卡５打玻璃墙环节不可触发飞碟。
  ⑤关卡３第一个场景限定必须有达到触发点的分数变动才能触发。

  ■分数条件(全流程理论上最多20个飞碟)
  实际上多数分数在正常情况下完全达不到。
  金手指开无限分数即可体验全飞碟触发。

[ 攻略提示 ]¶

[ 攻略提示 ]
──────────────────────────────
◎COIN的获取
  取得掉落在地上的钱币或通过特殊攻击打击敌人使之掉出钱币
  可增加COIN点数。
  1铜币=1点COIN  / 1银币=2点COIN / 1金币=5点COIN /
  1钱袋=10点COIN / 1钱箱=20点COIN

◎道具店
  每关过后会进入道具店，可买到以下特殊能力和道具：
  新必杀技或强化原有技能(共5种)
  VITAL GAIN (消费400点COIN，增加体力槽上限)
  BDL UNIT   (消费200点COIN，可增加1个，最多可同时携带5个。
              如果已经拥有5个，则道具店无此项目)
  EXTEND     (消费1000点COIN，增加1次生命)
     
◎额外奖励
  第三关的公路追逐战，其实是一个奖励关卡。角色可在此过程中，
  收集地面上的铜币。如果能收集全部铜币，便可得到额外的COIN
  点数奖励。
 
◎隐藏道具
  第5关一开始从巨型齿轮处逃生后，会看到被关的博士。
  破坏牢门，他会送增加体力槽上限的道具做答谢。

◎BOSS战得分
  BOSS战得分由三部分组成：BOSS击杀得分、TIME、VITAL。
  BOSS击杀得分每关都是固定的。
  VITAL得分是击杀BOSS时的剩余生命百分比值x300，最高30000。

  TIME得分是根据BOSS战耗费的时间评级奖分，共5个等级。
  RANK Ｓ  100000分
  RANK Ａ   50000分
  RANK Ｂ   10000分
  RANK Ｃ    5000分
  RANK Ｄ    3000分
  关卡８特殊奖励的TIME得分=上述得分x10

[ 秒杀BOSS方法 ]¶

[ 秒杀BOSS方法 ]
──────────────────────────────
    瞬间秒掉Ｎ管血的原理是——一瞬千击：当一种高频率多段判定
招式（如食人花的蜜蜂攻击）发动攻击中，用其他长持久性单发招
（如鸵鸟强化↓↑）攻击该敌人，将会使↓↑形成多段判定。多段攻
击频率越高，给单发招增加的判定次数越多。也就是说在蜜蜂发动攻
击的过程中，同时展开攻击的队友越多，攻击频率也就越高，每一招
增加的段数越多。最少需要三人组队，再利用蓝侠的性能提升技加成
攻击力，也就能秒杀BOSS了。不过能制造一瞬千击机会的多段技能非
常少，目前能成功秒BOSS的仅发现食人花的蜜蜂攻击，其他多段技配
合出来的效果都不理想。

达成这种玩法的基本条件：
① 至少需要三人组队，同时发动攻击的频率才能达到秒杀的要求。
② 至少需要一个高端食人花玩家。泼墨（蜜蜂）是该玩法的核心技
   能，食人花的水准直接影响该套路的稳定性，至少要熟练掌握对
   全体敌人倒地追加泼墨的时机。
③ 至少需要一个中端蓝侠玩家。理由有四个：部分开局需要蓝侠起
   手；秒杀依赖蓝侠的性能提升技加成双倍攻击力；性能提升技的
   暗屏时间很容易影响其他队友行动，选择爆Ｂ的时机至关重要；
   蓝侠的普通↓↑，持续时间长，伤害不低，是增加瞬间攻击频率
   的必备招式。
④ 至少需要一个理解秒杀思路，悟性好的输出，水准可以不用很高。
   三人的话这个位置推荐鸵鸟担当，强化↓↑单发威力巨大，在一
   瞬千击里面担任主要输出手段。一次↓↑打出５段或以上的判定，
   配上蓝侠的性能提升技就是两管多血，加上其他队友的伤害，秒
   杀就轻而易举了。

    满足该玩法的基本条件后，就是商量套路的阶段了。先要选出一
个领队，推荐水准最高的那位玩家担当。集合队友在群里面讨论每一
场BOSS战的打法。从开局，精确到每一个人每一次出的招，出招的时
机。一般秒杀流程都不长，每人出招的次数就１-２招左右。每人出
招时机安排合理，执行到位，就可以实现秒杀。

[ 隐藏BOSS「天帝」]¶

[ 隐藏BOSS「天帝」]
──────────────────────────────
■条件１
  单人玩时分数达到320W / 双人玩时一人分数达到260W /
  三人玩时一人分数达到230W / 四人玩时一人分数达到200W
■条件２-Ａ
  道具店购买了除BDL UNIT和EXTEND之外的其它全部6个升级道具。
  如果第5关开头处救博士得到增加体力槽上限的道具，则只需5个
  升级道具即可。
■条件２-Ｂ
  一人拥有1000点COIN

当满足条件(１+２-Ａ或１+２-Ｂ)，击败第7关BOSS-PLUTO后，
玩家会有两个选择：
  ①选择右边的选项是删除天帝系统，直接通关；
  ②选择左边的则与天帝系统主程序真BOSS对战。

    与天帝系统主程序对战前，玩家所有技能都会被升级(即使没从
道具店买过该道具，也可使用所有技能了)，所以玩家可以进入道具
店，钱全部买BDL UNIT和EXTEND。

    真BOSS大多数技能是全屏攻击，玩家可放护身保险躲避，发动后
会有一小段无敌时间。
