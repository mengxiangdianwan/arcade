[ 柱子相关技巧 ]
每个场景都有一些柱子类型的道具，
比如路灯杆、房柱。
靠近这类道具Ａ＋Ｂ会发动特殊的技巧。

○木马踢：柱子旁Ａ＋Ｂ。整个过程带攻击判定。
  适用于体型瘦小的角色，如WANGTANG、AYAME。
○横扫  ：柱子旁Ａ＋Ｂ。适用体型巨大的角色。
○跳起抱柱：在柱子旁跳起Ａ＋Ｂ。
  抱住柱子时ＡorＢorＡ＋Ｂ可发动远距离攻击。

[ 关于变身 ]
将对手击飞时可打出其身上的宝石。
得到三个宝石时会变身。
变身状态特性：
１不会被击倒，也不会被打出宝石。
２所有角色在柱子旁都可使用横扫的技能。
３通常技变化、可发动超必杀技。
４使用通常技和超必杀技会消耗下方的能量槽。
  能量槽归零时，变身状态解除。
☆变身超必杀技①：Ａ＋Ｃ
☆变身超必杀技②：Ｂ＋Ｃ

[ 艾德华·霍克(FALCON) ]¶

[ 艾德华·霍克(FALCON) ]
──────────────────────
主角地位的角色。性别男。职业是冒险家。
为了找到霍克家流传的书中所写的“力量宝石”
而在世界旅行着。
爱机名称是霍根海姆号。因为能力平均而好上手。
动画版中变成因为憧景瓦鲁卡斯、而立要成为像他
一样的格斗家。
台词是“红色的疾风”。
变身后的模样是以重火枪为武器的红色铠甲机器人

变身后必杀技：
☆跟踪飞弹：Ａ
☆灼热飓风：Ｂ
★爆裂飞弹：Ａ＋Ｃ
★火箭冲击：Ｂ＋Ｃ

[ 终极瓦鲁卡斯(FINAL VALGAS) ]¶

[ 终极瓦鲁卡斯(FINAL VALGAS) ]
──────────────────────
真正的最终魔王。在得到力量宝石的巨大力量后而
成为像丑陋怪物的模样。
头部只剩下原来的面貌、丧失任何意识而会对任何
人进行无差别攻击。因为身体巨大所操纵也有点差
异。
动画版中最终和威力提升的霍克进行死斗。
VERSUS模式中使用该角色时，不能同终极瓦鲁卡斯
对战。
平时即是变身状态，所以力量宝石无意义。

[ 武器介绍 ]¶

[ 武器介绍 ]
──────────────────────
燃烧瓶：可造成多段攻击。

炸弹：被丢弃时会有5秒倒数，倒数完毕会爆炸并
对周边玩家都造成伤害。捡取后会降低速度但是
爆炸半径很可观。

火箭筒：捡取后会大幅降低移动速度，但射击范围
广、威力很大。

大锤：威力大，被击中的敌人都会肥的像块煎饼。

重剑：攻击范围变为身前45度，空中Ｂ的威力大，
会少许降低移动速度。

大棒：和重剑相同。

手枪：共6发子弹，会降低速度。

火焰喷射器：中程火力，会大幅降速。

如意棒：威力大、射程覆盖全屏。

光线枪：火力半径很大，中等射程。

盾牌：自动防御玩家攻击。

迷你枪：威力非常大且射速快。

[ DC版隐藏要素 ]¶

[ DC版隐藏要素 ]
──────────────────────
·改变服装
选择角色时按Ｂ

·双屏VERSUS模式
使用Valgas通游戏后解锁。类似VERSUS模式，但画
面中间是分开的，两名角色以第一视角进行游戏。

·使用Kraken
除了Valgas之外的所有角色都通游戏，即可在
Arcade和Versus模式中解锁。

·解锁Valgas和VERSUS模式
使用Kraken打通Arcade模式。

·使用Final Valgas
先解锁Valgas，再任意难度下不续币通游戏，
即可在VERSUS模式使用。

·使用BOSS
使用所有角色通游戏。

·解锁Ayame的忍者之星小游戏
使用6名角色打通Arcade模式。

·解锁角色结局
使用不同角色通Arcade即可解锁对应角色的结局

·解锁Falcon的飞机追逐游戏
使用5名角色打通Arcade模式。

·解锁Gunrock的Gun Gun Slots游戏
使用角色选择界面所有初始角色通Arcade模式

·解锁Machine Gun道具
使用2名角色打通Arcade模式。

·解锁Nyun Boi道具
使用1名角色打通Arcade模式。

·解锁Power Shield道具
使用4名角色打通Arcade模式。

·解锁Ray Gun道具
使用3名角色打通Arcade模式。

·控制镜头视角
使用Ａ，X，Y，Ｂ和手柄以及摇杆控制
放大缩小以及旋转镜头视角。

·解锁画廊
在小游戏中赚得1000和2000金币分别能
解锁COLLECTION中的一个画廊内容。
