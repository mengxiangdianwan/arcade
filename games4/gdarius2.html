[ 系统设置 ]
─────────────────────────
武器最大等级为3级。对空是七个红球升一级；
对地是四个绿球升一级；防护罩是六个蓝球升一级。

·导弹　　  (00-01-02-03-04-05-06)
　→镭射　  (07-08-09-10-11-12-13)
　　→电波  (14-15-16-17-18-19-20)

[对地(绿)]
·炸弹　　　　  (00-01-02-03)
　→双向炸弹　  (04-05-06-07)
　　→四向炸弹  (08-09-10-11)

[防护罩(蓝)]
·防护罩　　　　　  [耐久力3/+1](00～05)
　　→强力防护罩　  [耐久力4/+2](06～11)
　　　→超级防护罩  [耐久力5/+2](12～17)

[ 光束对攻 ]¶

[ 光束对攻 ]
─────────────────────────
    玩家可发动α光束，而每一只BOSS也都装备有类似的
武器“β光束”。当双方的光束相互冲击的时候，玩家必
须快速连打以增强“α光束”的威力，胜利的一方可将对
方的能源吸收过来，化为二倍、三倍甚至全画面粗的巨大
光束炮，称为“光束对攻”，予以BOSS致命痛击。

[ 关卡选择 ]¶

[ 关卡选择 ]
─────────────────────────
1)倒数计时的时候按住"业务用开关"键(mameplus预设9)
，直到画面显示Service Switch Error
2)输入开始键(3次)，"业务用开关"键，开始键
3)投币开始游戏会进入关卡选择界面

[ 版本区别 ]¶

[ 版本区别 ]
──────────────────────────
Ver.2几个重要改变
1、增加初心者模式(3关但可选关卡)和正常模式(5关)选择
2、增加连射键
3、画面的进行速度大幅加快
4、敌方角色的强度、攻击方式大幅改变。以中BOSS为例，
其面罩的硬度增强62%，因此用CAPTURE BALL捕捉变得相当
困难；而一般敌人的弹数、弹量及耐久度，均有明显提升
5、第1版中放炸弹或吃到POWER UP自机发光的一瞬间是无
敌的，Ver.2中一律取消。
6、自机防护罩的防御能力大幅削弱。第1版中NOMAL等级的
防护罩即可防止一般敌人的冲撞，在Ver.2中仅“SUPER级
(银色)”和“HYPER级(金色)”的防护罩有此能力，但也只
能抵挡一次。
