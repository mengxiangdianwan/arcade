[ 基本操作 ]¶

[ 基本操作 ]
───────────────────────
①操纵杆
  ↖↑↗  后跳/垂直跳跃/前跳
  ←Ｎ→  后退&站立防御/回中/前进
  ↙↓↘  下蹲&下蹲防御/下蹲/下蹲

  →→      ：前冲
  →→↘    ：前冲滚翻
  →→按住不放·↓↓：前冲滚翻(取消)下蹲
  ←←      ：疾退
  ←←↓长按：疾退趴下
  ←←→长按：疾退前冲
  ←←→↘  ：疾退前冲接前翻
  ←←↘    ：疾退前翻
  ↙↙      ：后滚翻
  ↘↘      ：前滚翻
  ↓↓      ：趴下

②按钮
  Ａ：轻斩
  Ｂ：中斩
ＡＢ：重斩
  Ｃ：轻脚
  Ｄ：中脚
ＣＤ：重脚

    ＢＣ：小跳跃(下段闪避)(后方)
→＋ＢＣ：小跳跃(下段闪避)(垂直)

  拾起武器：在掉落的武器处ＡorＢorＡＢ
  挑衅：一定距离之外ＡＣorＢＤ

③投技
  ←or→＋ＢorＡＢ、←or→＋ＤorＣＤ

④防御反弹
  被击中(0.1秒)前防御
  成功的话，对手会出现极长硬直。

⑤空手入白刃
  空手状态←↙↓↘→

⑥格剑
  战斗中达成以下条件之一，就会形成格剑。
  1、双方的武器相交，严格的说是“斩”击的攻击
     判定区域重叠在一起，且双方都没有被击中。
  2、双方“前冲”撞在一起。

  形成格剑状态后，通过快速连打“斩”键加大格剑
  的胜率。只有连打次数较对手有较大优势，才有大
  几率胜出；否则胜方都是随机的，也可能双方不分
  胜负武器同时掉落。此外，还有极小几率，一方连
  打，另一方不按键的情况下，仍然会出现双方不分
  胜负武器同时掉落的情况。
  格剑落败的一方会被击倒，武器也会被弹飞。

  格剑胜者在格剑结束的瞬间，输入→可对敌进行一
  次追击，随机发动踢攻击或空中斩。

·Q版变身技：无任何限制，可随意使用
·秘奥义：无任何限制，可随意使用
·武器破坏技：怒气槽全满才可使用

[ 关于怒气槽 ]¶

[ 关于怒气槽 ]
───────────────────────
·气槽增加条件
  １防御对手的必杀技。
  ２被对手的必杀技击中。
  ３被对手的通常技击中。

·怒槽满的动作
  怒气槽满时会有一个发怒的动作。
  发怒动作中无法执行任何指令。
  发怒动作中无敌。

·怒气槽满时的效果
  １攻击力增加。
  ２可发动武器破坏技。

[ 指令符号说明 ]
───────────────────────

◇：伪技
□：变Q版人物指令

▲：有武器状态
△：空手状态

[ 隐藏要素 ]¶

[ 隐藏要素 ]
───────────────────────
①CPU黑子乱入条件
  1.第3人以后1/16的机率。
  2.在和第6/8/10人对战时、剩余时间在20秒以下
　  的时候用武器破坏必杀技取得胜利。
  以上条件满足其一、黑子即会出现。

②无敌状态
  站立拉防，当敌人的攻击快要命中你的瞬间，放开
  摇杆，敌人的攻击会打不到你。该技巧仅加尔福特、
  地震、柳生十兵卫、黑子这四名角色可用。

③选择胜利动作
  胜利后，按住↑↓→中任一方向。每个方向对应一
  个胜利动作。如果不按方向，默认是做出→方向的
  动作。
  角色的胜利动作不只3种，但每次胜利只能选其中2
  或3种，依角色以及是第1次胜利还是第2次胜利而定
  动作，以牙神为例，第1回合胜利时，只能指定其中
  2种胜利动作，决胜局获胜后则可指定3种胜利动作。

④选择第二种颜色
  使用Ｄ选择角色

⑤标题画面变化
  不同的机器设置时间，对应不同的标题画面。
  具体方法：
  机台设置中的 SETTING UP THE CALENDAR 中，
  上方的 CURRENT TIME 显示的是当前时间，
  下方的 THE TIME TO BE SET UP 表示设置时间。

  不同的月、日、小时的组合对应不同标题画面
  共７种变化：
  金花、枫叶、花札、樱花、彩纸、雪花、羽毛

⑥特殊失败状态
  决胜局击败对手，一般是出现后仰倒地的状态，但
  使用特定攻击击败对手有一定几率出现特殊状态。
  １武器攻击or武器技击败空中敌：空中分尸
  ２武器攻击or武器技击败地面敌：袈裟斩or纵向喷血
    or空中分尸(仅美版可分尸)

  满血时武器攻击or武器技击杀对方，一定几率爆出
  大量分数道具。
        王虎：佛珠
        地震：烤鸡
        黑子：心脏
      霸王丸：金币
      橘右京：苹果
      夏洛特：玫瑰花
      莱锡路：弹壳
    加尔福特：骨头
    服部半藏：手里剑
    娜可露露：樱花
    查姆查姆：香蕉
  牙神幻十郎：花札

  千两狂死郎：一番扇

⑦背景乱入“金甲唤”
  背景那个投掷炸弹和补血道具的人叫“飞脚”，
  相当于中国古代的信使。
  他出现时有1/128的概率变装成金甲唤，以拳皇中
  超杀凤凰脚的起手式划过版面，不会丢出任何道具。

  1币通关后能看到每名角色的特别演出和职员表

  背景中间大罐中熬的是异形
