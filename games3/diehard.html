[ 武器介绍 ]¶

[ 武器介绍 ]
────────────────────────────
[ 枪械类 ]远距攻击、高伤害
────────────────────────────
○手枪(Handgun)                 弹药量：12
  最多可同时持有两把
○机枪(Machine Gun)             弹药量：20
  比手枪威力高、可连射
○导弹发射器(Missile Launcher)  弹药量：6
  威力高、范围广
○反坦克步枪(Anti Tank Rifle)   弹药量：6
  威力高、对敌有吹飞效果
○催泪瓦斯(Spraycan)            使用次数：6
  有[打火机]时会变成火焰发射器
○胡椒(Pepper)                  使用次数：6
  使敌人丧失战斗能力。武器效果滞空时间长。

[ 利器类 ]近距攻击、可连续快速攻击
────────────────────────────
○匕首(Knife)
  威力小、使用简单
○消防斧(Fire Axe)
○破瓶(Bottle) 
  距离极短、威力一般

[ 棍棒类 ]中距武器
────────────────────────────
○甲板刷(Broom)
  打扫甲板用的长刷子。非常长、攻击距离较远
○机械臂(Robot Arm)
  机械人掉落的手臂
○棍棒(Club)
○高尔夫球杆(Golf Club)
  终BOSS的球杆

[ 投掷类 ]
────────────────────────────
○大座钟(Clock)    使用次数：1     
  复古风大座钟。击中敌人会散架
○铁桶(Barrel)     使用次数：1
  标有危险标志的挥发性燃料桶。击中敌人会大爆炸
○25寸显示器(TV Monitor)      使用次数：1
  使用同铁桶、击中敌人会爆炸。
○桧木椅子(Chair)  使用次数：1
  桧木制作的名贵椅子。击中敌人会散架

[ 弹夹 ]特定武器补给
────────────────────────────
○弹夹（Ｈ）  弹药量：12
  手枪弹药12发增加
○弹夹（Ｍ）  弹药量：20
  机枪弹药20发增加

[ 体力恢复类 ]
────────────────────────────
○胶囊
  体力少量恢复
○饮料
  体力中量恢复
○急救箱
  体力大量恢复

[ 人物介绍 ]¶

[ 人物介绍 ]
────────────────────────────
[ 布鲁诺·迪林格 ]
────────────────────────────
S.F.P.D的破案率极高的刑警达人、传说逮捕的犯人的损害赔偿
金一麻袋都塞不满，被尊称为「爆裂刑警」。
因为某些原因，和妻、子分隔两地。

年龄：42
出生地：德克萨斯州
职务：副警督
趣味：DIY
苦手：此次行动的SWAT队长

[ 辛迪·赫迪 ]
────────────────────────────
警察学校主席毕业、才华横溢，23岁就赴任SWAT队长。
也许是因为她的辉煌经历，言行总是自信满满、
对别人经常冷嘲热讽

年龄：23
出生地：纽约州
职务：警部
趣味：开车
苦手：像野兽一样的副警督

[ 沃尔夫·本乡（别名） ]
────────────────────────────
永恒之塔企业集团的中枢机构「渡边·安保·服务」理事。
包括名字在内，所有身份都包裹在迷雾当中。

年龄：40～50之间
出生地：不详
职务：WGS理事
趣味：坐禅·盆栽·日本酒
苦手：不明

[ 卡罗琳·洋子·鲍威尔 ]
────────────────────────────
恐怖分子的人质、12岁天才头脑大学生、总统唯一的女儿。
想像父亲一样、梦想成为合众国历史上第一位女总统。

年龄：12
出生地：犹他州
职务：VIP！？
趣味：高台跳水
苦手：总统秘书
