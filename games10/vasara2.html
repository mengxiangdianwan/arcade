[ 蓄力攻击 ]¶

[ 蓄力攻击 ]
───────────────────
不仅能对敌人造成较大伤害，还可消弹，
但[紫弹]不能被消。

与前作比，主火力的重要性更加突出。
蓄力攻击成为主要攻击手段。

蓄力攻击造成的连斩是加分的重要手段。

[ 擦弹加分 ]¶

[ 擦弹加分 ]
───────────────────
自机中弹判定点周围擦弹时会有金色火花显
现，同时有加分。

[ 连斩加分 ]¶

[ 连斩加分 ]
───────────────────
用蓄力攻击击杀敌人才可形成连斩。
当蓄力杀敌数达到3时，即会在人物头像下
方显示连斩数。以后蓄力杀敌数会在此基础
上累计，最高999。若死命，则需重新进行
连斩。

每一次蓄力斩杀的分数=连斩数×１００分

[ 全斩加分 ]¶

[ 全斩加分 ]
───────────────────
各关有大BOSS１·中BOSS４·武将２０，
合计２５个有名字的武将（兜首）存在。
击杀他们，在关底会有加分。

斩杀１人得５０００分。当斩杀所有武将时
，过关会显示“全PERFECT斩”，
得１５００００分。
（２４斩时是１２００００点）

[ 自机简介 ]¶

[ 自机简介 ]
───────────────────
·武田 信胜 [岚樱花]

移动速度：第４

主火力：窄范围集中型
副火力：一瞬停滞后消失

蓄力攻击：“虎裂斩”
          蓄力时间短、前方窄范围
　        攻击区域也包括自机侧面和斜后
          方


·百地 三太夫 [菊月]

移动速度：第２

主火力：窄范围弱扩散型
副火力：一定角度放射状发射

蓄力攻击：“螺旋舞”
　        蓄力时间最短、横向范围广，
          达半屏。


·清泷坊 随风 [慈眼]

移动速度：第３

主火力：窄范围集中型
副火力：佛珠绕自机转１周后飞出跟踪攻击

蓄力攻击：“百烈棍棒”
　        范围窄、消弹较难、
　        攻击力最强，但蓄力时间也最长


·明智 光秀 [桔梗]

移动速度：第５

主火力：广范围集中型
副火力：周围发射后、自动爆炸或触敌爆炸
        范围和攻击力不错

蓄力攻击：“乱舞刀”
　        攻击范围较大
　        蓄力时间长


·杂贺 孙一 [八咫乌]

移动速度：第１

主火力：窄范围弱扩散型
副火力：大量跟踪扇

蓄力攻击：“紫炎”
　        自机周围完全覆盖、
          完全防御普通敌弹。
　        攻击范围极其小，
          蓄力时间比三太夫略长。
