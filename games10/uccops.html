[ 五角星的含义 ]¶

[ 五角星的含义 ]
─────────────────────────
每关之后都有一个Assessment Day环节。该环节中，有一
个Finess Trick项目，代表你的战斗风格得分。玩家的不
同招式有不同的Finess Trick分数。可以通过观察血槽下
方的黄色五角星数目来了解当前的Finess Trick得分情况
。五角星上限100个，每10个小五角星会合成显示一个大
五角星。Finess Trick项目的主要目的是弥补关底评分和
最低要求分之间的差距。

世界版/美版击杀敌人和使用投技可得五角星。
阿尔法更新版/日版使用投技才可得五角星。

获得规则：
1、投技：克劳德每2次投技得1颗星，若使用的是空压落
且击杀敌人，则得2颗星；马特/罗莎每5次投技得1颗星
2、抓打投：发动1次均得1颗星。
3、连打投：克劳德发动1次得1颗星；马特发动1次得2颗
星；罗莎发动2次得1颗星。

[ Assessment Day简介 ]¶

[ Assessment Day简介 ]
─────────────────────────
Assessment Day将对玩家本关的表现进行评分，包含杀敌
数与Finess Trick两项评估。除了评价分之外，每关还有
一个Border Score，即每关玩家应当达到的评估得分下限
。
如果玩家的评估分大于等于该分值，则相当于完成了关卡
清理任务，会得到奖励。每得到两次奖励则会奖一命。
尽量多用Finess Trick高的技能杀敌，不然关底的Border
 Score难以达到。

[ 坏结局 ]¶

[ 坏结局 ]
─────────────────────────
第5关中，CRAYBORN博士正投下导弹期间，在其将导弹投
下前攻击一下博士便可停止投弹。否则，通关是坏结局

[ 版本区别 ]¶

[ 版本区别 ]
─────────────────────────
阿尔法更新版/日版可发耗血必杀技，也多了一些其它小
招式，多了某些场景道具(比如第2关中段的一个箱子)，
音乐以及获得五角星的方法和世界版/美版也不同。第2关
前半段的背景也不同。
